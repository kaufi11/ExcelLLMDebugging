- Problem/concern: C46 uses C41/C34 (Market value / total earnings) to produce a P/E. That is mathematically okay (Market Cap / Earnings = Price / EPS) but it obscures the per‑share calculation and can be misleading if earnings are negative or when using an average that includes negative years.
- Fix / recommendation: Compute P/E using per‑share earnings to avoid confusion:

- Why: Using totals masks issues when earnings are negative or volatile — per‑share measures are standard and clearer.

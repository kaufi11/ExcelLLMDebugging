Cell AK6 — Error: formula =AG6+Y6 references AG6 which is blank/misaligned, so the Q4 LPG cumulative is wrong.

Solution: fix the column alignment — place the Q4 LPG values into column AG (or update AK6 to reference the actual column that contains Q4 LPG), then copy the corrected formula down.

Why: many AK cells expect the Q4 LPG value in AG. Because AG is empty (columns shifted), AK sums a blank (0) + Y6 producing incorrect totals. Restoring the proper AG values or correcting the formula will give correct cumulative LPG totals.

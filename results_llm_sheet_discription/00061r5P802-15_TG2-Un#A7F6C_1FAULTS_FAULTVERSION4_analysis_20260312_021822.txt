Why this is a problem

Simple, robust solutions (pick one)

A — Best for only numbering rows that have a name (recommended)
- Put this formula in A2 and copy/fill down the whole column:
  =IF(TRIM($C2)="","",COUNTA($C$2:$C2))
- What it does: counts how many non-blank Name cells are in column C up to the current row, producing a running index only for rows with names. It does not depend on previous A cells, so manual edits or blank rows won’t break the sequence.

B — If you want a simple row-based number regardless of blanks
- Put this in A2 and fill down:
  =ROW()-1
- What it does: shows the worksheet row number minus the header row. This will give a number for every row (even blank ones).

- Use Fill → Series (Home > Editing > Fill > Series) to create a linear series (start=1, step=1) across the used rows.
- This is fine as a one-off but fragile: the series will break again if you insert/delete rows or manually change values.

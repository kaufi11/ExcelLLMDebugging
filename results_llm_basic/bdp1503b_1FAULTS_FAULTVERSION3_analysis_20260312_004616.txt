Summary of errors
- Cell G19 uses the wrong column: its formula is =SUM(D19-E19) but the annual allocation is in C19. This returns an incorrect remaining amount.
- Many "Remaining" formulas use the odd pattern =SUM(Cx-Ey) instead of simple subtraction (=Cx-Ey). Although Excel will evaluate =SUM(C10-E10), it's unconventional and makes the sheet harder to audit and more prone to mistakes (like the D19 typo). Totals also use =SUM(C10+C24+...) which works but is nonstandard.

- Using SUM around arithmetic hides mistakes and can make bulk fixes risky/unreliable.

How to do a quick bulk cleanup
- If many cells use =SUM(Cx-Ey): use Find & Replace carefully:
  - Find: =SUM(
  - Replace with: =
  - Then manually review any SUM(range) formulas (they will be affected), so only do this if you first restrict the Find to the G column (search within Sheet and within column G).
- Or better: edit formulas row-by-row to be safe and verify totals.

Why the corrected formulas are better
- =C19-E19 (and similar) is clear, easy to read and audit.
- Using SUM(range) or SUM(list) for subtotals is standard and avoids ambiguity.
- Clear formulas reduce the risk of future reference errors like the D19 typo.

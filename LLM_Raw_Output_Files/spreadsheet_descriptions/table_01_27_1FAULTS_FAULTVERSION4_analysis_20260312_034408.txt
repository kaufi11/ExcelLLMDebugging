Sheet name and layout
- Sheet: "1-24". Single main table starts at A1 with title "Table 1-27: Condition of U.S. Highway Bridges".
- Columns B:N contain years (B = 1990 … N = 2002). Row labels run down column A (TOTAL bridges, Urban bridges, Rural bridges, deficient breakdowns, notes, sources).

Row structure and primary groups
- Row 3: "TOTAL bridges" — computed as Urban (row 4) + Rural (row 5) for most years (e.g., B3 = B4+B5; L3/M3 use SUM of L4:L5/M4:M5). N3 is hard-coded: 590,877.
- Rows 4–5: Breakdown by area
  - Row 4: "Urban bridges" — many year cells are formulas that add several category counts (long literal sums) for 1990–1997; years 1998–2002 contain direct values in some cells (J4:N4).
  - Row 5: "Rural bridges" — similarly contains long literal-sum formulas for many years and direct values for later years (J5:N5).
- Rows 6–12: Deficient bridges (urban, rural, totals)
  - Row 6: "Urban deficient bridges, total" = row7 + row8.
  - Row 7: "Structurally" (urban structural deficient) — mostly literal-sum formulas for years; later years have direct values (J7:N7).
  - Row 8: "Functionally" (urban functional deficient) — mostly literal-sum formulas; note D8 = 6 (anomalous small literal) and other years have direct values in J8:N8.
  - Row 9: "Rural deficient bridges, total" = row10 + row11.
  - Row 10: "Structurally" (rural structural deficient) — literal-sums and some direct values (J10:N10).
  - Row 11: "Functionally" (rural functional deficient) — literal-sums and some direct values (J11:N11).
  - Row 12: "All deficient bridges, total" = urban deficient (row6) + rural deficient (row9) for most years. L12/M12 use SUM(L13:L14) rather than row6+row9; N12 is hard-coded: 162,791.
- Rows 13–14: Totals by deficiency type
  - Row 13: "Structurally" — many direct values for B13:N13; L13 is computed as L10+L7.
  - Row 14: "Functionally" — many direct values for B14:N14; L14 is computed as L11+L8.
- Rows 16–25: NOTES and SOURCES text blocks (definitions and data source citations).

Formula patterns and data types
- Many cells use literal addition of component counts (e.g., B4 = 24435+11890+21403+18580+11876+20586).
- Higher-level totals generally sum component rows (e.g., row3 = row4 + row5; row6 = row7 + row8; row9 = row10 + row11; row12 = row6 + row9).
- Some totals use SUM(range) (e.g., L3 = SUM(L4:L5); L6 = SUM(L7:L8); L9 = SUM(L10:L11); L12 = SUM(L13:L14)).
- Several cells contain hard-coded values instead of formulas (notably J4:J5:J7:J8:J10:J11:J13:J14 and many N-column totals).
- Minor anomaly: D8 is set to 6 (appears inconsistent with surrounding entries).

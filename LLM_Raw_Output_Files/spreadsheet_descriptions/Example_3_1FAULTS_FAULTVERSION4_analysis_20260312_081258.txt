Summary of workbook structure and contents

Sheets overview
- Q4 Homework
  - Columns A–H: Student names and eight columns of assignment categories (A: name; B–G: item scores; H: row total).
  - Row 1: headers; Row 2: "Points Possible" for each category; Rows 3–11: student scores.
  - Key formulas:
    - H2 = SUM(B2:G2) → total points possible = 300.
    - H3,H4,H6–H11 = SUM(B#:G#) → student homework totals.
    - H5 (Evans) = COUNT(B5:G5) — incorrect (returns count of numeric cells rather than sum).
  - Important values: Points possible per category B2:G2 sum to 300; student totals range ~251–291 (except Evans due to COUNT error).

- Q4 Summary
  - Columns A–G: category weights, points-possible per category, per-student totals, weighted score and weighted average.
  - Row 1: headers; Row 2: weight values (B2=0.3, C2=0.1, D2=0.4, E2=0.2) and F2 = SUM(B2:E2) → 1.0.
  - B3–D3: points possible for Tests/Quizzes/Projects = 300 each; E3 = 'Q4 Homework'!H2 → 300.
  - For each student (rows 4–12):
    - Column E pulls homework total from Q4 Homework (e.g., E4='Q4 Homework'!H3).
    - Column F = B#*0.3 + C#*0.1 + D#*0.4 + D#*0.2 — incorrect: last term repeats D# instead of using E# (homework). Should be + E#*0.2.
    - Column G = (F#/300)*100 → weighted percentage.
  - Result: weighted scores (F) and percentages (G) will be wrong because the 0.2 weight references Projects (D) twice instead of Homework (E).

- Year Summary
  - Columns A–I: names, SSNs, quarter scores (Q1–Q4), final average, and letter grade.
  - Q4 (column G) pulls Q4 weighted percentage from Q4 Summary (e.g., G2='Q4 Summary'!G4).
  - H# = AVERAGE(D#:G#) → final numeric average across quarters.
  - I# = nested IF mapping H# to letter grades: <60 F, <70 D, <80 C, <90 B, <101 A (works as intended).
  - Class averages in row 11: D11..G11 = AVERAGE of quarter columns across students.

- Quarter Grades-Graph
  - Present as a sheet placeholder for charts; no detailed contents provided.

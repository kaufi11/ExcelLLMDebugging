Sheet name: "2001 - 1994" — single worksheet, statewide TAKS/TAAS pass counts by grade for Spring 2001 down through Spring 1994.

Overall layout
- Column A: titles, section headers and footnotes (sheet title, year labels, explanatory notes).
- Column C: grade labels (Grade 3, Grade 4, … Grade 10, "All Grades") for each year block.
- Columns D, E, F: numeric data and percent formulas:
  - D: "Number" (tested) — raw tested counts by grade.
  - E: "Number" (passing) — raw passing counts by grade.
  - F: "Percent (Passing)" — formula dividing passing by tested for each grade.

Block structure (repeated for each year 2001 → 1994)
- Year header (e.g., "Spring 2001"), then a small header row in D–F ("Number", "Number", "Percent") and a Tested/Passing row label.
- One row per grade (Grade 3, 4, 5, 6, 7, 8*, 10) with D = tested count, E = passing count, F = E#/D# (e.g., F10 = E10/D10).
- "All Grades" row at the end of each block:
  - D = SUM(...) over the tested rows for that year.
  - E = SUM(...) over the passing rows for that year.
  - F = E (sum) / D (sum).

Formulas (patterns)
- Per-grade percent: F[row] = E[row] / D[row].
- Year totals: D[All] = SUM(D[start_row]:D[end_row]); E[All] = SUM(E[start_row]:E[end_row]); F[All] = E[All] / D[All].

Notable items / data issues
- Spring 2001 All Grades tested formula is inconsistent: D17 = SUM(D11:D16) excludes Grade 3 (D10), while E17 = SUM(E10:E16) includes Grade 3. That will produce an incorrect overall percent for 2001. (Probable fix: change D17 to SUM(D10:D16).)
- Footnotes at bottom:
  - Row A118: Grade 8 totals include reading, mathematics, and writing tests only.
  - Row A119: Spring 2001 results are preliminary.

Example key values (samples)
- Spring 2001, Grade 3: Tested = 265,357; Passing = 205,320; Percent = E10/D10.
- Spring 1994, Grade 10: Tested = 201,396; Passing = 100,813; Percent = E115/D115.

Summary
- A single-sheet, vertically repeated year blocks with tested/passing counts by grade and per-grade percent formulas. Totals computed by SUM and overall percent by dividing summed passing by summed tested. One formula inconsistency in the 2001 "All Grades (tested)" cell that should be corrected.

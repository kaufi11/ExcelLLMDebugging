Summary of problems
- Many student "Points" formulas sum the wrong range: C9:C24 use Dn:CCn (e.g. =SUM(D9:CC9)). CC is far beyond your actual columns (D:I), so the range is incorrect.
- One student (C19) uses COUNT instead of SUM, so it returns the number of numeric cells (6) instead of the total points.
- Grade % formulas (B9:B24) use =SUM(C9/$C$6). The SUM() wrapper is unnecessary and the result is a fraction (0.88). You probably want a percent display.

Repairs (apply and copy down)
1. Fix total points for each student (C9:C24):
   Replace with:
   =SUM(D9:I9)
   (copy down to C24)

   Explanation: This sums only the actual assignment columns D through I.

2. Fix C19 specifically (was COUNT):
   =SUM(D19:I19)

   Explanation: COUNT returns number of numeric cells; SUM returns the total points.

3. Fix Grade % (B9:B24):
   Option A — separate cells: keep C totals, use
   =C9/$C$6
   then format the cell as Percentage.

   Option B — single-step percent:
   =SUM(D9:I9)/$C$6
   (copy down) and format as Percentage.

   Explanation: Dividing the student's total by the absolute class total ($C$6) gives the correct fraction; format as % (or multiply by 100 if you prefer a numeric percent).

Notes

- After fixing, check formatting (Percentage, 0 or 1 decimal) so results read as expected.

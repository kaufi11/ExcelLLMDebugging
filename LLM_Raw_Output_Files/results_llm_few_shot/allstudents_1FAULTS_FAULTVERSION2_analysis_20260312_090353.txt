Faulty cell(s)
- D17 (All Grades - Tested)
- F17 (All Grades - Percent) — incorrect because it uses the wrong D17 result

Error type
- Omission / range error (D17 sums the wrong rows)

What’s wrong (short)
- D17 uses =SUM(D11:D16) and therefore omits Grade 3 (D10). E17 correctly sums E10:E16, so F17 (=E17/D17) gives a wrong percent because the denominator is too small.

Fix (formulas)
- D17 -> =SUM(D10:D16)
- F17 can remain =E17/D17 (it will recalculate correctly after fixing D17)

Why this fixes it (short)
- Including D10 adds Grade 3 tested counts to the total tested. With the correct denominator, the percent (E17/D17) reflects passing ÷ tested for all grades.

Corrected numbers (for reference)
- D17 = 1,882,232
- E17 = 1,536,561 (already correct)
- F17 = 1,536,561 / 1,882,232 ≈ 0.8164 (81.64%)

Quick check
- Scan other “All Grades” rows — they all use consistent ranges (include Grade 3) except the 2001 D17 shown above.

Findings (short)
- Cell D8 has a typo: the formula uses subtraction instead of addition. It reads:
  =4186-3424-4479-5125-3339-5690
  That produces a negative value (‑17,871), which is impossible for a bridge count and breaks totals that reference it.

Why it's wrong
- All other similar cells sum several component counts; D8 should also be a sum. The minus signs were entered by mistake.

Fix (exact steps)
1. Select cell D8.
2. Replace the formula with either:
   =4186+3424+4479+5125+3339+5690
   or better:
   =SUM(4186,3424,4479,5125,3339,5690)
   (or, ideally, replace the hard numbers with cell references and use SUM on those cells).
3. Press Enter.

Immediate effect (after fix)
- D8 becomes 26,243.
- D6 (Urban deficient total for 1992) = D7 + D8 becomes 16,323 + 26,243 = 42,566.
- D12 (All deficient total for 1992) = D6 + D9 becomes 42,566 + 156,525 = 199,091.
- Other dependent totals update automatically.

Recommendation
- Use SUM with cell references (e.g., =SUM(D7:D8) or =SUM(cell1:cell6)) instead of hard-coded arithmetic to reduce manual typing errors.
- Run an error check for negative counts or use conditional formatting to highlight negative or unexpected values.

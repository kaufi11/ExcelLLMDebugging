Page 2 — cell A83

Error: formula uses =A$9 (local reference) but the header value lives on Page 1, so the cell pulls the wrong/blank value after the header was copied to other sheets.

Fix: change the formula to reference the original sheet explicitly, for example:
='Page 1'!$A$9
(or define a named range for the header on Page 1 and use =HeaderName)

Why this works: a plain A$9 points to A9 on the current sheet. Adding the sheet name (or a named range) forces Excel to pull the single correct master header cell on Page 1 for all pages, keeping headers consistent.

I found two formula errors in the workbook. Fixing them will correct several downstream numbers.

1) Wrong sign in Earnings from Operations (Select Financial Data, row 14, column H)
- Error: H14 = H12 + H13
- Why wrong: The Earnings from Operations formula everywhere else is Earnings = (gross margin) − (other operating (income) and expenses). Other operating numbers are negative (income), so the correct operation is subtraction (which effectively adds when the other item is negative). H14 currently adds H13 again and therefore gives the wrong result for 1997.
- Fix: Replace H14 with the same formula used in the other years:
  H14 = H12 - H13
- Effect: Corrects H14 and all later calculations that depend on it (earnings before taxes, etc.).

2) Net Earnings SUM range off by one row (Select Financial Data, row 25)
- Error: B25 (and C25:K25) = SUM(B20:B24) — this misses the cell that actually contains "Earnings before Extraordinary Item..." which is in row 21.
- Why wrong: The sheet places the pre-extraordinary earnings in row 21 (B21, C21, …). The SUM starts at row 20 (a label/blank), so the formula is omitting the main component and producing wrong Net Earnings.
- Fix: Change the Net Earnings formula to sum the correct rows:
  B25 = SUM(B21:B24)
  Then copy/drag this corrected formula across C25:K25.
- Effect: Net Earnings will include (earnings before extraordinary) + (extraordinary) + (cumulative effect) as intended.

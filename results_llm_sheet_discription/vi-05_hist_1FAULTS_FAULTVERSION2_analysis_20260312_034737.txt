Findings
- On the "2003-04 per credit hour" sheet cell H14 (Subtotal State Components for Veterinary, per term) contains =COUNT(H10:H13) instead of =SUM(H10:H13).
- COUNT returns the number of numeric cells (4), not the money total, so the Veterinary subtotals and totals on that sheet are wrong (H14, H21, H25).

Why this is wrong (short)
- COUNT counts how many numeric cells are in the range; SUM adds their values. You need a sum for monetary totals.

Fix (step‑by‑step)
1. Open sheet "2003-04 per credit hour".
2. Select cell H14.
3. Replace the formula =COUNT(H10:H13) with =SUM(H10:H13) and press Enter.
4. Verify H21 and H25 update automatically.

Corrected numbers (after change)
- H14 (Subtotal State Components, Veterinary per term) = 5,435.23
- H21 (Total Florida Resident Rate/Credit Hour, Veterinary per term) = 5,862.03
- H25 (Total Non‑Resident Rate/Credit Hour, Veterinary per term) = 15,892.53

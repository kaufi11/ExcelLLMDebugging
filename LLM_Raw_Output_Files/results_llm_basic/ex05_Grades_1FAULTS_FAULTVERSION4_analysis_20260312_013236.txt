Findings
- Error: On Exams sheet cell G19 has the wrong formula. It is =SUM(E19:F19) (it omits Exam 1).
- Effect: G19 currently = 73 + 70 = 143. The correct total should be 71 + 73 + 70 = 214.

Fix (quick)
1. In Exams!G19 replace =SUM(E19:F19) with =SUM(D19:F19)
   — or copy G6 (which is =SUM(D6:F6)) and Paste (or drag AutoFill) down to G19.

Why this fixes it
- =SUM(D19:F19) includes all three exam columns (D, E, F) for that student, restoring the consistent pattern used in the other rows and giving the correct total 214.

How to avoid / check for similar errors
- Turn on formula view (Ctrl + `) and scan column G for inconsistent formulas.
- Use Excel’s Error Checking > “Inconsistent formula” or select the G column and use Home → Find & Select → Go To Special → Formulas to spot differences.
- Convert the range to an Excel Table so totals auto-fill consistently for new rows.

Findings
- Error on the "Spring 2003 per credit hour" sheet: cell H14 contains =COUNT(H10:H13) instead of =SUM(H10:H13).
- Result: H14 shows 4 (the count of numeric cells) rather than the subtotal of State Components for Veterinary per term. That wrong value propagates to H21 (Total Florida Resident Rate) and H25 (Total Non‑Resident Rate), making those totals far too low.

Why it's wrong (short)
- COUNT returns the number of numeric cells (4), not their sum. You need SUM to add the amounts.

Fix (steps)
1. Go to the "Spring 2003 per credit hour" sheet.
2. Select cell H14.
3. Replace the formula =COUNT(H10:H13) with =SUM(H10:H13) (or use AutoSum on H14).
4. Press Enter. Excel will recalculate H21 and H25 automatically.

Corrected values (after fix)
- H14 (Subtotal State Components) = 4,706.01
- H21 (Total Florida Resident Rate/Per Term for Veterinary) = 5,104.01
- H25 (Total Non‑Resident Rate/Per Term for Veterinary) = 13,890.29

Quick validation tip
- Check that each subtotal equals the visible sum of its component rows (e.g., H14 = H10+H11+H12+H13). If a subtotal looks very small/round (like 4), it’s probably a COUNT by mistake.

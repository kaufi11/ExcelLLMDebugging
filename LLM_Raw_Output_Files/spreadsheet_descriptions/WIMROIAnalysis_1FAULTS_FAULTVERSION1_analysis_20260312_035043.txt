Formula patterns and conventions
- E (Min) typically multiplies count * hours/year * rate * lower% (e.g., *0.16 or calculated days*interest).
- G (Max) uses the upper% (e.g., *0.25, *0.44, or *4 days).
- I = average of E and G.
- K = user-selected/specific scenario: many K cells use a percentage value from a nearby C cell (e.g., C12, C23, C57) so the "Your Savings" column is driven by those input % cells.
- Many SUM rows aggregate the E/G/I/K columns for subtotals and totals.

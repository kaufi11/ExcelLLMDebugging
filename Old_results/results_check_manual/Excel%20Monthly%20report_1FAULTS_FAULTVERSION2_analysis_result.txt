Major issues (what's wrong and why)
1. Cascading single-cell SUM references across columns (rows 9–39)
   - Pattern: F9 =SUM(E9), G9 =SUM(F9), H9 =SUM(G9), … M9 =SUM(L9) (and the same pattern repeated for rows 10–39).
   - Problem: This copies the value from the first referenced cell across the whole row rather than recording independent service counts per column.    - Problem: E42 is outside your data range (rows 9–39). SUM(E42) will return 0 if E42 is blank; it looks like a typo or wrong reference. E9 should probably be a data entry or a correct reference (not SUM(E42)).

1) Row 12 — squared error (cell E12) is wrong
- Current formula in E12: =(463-D12)^2
- Why it's wrong: it uses the constant 463 instead of the observed theta in C12. That produces a huge incorrect error for row 12 (214,359.78).

- Current predicted theta formula (D12..D16) uses -B12 to get a positive magnitude (e.g. $B$5*(-B12)^$B$6). This works only if pressure values in column B are negative (as in your data).
- Recommendation (more robust): use ABS(B12) or ABS(Bn) to get magnitude explicitly and avoid errors if any pressure is zero or becomes positive:

5) LN() formulas in column F
- Current F12..F16 use =LN(-B12) etc. This is consistent with using -B (to get positive magnitudes) but will error if any B value becomes 0 or positive.
- Recommendation: use =LN(ABS(Bn)) for robustness (and handle B=0 separately if needed).

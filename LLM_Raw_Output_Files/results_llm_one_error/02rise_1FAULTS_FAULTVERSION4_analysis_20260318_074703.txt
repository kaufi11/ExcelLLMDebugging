AN94 — Error: formula uses COUNT(AN93*$E$93) instead of summing/multiplying.  
Fix: replace with =AN93*$E$93 (or =SUM(AN93*$E$93)).  
Why: COUNT returns 1 for a numeric value, so the "Total" shows 1 (wrong) instead of AN93 times the RISE cost; this breaks that row’s totals and the final aggregated totals.

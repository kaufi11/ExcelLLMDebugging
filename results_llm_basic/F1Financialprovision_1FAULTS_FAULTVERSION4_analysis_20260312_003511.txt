Summary of the error
- Sheet "B Direct Payments", cell I11 contains the value 3 (hard-coded) instead of a formula. It should be the total value of payments across age groups for "People with Learning Disabilities" (i.e. sum of F11:H11).

Why this is a problem
- A hard-coded number breaks automatic updating. If any of F11, G11 or H11 change, I11 (and the grand total I15) will not update correctly. This can give wrong totals and hide errors.

Fix (quick)
1. Select cell I11.
2. Replace the 3 with this formula:
   =SUM(F11:H11)
3. Press Enter.

What this does
- The cell will now always show the total of the three age-group payment values (F11, G11, H11) and I15 (which sums I10:I14) will update correctly.

Quick checks to avoid similar errors
- Turn on Show Formulas (Ctrl+`) to scan for other hard-coded values in total columns.
- Use Formulas → Error Checking or Trace Precedents to confirm totals reference the correct cells.
- If you want an automatic audit, use a conditional formatting rule to highlight numeric cells in total columns that are not formulas.

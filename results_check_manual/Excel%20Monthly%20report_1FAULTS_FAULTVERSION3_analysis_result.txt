I inspected the formulas you provided on Sheet1 and found multiple incorrect / inconsistent formula patterns. Summary first, then concrete examples and recommended fixes.

Summary of problems
- Many category cells (columns F:M for rows 9–39) are formulas that simply reference the previous column (e.g. F9 = SUM(E9), G9 = SUM(F9), …). That duplicates/propagates values across columns instead of using the actual category values or summing the correct ranges.
- E9 contains =SUM(E42) which points to an unrelated cell (row 42) and is almost certainly wrong.
- H19 uses COUNT(G19) while the surrounding pattern uses SUM on the previous cell; COUNT on a single cell will return 1 for any numeric value (including 0) and is inconsistent here.
- The totals row (row 40) has an off-by-one / chained pattern: after E40 = SUM(E9:E39) the rest of totals chain left (F40 = SUM(E40), G40 = SUM(F40), …) instead of summing the respective column ranges. That produces identical or incorrect column totals.
- Use of SUM() on a single cell is unnecessary (not an error by itself) but most of these single-cell SUM references reveal the larger referencing mistakes.

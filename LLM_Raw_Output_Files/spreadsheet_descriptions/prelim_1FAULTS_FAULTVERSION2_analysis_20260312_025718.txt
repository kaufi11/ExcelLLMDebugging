Overview
- Workbook has two sheets: "program" (master event timetable) and "schedule" (detailed paper/session listings).

Sheet: program
- Top metadata: A1 conference title, A3/A5/A7 co-chairs, organizers and sites (CITIDEP, MIT-DUSP; web addresses in F8).
- Column headers in row 11: A=EVENT (sequential ID), B=DAY, C=START (time), D=END (time), E=TYPE (e.g., Session, Break, Registration), F=TOPIC or speakers, G=Length (minutes), H=ROOM.
- Rows 12–41 list events across Monday, Tuesday and Wednesday with start/end times, types, topics and rooms (primary rooms: 66-110, posters 56-191, AVT room 7-431, Lobby).
- Formulas: column A mostly uses simple sequencing (e.g., A13 =A12+1, A14 =A13+1, etc.). Exceptions/oddities:
  - A15 is =A14+1 (Break-A) but has identical start/end (10:15) and is marked "Eliminated".
  - A24 uses =A22+829 (large jump) — atypical and likely a mistake if the intent was simple incrementing.
  - Several subsequent A-cells continue the +1 pattern from different starting points.
- Notable values: conference dates "MIT - 10, 11 and 12 November 2003" (A7); keynote entry (A31 row & F31 listing Joan Blades and Jane Fountain); conference dinner Tuesday 20:00 (A33); MIT guided tour Wednesday 14:45 (A41 / A119 on schedule sheet cross-reference).

Sheet: schedule
- Column headers: B=PAPERID, C=SESSIONID, E=AUTHOR(S), F=INSTITUTION, G=AFFILIATION, H=COUNTRY, I=TOPIC. Column J contains paper titles but no explicit header shown in the extract.
- Column A is used for section labels and human-readable session/time headings (e.g., "MONDAY 10", registration, "POSTERS", session start times and session names).
- Rows enumerate individual papers with PAPERID (B), SESSIONID (C, e.g., "1-1", "5-poster"), authors (E), institutions (F), country (H), topical category (I) and paper title (J).
- Rooms for session blocks are repeated in A/F cells (e.g., ROOM 66-110 for main sessions; ROOM 56-191 for posters; ROOM 7-431 (AVT) for some Wednesday sessions).
- Contains keynote and social events (keynote details in rows ~88–92; dinner in row ~94; guided tour in 119).

Data/formatting notes and potential issues
- Times in program are time-formatted (C/D columns); lengths in G are numeric minutes for several sessions but missing for some breaks and special entries.
- Column A sequencing in program relies on formulas; the large jump (A24 =A22+829) and occasional manual values (A41 =26) break the sequence — likely unintended.
- Some rows (breaks, eliminated items) have zero or missing durations and no G values.
- In schedule, J contains paper titles but no header was provided in the excerpt; column D appears unused/omitted in the snippet.
- Overall structure: use "program" as a chronological event/timeline table and "schedule" as the session-by-session paper roster with author/institution/meta details.

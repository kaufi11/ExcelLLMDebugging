Faulty cell
- Sheet "B Direct Payments", I15

Error type
- Formula / mechanical error (wrong function used)

What's wrong (short)
- I15 uses =COUNT(I10:I14). COUNT returns how many numeric cells there are (usually 5), not the sum of the payment values. You need the total value of payments.

Fix (repair formula)
- Replace with: =SUM(I10:I14)

Why this fixes it (short)
- SUM adds the payment amounts in I10:I14 to give the total payments. COUNT only counts entries, so it can't give a monetary total.

Extra check (optional)
- Ensure I10:I14 are numeric (not text). If any are text that look like numbers, SUM will ignore them — convert to numbers or use VALUE if needed.

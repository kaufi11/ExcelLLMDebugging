1) Sheet "Student-teacher"

  - Problem: COUNT returns the number of numeric cells in F45:F55 (11) rather than the sum of teachers. The value shown (11) is incorrect for a "TOTAL" of teacher counts.
  - Fix: replace with =SUM(F45:F55)

- General inconsistency: several "TOTAL" rows in this sheet are hard-coded values rather than formulas (for example the TOTAL row in the 1990–1993 block uses entered values instead of SUM formulas). 2) Sheet "university"

Notes / suggestions

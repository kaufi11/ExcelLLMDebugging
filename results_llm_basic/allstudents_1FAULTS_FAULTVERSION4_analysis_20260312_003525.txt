Findings
- One clear formula error: cell F44 (Grade 8 for Spring 1999) contains =E44/D45. It should divide passing (E44) by tested (D44).
- Effect: F44 is calculating Grade 8 percent using the Grade 10 tested count (D45), giving a falsely high percent.

Quick proof (numbers from the sheet)
- Correct percent should be 208,412 / 274,873 ≈ 0.7580 (75.80%).
- Current (wrong) percent is 208,412 / 233,657 ≈ 0.8921 (89.21%) — clearly incorrect.

How to fix (short)
1. In F44 replace the formula with:
   =E44/D44
   then press Enter.
2. To ensure no other copy/paste mistakes, put a correct formula in the first percent cell of a block (for example F10 = E10/D10) and drag the fill handle down through the block (F10:F17, F25:F32, etc.) so each row uses its own D and E cells.

Optional improvements (recommended)
- Wrap formula to avoid #DIV/0! if any D cell could be zero:
  =IFERROR(E44/D44,"")
- Format F cells as Percentage with the needed decimal places.
- Audit the sheet for other inconsistent formulas: use Home → Find & Select → Go To Special → Formulas or Formula Auditing tools (Trace Precedents) to spot any mismatched references.

Spreadsheet overview (concise)

Sheets (6)
- Intro — Title and summary of the DUCK94 program; master index table of experiments with column headers (Experiment # / PI code, Experiment Title, Investigators, Time period). Lists experiments that each have their own sheet.
- RHBJune — Experiment 2 (PI code RHB) — June deployment. Sensor inventory table (PA0, PA1, PD0–PD5) with sampling rates, record lengths, FRF coordinates, depth/height fields and data type/format. Contains a note/question in D19–D20 about number of pore-pressure sensors.
- RHBSept — Same experiment as RHBJune but for September; identical layout and fields (PA0, PA1, PD0–PD5) and the same note about sensor count.
- FRF — Experiment 21 (FRF Measurements Program). Larger inventory of pressure gauges, current meters, sound/atmospheric sensors, etc. Columns include ID, Description, Sampling rate, Record length, Interval, Cross-shore, Longshore, Depth, Bottom/Height, Gauge depth, Type, Format, Additional info.
- DOZRJune — Experiment 29 (DOZR) — June. Swash-zone pressure gauges p01–p08 plus video records. Columns similar to RHB sheets; date/timestamp entries present (e.g., 1994-07-04 00:00:00).
- DOZRSept — Same as DOZRJune but for September (p01–p08 and video lines), similar column layout.

Common layout and key columns (across experiment sheets)
- ID (sensor/gauge ID)
- From / To (date ranges or day numbers)
- Description (sensor type)
- Sampling Rate (e.g., 2 Hz, 10 Hz, 30 Hz)
- Record Length (seconds or ?)
- Interval (continuous, ? or every X hours)
- FRF Coordinates: Cross-shore and Longshore (numeric coordinates)
- Depth (m), Height above bottom / Bottom / Gauge Depth (numeric or N/A)
- Type (Time Series, Video, etc.), Format (Digital, SVHS Tape, etc.)
- Additional information (array names, tripod location, etc.)

- One FRF row (I20) uses an inconsistent constant: =SUM(482-J20) (appears to be an erroneous hard-coded reference).
- Many rows show "N/A" for height/depth where not applicable.
- No other complex formulas observed in provided content; most cells are data/text.

Overview
- Workbook has 3 sheets (2003-04 per credit hour; Spring 2003 per credit hour; Fall 2002 per credit hour). Each sheet is the same layout: tuition/fee components by program type with per-credit columns for undergraduate (col B), graduate (C), law (D) and per-term columns for professional schools: medical (F), dental (G), veterinary (H). Rows list state components, local fees, resident totals, and non‑resident additions. Footnote about medical/dental/vet being billed per academic year (two installments) appears in rows A27–A28.

Structure / key rows
- Header rows: A1–A3 titles; B5 / F5 indicate Per Credit Hour and Per Term.
- Program columns:
  - Per-credit: B = Undergraduate (0000–4999), C = Graduate (5000–9999), D = Law courses.
  - Per-term: F = Medical, G = Dental, H = Veterinary.
- State components (A9–A14): Matriculation (A10), Building (A11), Capital Improvement Trust Fund (A12), Student Financial Aid (A13). Row 14 = Subtotal State Components.
- Local fees (A15–A20): Activity & Service, Athletic, Health, Transportation Access. Row 20 = Subtotal Local Fees.
- Resident totals (A21): Total Florida Resident Rate/Credit Hour = Subtotal State Components + Subtotal Local Fees.
- Non‑resident section (A22–A25): Non‑Resident Student, Non‑Resident Student Financial Aid; Row 25 = Total Non‑Resident Rate (SUM of resident total + non‑resident rows).

Formulas
- Subtotal state: e.g., B14 = SUM(B10:B13) (same pattern in C14, D14, F14, G14, H14).
- Subtotal local: e.g., B20 = SUM(B16:B19) (and corresponding C20, D20, F20, G20, H20).
- Resident total: e.g., B21 = SUM(B14,B20) (and corresponding for other columns).
- Non‑resident total: e.g., B25 = SUM(B21:B24) (same pattern for C25, D25, F25, G25, H25).

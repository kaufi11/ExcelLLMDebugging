Cell D11 — Error: formula references wrong/detail cells. It uses =D98+D99+D102+... and omits D100 and D101 (A/R‑OFFICERS, A/R‑STAFF) and includes D98 (incorrect). Consequence: the "Accounts receivable" line in Current Assets (D11) does not match the detailed Receivables total (D114), so Total Current Assets / Total Assets won't reconcile with Liabilities & Capital.

Fix: replace D11 with a single, correct total — e.g. set D11 = D114 (or D11 = SUM(D99:D113)). Also update E11 the same way for the 2001 column.

Why: using the detailed receivables total ensures the summary line equals the detailed subtotal, removing the mismatch and allowing Assets = Liabilities + Net Assets.

4) Variance / percent-change formulas — DIV/0 risk
- Problem: Many variance % formulas are (Actual - Plan) / Plan. If Plan = 0 this produces #DIV/0! errors.
- Fix: wrap with IF or IFERROR. Example replacement:
  - Old: =(D8 - C8) / C8
  - New (recommended): =IF(C8=0, "", (D8 - C8) / C8)
  - Or using IFERROR to keep blanks on errors: =IFERROR((D8 - C8) / C8, "")
- Why: prevents #DIV/0! and shows blank (or NA, or 0) where plan is zero. Apply the pattern to all variance cells.

Summary of the error
- Sheet 1856 (Fund 1856) has H25 = 3 (hard‑coded) instead of the normal formula =SUM(H9:H24). That is a manual overwrite of the Total Revenues (Projected) cell.
- Because Summary and other calculations link to each fund's H25, that single bad value will make the Projected column (column H) totals (revenues, resources, ending balances, etc.) wrong.

Why it's wrong (short)
- All other fund sheets use H25 = SUM(H9:H24). The Summary sheet pulls each fund's H25 into its totals (e.g. Summary H14 = SUM(H9:H13) where H9 = '1003'!H25 etc). A hard coded 3 in 1856!H25 breaks those linked totals.

Fix (what to do)
1. On sheet 1856, cell H25 replace the value 3 with the formula:
   =SUM(H9:H24)
2. Press F9 (calculate) and confirm Summary H totals change.
3. Quick check: on each fund sheet (1003, 1043, 1166, 1198, 1856) verify H25 is a formula (not a constant). Use Home → Find & Select → Go To Special → Constants (Numbers) while on the fund sheets to spot accidental hard numbers in totals.

Effect after fix
- Summary column H (Projected) totals for Total Revenues, Total Net Resources, Total Expenditures and Ending Balance will update to the correct values because they pull from each fund's H25.

Optional extra checks / prevention
- Use Excel’s Inquire or Show Formulas to catch overwritten formulas.
- Protect total cells (H25, I25, etc.) or apply data validation to prevent accidental overwrites.

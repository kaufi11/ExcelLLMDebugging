Summary — two problems found

1) Wrong operator in one formula (Hospitality, cell O14)
- Error: O14 has =SUM(100-(E14-G14-I14-K14-M14)). The formula uses minus signs between the percentage cells, so it computes 100 - E14 + G14 + I14 + K14 + M14 (incorrect).
- Fix: Replace it with a subtraction of the sum, e.g.
  =100 - SUM(E14,G14,I14,K14,M14)
- Why: You want 100 minus the total of those five category percentages. Using minus signs inside the parentheses reverses the sign for all but the first term.

2) Misplaced / shifted data in normally-empty columns (extra values in F, H, Z, etc.)
- Symptom: A few rows (for example Education row 18, Business row 10, Hospitality row 14 and Engineering row 22/23/24) have numbers in columns F, H, Z that look like stray entries. The “Other” formulas (and some totals) are written to subtract only the five expected columns (E,G,I,K,M for lower level; S,U,W,Y,AA for upper level). - Fix:

- Why: The formulas expect a rigid column pattern. Data typed in the wrong column will be omitted from the 100% calculation, producing incorrect “Other” percentages and inconsistent totals.

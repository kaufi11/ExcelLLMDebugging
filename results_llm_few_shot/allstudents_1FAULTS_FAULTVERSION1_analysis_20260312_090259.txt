Faulty cell
- D60

Error type
- Range-omission (inconsistent totals)

What’s wrong (short)
- D60 =SUM(D53:D58) excludes Grade 10 (D59). But E60 =SUM(E53:E59) includes Grade 10 passing. So the All Grades percent F60 = E60/D60 uses a numerator that includes Grade 10 but a denominator that does not — the percent will be wrong (too high).

Fix (exact formula)
- Replace D60 with: =SUM(D53:D59)

Optional improvements (recommended)
- Keep the numerator/denominator ranges identical for every “All Grades” row.
- Use a safe percent formula to avoid divide-by-zero: =IF(D60=0,"",E60/D60)
- Or convert the data to an Excel Table so totals automatically adjust when rows are added/removed.

Why this fixes it (brief)
- Including D59 adds the missing tested count so totals match the passing count. Then F60 = E60/D60 computes the correct overall percent.

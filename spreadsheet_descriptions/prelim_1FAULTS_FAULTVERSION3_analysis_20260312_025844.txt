Summary of workbook

Sheets
- program — master event timetable / program header and schedule.
  - Top metadata: A1 conference title ("INTERNATIONAL CONFERENCE ON PUBLIC PARTICIPATION AND INFORMATION TECHNOLOGIES - 2003"), co-chairs and organizers (Pedro Ferraz de Abreu, Joseph Ferreira Jr., João Joanaz de Melo; organized by CITIDEP and MIT-DUSP), dates/location (MIT — 10, 11 and 12 November 2003), websites in F8.
  - Table header (row 11): EVENT (A), DAY (B), START (C), END (D), TYPE (E), TOPIC (F), Length (G), ROOM (H).
  - Rows A12–A41 enumerate individual events (registration, sessions, breaks, poster sessions, keynote, dinner, guided tour). Column G holds durations in minutes for many sessions; column H shows room assignments (main: 66-110; posters: 56-191; AVT: 7-431; guided tour: 7-Lobby).
  - Numbering in column A is automated: most cells use formulas like =A12+1, =A13+1 etc. One odd formula is A20 (=A19-1) which subtracts 1 (likely an error causing a duplicate/out‑of‑sequence event number). Some rows (e.g., A15 Break-A marked "Eliminated") have identical start and end times.
  - Key entries: Session-1..7 with topics (Public Access; PP-IT and e-Planning; PP-IT and e-Government; PP-IT and Education); Keynote (Joan Blades and Jane Fountain) at Tue 16:45–18:00; Conference dinner Tue 20:00; MIT guided tour Wed 14:45.

- schedule — detailed paper/session listing mapped to the program.
  - Column headers: PAPERID (B1), SESSIONID (C1), AUTHOR(S) (E1), INSTITUTION (F1), AFFILIATION (G1), COUNTRY (H1), TOPIC (I1). Column J used for paper titles.
  - Column A used for day/session labels and time ranges (e.g., "MONDAY 10", "Session I - Public access to information", "LUNCH BREAK", "KEYNOTE").
  - Each paper row lists PAPERID (col B), SESSIONID (col C, e.g., "1-1", "5-poster"), author(s) (E), institution (F), country (H), topic (I) and paper title (J). Rooms for sessions are noted in nearby A/F cells (e.g., ROOM 66-110; posters ROOM 56-191).
  - Key elements: complete listing of individual presentations across Sessions I–VIII and posters; keynote/conversation details (Joan Blades, Jane Fountain; Pedro Ferraz de Abreu as interviewer); conference dinner location (Restaurant in Cambridge); guided tour time/location (Lobby 7, 14:45).

Formulas and structure notes
- program column A: sequential numbering via =previous+1; maintain insertion/deletion caution since numbering is relative (one cell A20 uses =A19-1 and appears to be a mistake).
- Times are stored as time values in columns C and D (HH:MM:SS format), and durations are provided in minutes in G where filled.
- schedule uses a mix of layout labels in column A and structured paper records in columns B–J (consistent mapping of metadata: paper ID → session → author → institution → country → topic → title).

Potential data issues / items to check
- A20 formula (=A19-1) likely incorrect — will break sequential numbering.
- Some events have no duration in G (breaks, some items) — durations could be computed from START/END where both exist.
- A15 (Break-A) has identical start and end times and is labeled "Eliminated" — confirm whether to keep or delete.
- Mixed use of column A for both labels and numeric IDs in schedule may complicate filtering/sorting.

Overall layout
- program = high-level daily program with auto-numbered events, times, room and topics.
- schedule = detailed roster of papers mapped to session IDs, with authors, institutions, countries, topics and titles; includes session/day labels and logistical notes (rooms, keynote, dinner).

Findings — the two real errors (short & clear)

1) Wrong formula in Endowments (Balance Sheet)
- Location: Sheet "Table 2.0 BS - Annex E", cell H59 and I59.
- Problem: H59 = H15 and I59 = I15 (Endowments are linked to Total Income). Endowments should be a separate balance-sheet figure, not a copy of the I&E total. This will inflate Total Funds (H69/I69) and give incorrect balances/reconciliations.

Fix:
- Replace the formulas in H59 and I59 with the correct endowment values (enter the actual endowment numbers manually) or with links to the correct data source (if you have a separate endowment input cell). In other words: remove =H15 and =I15 and put the actual endowment amounts (or a proper reference cell).

Why this fixes it:
- Endowments are a balance-sheet item. Removing the incorrect link prevents double‑counting income as endowment capital so Total Funds and all downstream totals/reconciliations become correct.

2) Wrong variance % formula for Total Funds
- Location: Sheet "Table 2.0 BS - Annex E", cell K69.
- Problem: K69 is set to =K49. That simply copies the % variance of Total Net Assets instead of calculating % variance for Total Funds. This is inconsistent and can be wrong if H69 (original Total Funds) ≠ H49 (Total Net Assets) or if one of the checks fails.

Fix:
- Change K69 to the standard percent-variance formula used elsewhere:
  =IF(H69=0,"N/A",(J69/H69)*100)

Why this fixes it:
- This computes percent variance for Total Funds as (Revised − Original) / Original, which is the intended behavior and consistent with other % cells.

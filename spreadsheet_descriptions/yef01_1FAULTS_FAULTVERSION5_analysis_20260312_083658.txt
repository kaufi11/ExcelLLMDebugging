Overview

4) General State Financial Aid
- Location: Table 25.
- Columns: A College, B General State Financial Aid (dollars), C Total % of State Financial Aid (C10:C24 = Bx / $B$26), D % of Total FTEE (given decimals in D10:D24).
- Individual B amounts listed for each college (e.g., DMACC-11 B19 = 21,642,884; KCC-10 B18 = 20,391,658; IHCC-15 B23 = 12,027,969 as a formula/constant).
- Totals: B26 = SUM(B10:B24); C26 = SUM(C10:C24); D26 = SUM(D10:D24).
- Note: May not total exactly 100% due to rounding.

5) FTEE Calculation
- Location: Table 26.
- Purpose: Convert credit/contact hours to FTEE by college and compute each college’s share.
- Credit side: Cx = Bx / 24 (credit hours divided by 24).
- Non-credit side: Fx = Ex / 600 (contact hours divided by 600).
- FTEE by college: Hx = Cx + Fx (many rows include small adjustments like +0.01).
- Percent of total FTEE: Ix = Hx / $H$28 (with small manual tweaks on some rows).
- Totals row (row 28): 
  - B28 = SUM(B12:B27) (total credit hours),
  - C28 = SUM(C12:C27) + 0.02 (slight adjustment),
  - E28 = SUM(E12:E27) (total non-credit hours),
  - F28 = SUM(F12:F27) + 0.03,
  - H28 = SUM(H12:H27) - 0.02 (total FTEE),
  - I28 = SUM(I12:I27) + 0.000002.
- Notes: Several manually inserted small constants (+0.01, +0.02, -0.02, -0.000001) used to address rounding/floating totals.

6) FTEE Graph
- Location: Figure 25.
- Uses the percent-of-total FTEE values (D10:D24) from the FTEE Calculation sheet for charting.
- Includes the same source notes.

General observations / formula patterns
- Percent columns consistently compute Cx / $C$Total (or Hx / $H$28), with totals calculated by SUM ranges.
- Rounding handled via ROUND(...,4) for category percentages and via manual small adjustments (+/- 0.01, +0.02, -1, -0.000001, etc.) to force subtotals/percentages to reconcile.
- Some formulas include a leading plus sign (=+...), which is functionally equivalent to = but inconsistent style.
- Sources/figure captions repeat across sheets; asterisks denote rounding caveats.

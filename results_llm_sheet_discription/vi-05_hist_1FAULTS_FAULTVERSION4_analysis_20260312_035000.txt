Findings (short)
- Error on Fall 2002 sheet: cell H14 uses =SUM(H10:H12) and omits H13 (Student Financial Aid for Veterinary). All other subtotal formulas use H10:H13 and the other sheets are correct.

Why it's a problem
- Omitting H13 understates the "Subtotal State Components" for Veterinary (H14) by the H13 amount (219.555). That understate flows into the resident total (H21) and the non‑resident total (H25) — each is too low by the same 219.555.

Fix (exact)
- Change H14 to: =SUM(H10:H13)

Corrected numbers (Fall 2002 Veterinary column)
- H14 (subtotal state components) = 4391.26 + 46.40 + 48.80 + 219.555 = 4,706.015 (displayed rounding: 4,706.02)
- H21 (resident total) = H14 + H20 = 4,706.015 + 398 = 5,104.015 (displayed 5,104.02)
- H25 (total non‑resident) = H21 + H23 + H24 = 5,104.015 + 8,367.89 + 418.39 = 13,890.295 (displayed 13,890.30)

How to apply the fix (steps)
1. Select H14 on the Fall 2002 sheet.
2. Replace formula with =SUM(H10:H13) and press Enter.
3. Verify H21 and H25 recalc automatically. If not, press F9 to recalc.
4. Optionally, format numbers to two decimals for consistency.

Recommendation
- Scan other subtotal formulas to ensure they all sum the same component rows (10:13). Consider locking the component range (or using a named range) to prevent future omissions.

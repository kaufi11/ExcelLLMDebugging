Findings
- Error in Spring 1997 All Grades row (row 74). D74 is =SUM(D68:D73) but E74 is =SUM(E67:E73). The D-range omits Grade 3 (D67), so the denominator is too small while the numerator includes Grade 3 — the All Grades percent (F74) is therefore wrong.

Effect (numbers)
- Correct tested total (should be SUM D67:D73) = 1,770,615
- Current (wrong) tested total (SUM D68:D73) = 1,531,995
- Passing total (E74) = 1,217,643
- Wrong percent (current F74) = 1,217,643 / 1,531,995 = 79.5%
- Correct percent = 1,217,643 / 1,770,615 = 68.78%

Solution
1. Fix the tested total formula in D74 to include Grade 3:
   - Change D74 to =SUM(D67:D73)
2. Confirm F74 uses =E74/D74 and format F74 as Percentage (with desired decimal places).

Prevention tip
- For each "All Grades" row, make sure the SUM ranges for Tested (D) and Passing (E) cover exactly the same rows (Grade 3 through Grade 10). You can copy the correct pattern (e.g., =SUM(D10:D16) / =SUM(E10:E16)) down each year to avoid off-by-one mistakes.

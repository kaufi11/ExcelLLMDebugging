C19 — Error: formula uses COUNT(D19:CC19) which counts numeric cells (returns 6) instead of summing the student’s points (should be 88), so the % is wrong.  
Fix: change C19 to =SUM(D19:I19) (then set B19 to =C19/$C$6 and format as Percent). Explanation: COUNT returns how many numbers exist; SUM adds their values — use SUM to get the total points.

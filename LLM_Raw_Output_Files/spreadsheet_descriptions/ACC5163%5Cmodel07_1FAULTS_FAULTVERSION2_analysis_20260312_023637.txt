Spreadsheet: MODEL07 (single worksheet titled MODEL07 — "PUSH‑DOWN WORKSHEET & CONSOLIDATION WORKSHEET (for Chapter 7)")

Purpose
- Template for performing push‑down accounting adjustments and consolidation eliminations for Chapter 7 problems. Includes areas for analyzing excess cost elements, preparing push‑down entries, and building a multi‑column consolidation worksheet. Macros load problem data and post/eliminate entries.

Top area (rows 1–37)
- Header, file name, general instructions and cautions (how to load problem data, use macros Q and P).
- Analysis sections:
  - “ANALYSIS OF INVESTMENT ACCOUNT — EXCESS COST ELEMENTS” (rows ~15–23): labels for excess cost components (inventory, land, buildings & equipment, accum. depr., patent, goodwill). B22 shows assigned amounts with B22 formula =SUM(C22:I22).
  - “ENTRIES TO IMPLEMENT PUSH‑DOWN ACCOUNTING” (rows 24–46): debit/credit lines to be posted; many lines marked “(macro Q will post)” for automatic posting to the push‑down worksheet. Row 39 totals use C39 = SUM(C27:C38) and D39 = SUM(D27:D38).

Push‑down worksheet area (rows ~49–76)
- Title: WORKSHEET TO REFLECT THE PUSH‑DOWN BASIS OF ACCOUNTING.
- Column structure: C = Old Basis, D = Adjusting Entries Dr., E = Adjusting Entries Cr., F = New Basis.
- Repeated pattern/formula: For each balance sheet line Fxx = (Cxx + Dxx − Exx). Examples: F54 = (C54 + D54 − E54), F55 = (C55 + D55 − E55), etc.
- Balance sheet lines are present (Cash, AR net, Notes receivable, Inventory, Land, Bldg & Equip, Accum. Depr, Patent, Goodwill, totals).
- Proof of posting: D76 = D65 + D74 and E76 = E65 + E74 (sums of push‑down asset and liability/equity adjustments).

Book‑value / Basic elimination analysis (rows ~78–95)
- “ANALYSIS OF PARENT'S INVESTMENT ACCOUNT — BOOK VALUE ELEMENT” (rows ~80–85) with a SUM of equity pieces (B85 = SUM(C85:F85)).
- THE BASIC ELIMINATION ENTRY (rows 88–95): lines for eliminating subsidiary equity against Investment in Sub. Formulas pull amounts from analysis:
  - C90 = $C$85, C91 = $D$85, C92 = $E$85, C93 = $F$85, D94 = $B$85.
  - Totals C95 = SUM(C90:C94), D95 = SUM(D90:D94).
- Macro P posts this basic elimination entry to CONSOLIDATION.

Consolidation worksheet (starts row 100; main area rows 104–132)
- Title and column mapping: Columns K through U hold company/problem data and consolidation columns labeled by keys (A, B, C, D, E, F, G across K:U). Row 100 and 101 contain headings/keys.
- Balance sheet layout (rows 104–118 assets; rows 120–132 liabilities & equity):
  - For each asset/liability line the template includes: a consolidated formula area (columns B–F) and populated company data in K:U (problem data).
  - Many consolidated cells use formulas referencing the push‑down worksheet results: e.g., C105 = ($F$54), C106 = ($F$55), etc. Consolidated row formulas: F105 = (B105 + C105 + D105 − E105) etc.
  - Totals use SUM ranges: A118–F118 and K118–U118 are SUMs of their respective item ranges (e.g., K118 = SUM(K105:K117)).
- Proof of posting: D134 = D118 + D132 and E134 = E118 + E132 (consolidation level debit/credit checks).

Problem data storage (columns K:U)
- Problem data for multiple problems is stored at K105:U132 (explicit numeric sample values provided for each company/column). Row 136 notes data stored to the right in K105:U132.
- Example populated values: K105 = 55,000 (Cash for Subsidiary in one problem), K106 = 180,000 (AR), K112 = 723,000 (Bldg & equip), K117 = 120,000 (Goodwill), etc. (Full set of numbers present in K105:U132 for use by macros.)

Macros (rows ~138–184)
- Macro keys and functions:
  - A–G: load UNADJUSTED problem data into the PUSH‑DOWN (and for some macros also into CONSOLIDATION) — macros map to problems 7‑1 through 7‑7 (sources noted).
  - P: posts the basic elimination entry to the CONSOLIDATION worksheet.
  - Q: posts the unamortized excess cost reclassification entry to the PUSH‑DOWN worksheet.
  - U: erases columns C, D & E of the PUSH‑DOWN worksheet.
  - W: erases column B of the CONSOLIDATION worksheet.
  - X: erases postings made to columns D & E of the CONSOLIDATION worksheet.
- Instruction: Invoke macros via Ctrl + letter (e.g., Ctrl+Q).

Key formula patterns and references
- New basis (push‑down): F = C + D − E for each line.
- Line totals: many rows use =SUM(range) for asset, liability, equity subtotals and overall totals.
- Basic elimination entry pulls analysis totals via absolute references ($C$85, $D$85, $E$85, $F$85 and $B$85).
- Consolidation consolidated column formulas aggregate component columns and push‑down results (e.g., F105 aggregates B, C, D, and E columns per line).

Where to act
- To load sample problem data: use macros A–G (row 149–173 mapping).
- To post push‑down excess cost reclassifications: use macro Q (posts starting row 49 push‑down area).
- To post basic elimination to consolidation: use macro P (posts to consolidation starting near row 100).
- Problem data area to the right (K105:U132) contains the numeric inputs used by the sheet and macros.

Overall layout summary
- Top: instructions and analyses (excess cost & book value).
- Middle left: push‑down worksheet (old/adjust/dr/cr/new basis; proofs).
- Middle right: consolidation worksheet (company columns K:U with problem data; consolidated columns B–F aggregate and compute final totals).
- Bottom/right: macros list and controls; problem data repository K105:U132.

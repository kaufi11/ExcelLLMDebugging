Overview
- Workbook contains 3 sheets (2003-04 per credit hour; Spring 2003 per credit hour; Fall 2002 per credit hour). Each sheet is the same layout: tuition/fee components shown Per Credit Hour (columns B–D) and Per Term* (columns F–H) for program groups (Undergraduate, Graduate, Law, Medical, Dental, Veterinary).
- Top headings: A1–A3 title lines; B5 and F5 show "Per Credit Hour" and "Per Term*"; B6–H6 list program columns; B7/C7 note course number ranges.
- Footnote (A27–A28) explains medical/dental/vet assessed annually, payable in two installments.

Structure and contents (common pattern)
- Section “State Components” (starting A9) lists:
  - Matriculation (row 10)
  - Building (row 11)
  - Capital Improvement Trust Fund (row 12)
  - Student Financial Aid (row 13)
  - Subtotal State Components (row 14) — formula SUM of rows 10:13 (e.g., B14 = SUM(B10:B13))
- Section “Local Fees” (starting A15) lists:
  - Activity and Service Fee (row 16)
  - Athletic Fee (row 17)
  - Health Fee (row 18)
  - Transportation Access (row 19)
  - Subtotal Local Fees (row 20) — formula SUM of rows 16:19 (e.g., B20 = SUM(B16:B19))
- Total Florida Resident Rate/Credit Hour (row 21) — formula SUM(State Subtotal, Local Subtotal) (e.g., B21 = SUM(B14,B20))
- Non-Resident section (A22–A25):
  - Non-Resident Student (row 23)
  - Non-Resident Student Financial Aid (row 24)
  - Total Non-Resident Rate/Credit Hour (row 25) — formula SUM(B21:B24) etc.

Formulas and notable cells
- Repeated SUM formulas at rows: 14 (state subtotal), 20 (local subtotal), 21 (resident total), 25 (non-resident total) for each applicable column.
- Per-term columns (F, G, H) mirror the same structure but show large numbers (annual/per-term professional program totals).
- Inconsistencies/possible errors to check:
  - Fall 2002 sheet: H14 uses COUNT(H10:H13) (A14 shows COUNT) instead of SUM — likely a mistake.
  - Some precision/inconsistency: Fall 2002 H13 = 219.555 (extra precision), slight differences in F/G per-term decimals across sheets.
  - Column E is unused/blank in many rows.

Overall layout summary
- Three parallel sheets showing the same fee-component breakdown by program and by per-credit vs per-term amounts.
- Core formulas are straightforward SUMs to build subtotals and totals; check the H14 COUNT in Fall 2002 and minor numeric inconsistencies if exact auditing is required.

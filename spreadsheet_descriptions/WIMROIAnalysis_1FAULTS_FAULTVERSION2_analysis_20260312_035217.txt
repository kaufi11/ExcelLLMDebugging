Common formula patterns
- Labor savings: typically computed as #people * hrs/yr * rate/hr * savings% (e.g., E9 = C9*C10*C11*0.16; G9 uses 0.25).
- Averages: Ixx = (Exx + Gxx) / 2.
- “Your Savings” column Kxx uses a chosen savings percentage or dollar-per-item from column C (e.g., K9 = C9*C10*C11*C12).
- Subtotals: SUM() over the block’s E/G/I/K cells.
- Summary mapping: summary tables pull individual E/G/I/K cells into consolidated rows (e.g., C210 = E9, E210 = G9, G210 = I9, I210 = K9).

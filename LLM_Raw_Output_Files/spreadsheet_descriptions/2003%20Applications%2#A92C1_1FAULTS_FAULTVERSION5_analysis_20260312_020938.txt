Overview
- Workbook contains 6 primary sheets: Municipal, Non Profit, County, Rehab, HOME, AHTF. Each sheet uses a consistent layout: A = Applicant, B = Project, C = Code, D = Description, E = Request (dollar amount), F = Recommend (dollar amount or formula).
- Key pattern: column E holds the requested amounts; column F holds recommended awards (often a formula referencing the request or a fixed number).

Sheet-by-sheet summary (concise)

1) Municipal
- Rows: header in row 1, applicants in rows 2–21, totals in row 22.
- Total formulas: E22 = SUM(E2:E21), F22 = SUM(F2:F21).
- Notable items: several large requests (e.g., Upper Dublin E18 = 750,741.85; West Conshohocken E20 = 544,539.50; New Hanover E11 = 519,812.90).
- F column: many rows use F# = E# (copy of request); some F cells are blank or set to specific amounts (e.g., F5 = 730, F8 = 190,000, F11 = 190,000).  

2) Non Profit
- Rows: header row 1, applicants rows 2–21, totals at row 22; additional ESG applicant at row 27 and ESG totals in row 28.
- Total formulas: E22 = SUM(E2:E21), F22 = SUM(F2:F21). ESG totals: E28 = SUM(E27), F28 = SUM(F27).
- Mixed recommendations: some F cells equal E, some are 0, some have partial awards (e.g., Community Housing Services F5 = 49,000; Fair Housing F6 = 70,000).
- Note: an extra section labelled "2003 ESG Non-Profit Application" for the Salvation Army (row 27).

3) County
- Contains county-level applicants (rows 3–5) and county total in D6/E6/F6.
- Total formulas: E6 = SUM(E3:E5), F6 = SUM(F3:F5).
- Inter-sheet summary links (located D9–D13 / E9–E13):
  - D9/E9: Total Municipal Applications -> E9 formula = Municipal!E22
  - D10/E10: Total Non-Profit Applications -> E10 formula = 'Non Profit'!E22
  - D11/E11: Total Non-Profit Rehabilitation Applications -> E11 formula = Rehab!E8
  - D12/E12: Total County Applications -> E12 = E6
  - D13/E13: Total Requested -> E13 = SUM(E9:E12)
- County acts as the summary sheet aggregating other sheets' totals.

4) Rehab
- Rows: header row 1, applicants rows 2–7, totals at row 8.
- Totals: E8 = SUM(E2:E7), F8 = SUM(F2:F7).
- Many rehab entries also appear on Non Profit (e.g., CADCOM, Community Cupboard) — Rehab!E8 is used by County sheet.

5) HOME

- F8/F11 likewise sum recommendations.
- E8 = SUM of HOME project requests; E11 includes HOME admin (E10 = 159,490.1).
- Important link: AHTF sheet references HOME!E11 (AHTF D19/E19).

6) AHTF (Affordable Housing Trust Fund)
- Applicants rows 2–13; E14 = SUM(E2:E13) (TOTAL AHTF FUNDING REQUESTED).
- Continuum of Care match row: E16 = 2432890 * 0.15 (formula shown).
- E17 is an additional AHTF project (Genesis rehab 300,000). E18 = SUM(E14:E17) (Total AHTF).
- E19 = HOME!E11 (Total HOME funding requested). E20 = SUM(E18:E19) (Total HOME/AHTF).

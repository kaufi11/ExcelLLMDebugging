Ambiguous operator-precedence formulas in column N
- Examples: N10 =17+5/2, N15 =18+5/2, N28 =17+5/2, N30 =18+5/2, N32 =13+5/2, N34 =17/2+2, etc.  
  Problem: Excel evaluates division before addition. So 17+5/2 = 17 + (5/2) = 19.5. If you intended (17+5)/2 use parentheses.

How to fix:
- If you intended to add then divide (average of two numbers): use =(17+5)/2.
- If you intended add and a half-credit (i.e., 17 + 5/2), leave it as is, but confirm intention.

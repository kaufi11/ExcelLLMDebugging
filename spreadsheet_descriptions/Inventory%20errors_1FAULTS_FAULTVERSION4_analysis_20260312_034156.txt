High-level summary
- File is a 3-sheet comparative accounting example (Sheet1, Sheet2, Sheet3). Each sheet shows a 3‑year Income Statement (Sales → COGS → Gross profit → Operating expenses → Net income) side‑by‑side with a Balance Sheet (Assets, Liabilities, Equity) for years 2001, 2002, 2003.
- Layout: column A contains labels; the three years are presented in repeating column pairs across the sheet (2001 in B/C, 2002 in D/E, 2003 in F/G). Numeric values and formulas are placed in those year columns. Totals use SUM formulas.

Sheet-by-sheet details

Sheet1 — “Inventory error example”
- Purpose: starter worksheet demonstrating how changing inventory affects COGS and net income; intentionally does not balance.
- Income statement rows: Sales (C6/E6/G6 = 440,000 / 510,000 / 460,000), Purchases (B9/D9/F9 = 220,000 / 180,200 / 167,200), Operating expenses (C14/E14/G14 = 60,000 / 85,930 / 80,060).
- Key formulas:
  - Cost of goods available = beginning inventory + purchases (e.g., B10 = B8 + B9).
  - COGS = cost available − ending inventory (C12 = B10 − B11; analogous for other years).
  - Gross profit = Sales − COGS; Net income = Gross profit − Op. expenses.
- Balance sheet:
  - Inventory on the balance sheet is linked to the ending inventory (B21 = B11; D21 = D11; F21 = F11).
  - Total assets, total liabilities and total equity are SUM formulas (e.g., C24 = SUM(B20:B23); C35 = SUM(C29:C34)).
  - Retained earnings are linked to that year’s net income (B33 = C15, D33 = E15, F33 = G15).
- Problem: Sheet1’s balance sheet does not balance because an inventory error (wrong beginning/ending amount) was not adjusted through retained earnings — assets change but equity hasn’t been corrected for the prior‑period error.

Sheet2 — shows the raw errors (7‑4B example)
- Ending inventory intentionally altered by formula:
  - 2001 ending inventory B11 = 179,800 + 17,306 (introducing an error increase).
  - 2002 ending inventory D11 = 147,200 − 25,000 (introducing an error decrease).
- All other formulas are analogous to Sheet1; inventory on the balance sheet still links to these (so totals are out of balance).
- Purpose: demonstrates the out-of-balance amount equals the inventory error difference.

Sheet3 — corrected (prior-year error closed to retained earnings)
- Ending inventory formulas:
  - 2001 B13 = 179,800 + 17,000 (slightly different number used here to illustrate example).
  - 2002 D13 = 147,200 − 25,000; 2003 F13 = 203,970.
- Key correction: retained earnings incorporate the prior‑year inventory adjustment:
  - D35 = E17 + 17,000 and F35 = G17 − 25,000 (adjusts retained earnings for the prior period errors).
- Result: Balance sheet totals (Total assets vs. Total liabilities & equity) match because prior‑period inventory errors have been closed to retained earnings.

Key formulas and links to note
- Beginning/ending inventory and purchases drive Cost of Goods Available and COGS:
  - CostAvailable = BeginningInventory + Purchases
  - COGS = CostAvailable − EndingInventory
- Balance sheet inventory = ending inventory cell (so income statement ending inventory drives asset amount).
- Retained earnings = net income (and in Sheet3, adjusted for prior-year inventory errors).
- Totals use SUM ranges for assets, liabilities, equity, and T/L&E.

Conclusion / how to fix the imbalance
- If a prior‑period inventory error exists, you must adjust retained earnings (or restate prior period) so equity reflects the correction. Sheet3 demonstrates closing the prior-year error into retained earnings so the balance sheet balances.

Overview
- Workbook contains 11 sheets, each for a University of Hawaii campus (HawCC, HonCC, KapCC, KauCC, LeeCC, MauCC, UHH, UHM, UHWO, UHCWH, WinCC).
- All sheets follow the same layout: A1 = "University of Hawaii", A2 = report title, A4 = campus name.
- Row 7 contains main column headers: Database Name (A7), Searches (B7), Articles Viewed (C7). Row 8 splits "Articles Viewed" into Full‑text (C8) and PDF (D8). Column E is Total (E8).
- Rows 10–28 list individual databases (Academic Search Premier, ERIC, MEDLINE, etc.) with numeric values in:
  - B = Searches
  - C = Full‑text article views
  - D = PDF views
  - E = per‑database Total calculated with E# = SUM(C#:D#)
- Row 29 is the sheet total: B29 = total Searches, C29 = total Full‑text, D29 = total PDF, E29 = formula =SUM(C29:D29) (Total = Full‑text + PDF).

Formulas and calculation notes
- Per database totals use E(row) = SUM(C(row):D(row)). Sheet totals in column E use E29 = SUM(C29:D29).
- Column B (Searches) is not included in the E (Total) formulas — Totals are article views only (Full‑text + PDF).
- Anomaly: KauCC cell E20 uses =SUM(D20:D20) (only the PDF cell) instead of =SUM(C20:D20); this will understate Image Collection total on that sheet.

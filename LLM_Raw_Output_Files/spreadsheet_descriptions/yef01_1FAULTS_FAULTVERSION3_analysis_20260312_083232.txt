Sheet: FTEE Calculation
- Purpose: Compute FTEE by college from credit/contact hours and derive percent of total.
- Structure:
  - Credit conversion: C = B / 24 (credit hours ÷ 24 → credit FTEE).
  - Non-credit/contact conversion: F = E / 600 (contact hours ÷ 600 → non-credit FTEE).
  - College total FTEE H = C + F (some rows add +0.01 or omit; a few rows use slightly different adjustments).
  - I = H / $H$28 gives percent of total FTEE.
- Important formulas/behaviors and anomalies:
  - Many rows: Cxx = Bxx/24; Fxx = Exx/600; Hxx = Cxx + Fxx (+0.01 on several rows).
  - Row ILCC-03: F14 = -E14/600 (negative sign — possible data entry anomaly).
  - Totals row (row 28):
    - B28 = SUM(B12:B27); C28 = SUM(C12:C27) + 0.02; F28 = SUM(F12:F27) + 0.03.
    - H28 = SUM(H12:H27) - 0.02; I28 = SUM(I12:I27) + 0.000002.
    - Several small manual adjustments (+0.02, +0.03, -0.02, +0.000002) applied to totals—likely to force rounding/percent totals to reconcile.
  - Some percent cells subtract 0.000001 in I20/I21.
- Notes: Footnote about rounding; source citation. J28 = * sign; A30 note about percent rounding.

Sheet: FTEE Graph
- Purpose: Presentation/figure of FTEE percent distribution.
- Key content: Column D10:D24 contains the percent-of-total FTEE values (matches D column percentages shown on FTEE Calculation sheet). Source and caption rows present.

- Percent columns are computed as value / sheet total (absolute references used like $C$16, $C$20, $H$28).
- Totals computed with SUM; many small manual arithmetic adjustments (e.g., +/-1, +/-0.01, +/-0.02, +0.03, tiny additions/subtractions) are present — likely to address rounding or display constraints.

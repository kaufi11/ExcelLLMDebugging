Faulty cell
- N24

Error type
- Mechanical / sign typo (formula subtracts instead of sums)

What’s wrong (short)
- N24 = B24 - C24 - D24 - ... - M24 so it produces a negative/incorrect total. It looks like a typing or copy/paste error.

Fix (quick)
- Replace N24 with a sum formula, e.g.:
  =SUM(B24:M24)

Why this fixes it (short)
- SUM(B24:M24) adds all week values in that row (B→M). The current formula subtracts each column after the first, so it returns the wrong sign and magnitude.

Extra recommendation (optional)
- Use consistent range SUM formulas for all N cells (e.g. N4 = SUM(B4:M4), N5 = SUM(B5:M5), etc.) to avoid manual +/− mistakes. You can copy N4 down for consistency.

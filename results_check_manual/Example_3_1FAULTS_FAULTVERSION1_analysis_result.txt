1) Wrong function used in Q4 Homework H5
- Cell: Q4 Homework!H5
- Current formula: =COUNT(B5:G5)
- Problem: COUNT returns the number of numeric entries (6), not the sum of the assignment scores. All other rows use SUM to get the student’s total.

- Effect: will change the value from 6 → 273 (81+21+25+25+50+71).

2) Homework weighting formula duplicates the wrong column (D) instead of using Homework (E)
- Affected cells: Q4 Summary!F3 and Q4 Summary!F4:F12 (every student/row except the header F2)
- Current pattern (example F4): =B4*0.3+C4*0.1+D4*0.4+D4*0.2
- Problem: The last term uses D (Projects) twice. It should use E (Homework) multiplied by the 0.2 weight. Because some rows have different values in D and E, these weighted scores are incorrect.
- Correct formula pattern (row 4 example): =B4*0.3 + C4*0.1 + D4*0.4 + E4*0.2

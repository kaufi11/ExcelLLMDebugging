Cell: Formulas!C18

Error: contains a hard-coded value (207) instead of the formula linking to the COGS schedule (=C34). This causes Gross Margin, operating income, taxes, net income and other downstream figures to be wrong.

Solution: Replace the value in Formulas!C18 with the formula =C34 (or =Example!C34 if referencing the other sheet). After fixing, C18 will show the correct COGS (105,000) and all dependent calculations will update automatically.

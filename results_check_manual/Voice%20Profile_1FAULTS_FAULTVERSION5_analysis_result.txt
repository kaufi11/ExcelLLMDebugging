Summary of the most important issues
- Several weighted-average formulas in the FINANCIAL sheet ignore the third data column (column G) and therefore do not match the totals (columns I or the pattern used elsewhere). That produces inconsistent results when data exists in all three source columns (C, E, G).

- Several "total" cells on the FINANCIAL sheet reference only one source column (usually G) instead of summing all three.
- The Example sheet contains a few formulas referencing blank/incorrect cells (likely copy/paste errors) that produce incorrect or meaningless results.

Other notes / best practices
- Where totals use all three columns (C + E + G) you should base weighted averages on the same set (or use the computed I totals as denominator) to avoid overlooking one source.
- To avoid repeating the same summations and to reduce errors, consider:
  - Use the Ixx total cells as denominators (e.g., =IF(I14<>0, (C14*C16+E14*E16+G14*G16)/I14, "") ). That reduces risk of leaving out G.
  - Or use named ranges or a consistent formula template and copy it across blocks.
- Any formula that references a header cell (like C2) is almost certainly a typo — fix those.

Summary (concise)

Workbook contains 3 sheets: Summary Info, Participant List, and Speakers & Mentors. Layout, key tables, formulas and important values are listed below.

1) Sheet: Summary Info
- Purpose: high-level workshop recruitment summary and invitation breakdowns by NSF directorate and Carnegie classification.
- Top narrative (A1:A9): key stats and context
  - 1387 (~85%) CAREER awards (00–03) at Research Extensive (RE*) institutions (A1)
  - 238 (~15%) of 1625 CAREER awards identified as HBCU &/or non-RE (candidate pool) (A2)
  - 193 invited (~80% of eligible) with expected response ~1-in-4; actual interest: 66 interested, 21 "would like to attend but can't" (A7–A9)
  - Note: only 61 participants could be accommodated; 5 on waiting list (J15–J17)
- Directorate table (headers row 11: Directorate | would like to attend | invited | eligible invitees)
  - Rows A12:A19 list OPP, EHR, SBE, GEO, BIO, CISE, ENG, MPS with numeric values in B12:B19, C12:C19, D12:D19.
  - Totals row:
    - B20 = COUNT(B12:B19) — counts numeric “would like to attend” entries (counts number of directorates with numeric entries)
    - C20 = SUM(C12:C19) — result = 193 (total invited)
    - D20 = SUM(D12:D19) — result = 238 (total eligible invitees)
- Carnegie classification table (A22, columns E–G):
  - Rows E23:E29 (would like to attend) and F23:F29 (invited), G23:G29 (eligible invitees) by institution type (e.g., BCLA, MCU-1, DRU-i, R1, HBCUs).
  - Totals row:
    - E30 = SUM(E23:E29) — result = 66 (interested)
    - F30 = SUM(F23:F29) — result = 193 (invited)
    - G30 = SUM(G23:G29) — result = 238 (eligible)
- Notes / comments in column J provide context and constraints (e.g., data source, space limit).
- Footnote link to Carnegie definitions at A33.

- Header row (row 2) columns:

- Content:
  - Rows 3–63: 61 participant rows with full project metadata (dates sometimes include timestamp), award amounts in J, PI contact and institutional address in L/M.
  - Waiting list block begins at A68 with additional candidate rows (A68–A74 and further).
  - Definitions/legend for Carnegie codes in B80:B86 (BCLA, MCU-1, DRU-i, M, R1s, R1).
- Data types: text fields (PI, title, institution), numeric (award amounts), dates (H/I — some stored with time component), and categorical institution/classification codes.

3) Sheet: Speakers & Mentors
- Simple contact list with columns:
  - A: Name, B: Affiliation, C: Position, D: e-mail
- Organized into sections (Organizing Committee, Dinner Speaker, Administration, Education Speakers, Mentors).
- Rows A4–A34 list organizers, invited speakers, mentors and their emails.

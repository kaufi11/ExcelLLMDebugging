Summary — I found two errors.

1) Faulty cell: Q4 Homework!H5
- Error: Wrong function (mechanical). H5 uses =COUNT(B5:G5) so it returns the number of numeric cells (6) instead of the student’s total points.
- Fix: Replace with a sum: =SUM(B5:G5)
- Why: COUNT only counts cells containing numbers; SUM adds their values (the total points you want).

2) Faulty cells: Q4 Summary!F3:F12
- Error: Wrong cell referenced (logical). Each weighted-score formula ends with +D#*0.2 (re-using Projects) instead of using the Homework column E#*0.2. That double-counts Projects and ignores Homework.
- Fix (recommended, drag/fill down): in F3 enter
  =B3*$B$2 + C3*$C$2 + D3*$D$2 + E3*$E$2
  then fill down F3:F12.
  (Or explicitly: =B3*0.3 + C3*0.1 + D3*0.4 + E3*0.2)
- Why: The last term should use the Homework score (column E) with the homework weight. Using the weights in row 2 with absolute row references makes the formula easier to copy and less error-prone.

Quick tip: After fixing, fill/copy the corrected formulas down instead of typing each one to avoid similar slip-ups.

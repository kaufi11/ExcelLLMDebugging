Error:
In the DATA TEXT sheet, cell C13 has the formula =VLOOKUP(D14,sort0,2,FALSE), but it should reference D13 (the current row's Tree Species) instead of D14. This causes a mismatch and wrong lookup for "Conifer - Other conifers".

Explanation:
- VLOOKUP is used to find the matching Tree Species in the named range "sort0" and return the corresponding value.
- Using D14 in C13 means it looks up the species in the next row, not the current row, causing incorrect data retrieval.

Solution:
Change the formula in C13 from:
`=VLOOKUP(D14,sort0,2,FALSE)`
to
`=VLOOKUP(D13,sort0,2,FALSE)`

Explanation of solution:
- By referencing D13, the formula correctly looks up the Tree Species in the same row, ensuring accurate data matching and retrieval.

Check for similar errors:
- Verify other VLOOKUP formulas to ensure each references the correct row's Tree Species cell. In this case, only C13 has this error; others correctly reference their own row.

Fix the incorrect cell reference in C13's VLOOKUP formula to match the current row's Tree Species for accurate data lookup.

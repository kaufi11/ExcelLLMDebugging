F56 — Error: formula uses COUNT(F45:F55) so it returns the number of numeric cells (a count) instead of the total number of teachers.  
Fix: replace with =SUM(F45:F55) (or =SUBTOTAL(9,F45:F55) if you need it to respect filters).  
Why: COUNT only counts how many cells contain numbers; SUM adds their values — use SUM to get the correct total.

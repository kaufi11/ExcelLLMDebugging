pricing (named sheet)
- Layout:
  - Left: decision-variable table (A3:K10) with columns for Ht (# hired), Lt (# laid off), Wt (workforce), Ot (overtime), It (inventory), St (stockout), Ct (subcontract), Pt (production), J = demand, K = price.
  - Right (M1:P?) contains constraint-check formulas (workforce, production, inventory, overtime).
- Initial/period values:
  - Period 0: workforce 80, inventory 1000.
  - Periods 1–6 show a plan with workforce ~64.5833 each month, production Pt = 2,583.3333 units each month, price = 40.
  - Inventories and one stockout in period 4 (stockout = 266.6667) appear; inventory values vary per month.
- Key formulas (per period):
  - Workforce balance (M#): = Dn - D(n-1) - Ht + Lt
  - Production check (N#): = 40*Wt + (Ot/4) - Pt  (40 = productivity factor: 8 hrs/day * 20 workdays / 4 labor-hours-per-unit)
  - Inventory balance (O#): = previous inventory - stockout + Pt + subcontract - demand - inventory change + stockout (formula written as +Fprev - Gprev + I + H - J - F + G)
  - Overtime relation (P#): = -Ot + 10*Wt  (seems to enforce a limit or relation)
- Costs section (A12:I20):
  - Per-period cost rows compute hiring, layoff, regular-time labor, overtime, inventory holding, stockout penalty, subcontracting, material cost.
  - Examples:
    - Hiring cost: = Sheet1!$B$19 * # Hired (hiring cost/worker * hires)
    - Regular time: = Dn * Sheet1!$B$22 * 8 * 20  (workforce * regular hourly wage * 8hrs/day * 20 days)
    - Material cost: = Pt * Sheet1!$B$16
  - Totals:
    - Total Cost (C22) = SUM(B15:I20)
    - Total Revenue (C24) = SUMPRODUCT(J5:J10, K5:K10) — demand * price
    - Profit (C25) = Total Revenue - Total Cost
- Note: some cost formulas show anomalies (e.g., C15 = 519*Sheet1!$B$20) — likely typos (see inconsistencies below).

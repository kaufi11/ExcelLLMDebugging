I checked the workbook formulas. Two problems stand out — one in "Early or Late" that breaks the early-saver balance chain, and several in "Vanguard" that are not real formulas but literal text. Details and suggested fixes below.

1) Sheet "Early or Late" — broken chain at row 27
- Problem: Cell B27 contains the literal formula/value "=460" (i.e. a hard-coded 460) instead of continuing the chain from the prior end balance. Up through row 26 the early-saver Beginning Balance cells follow Bn = En-1 (B5 = E4, B6 = E5, ...). B27 breaks that by not referencing E26 and therefore the subsequent rows (B28, B29, …) follow a tiny sequence instead of continuing the correct large balance growth.
- Effect: All early-saver balances from row 27 downward are wrong (far too small) because they depend on B27. This is why the early-saver End Balances after row 26 are unrealistically small.
- Fix: Set B27 = E26 (formula: =E26). After that B28 should be =E27, B29 =E28, etc. (Many of those are already set to =Eprev; only B27 is the incorrect literal.)

Notes on related cells:
- Many D formulas on the early side reference C cells (e.g. D19 = (B19+C19)*0.06). After row 18 the deposits (C19:C48) appear intentionally blank (no more contributions). Blank cells are treated as zero in arithmetic, so those D formulas are OK — just be aware that the blank C cells are intentional (and the Cash Outlay sum C4:C48 = 15,000 confirms only the first 15 deposits exist).

2) Sheet "Vanguard" — text strings instead of actual formulas
- Problem: Cells D15:D21 contain text (strings that look like formulas or manual arithmetic) rather than real cell-reference formulas. Examples:
  - D15 value is the string "= Ending Balance - Beginning Balance"
  - D16 value is the string "= $23,833.35 - $10,000"
  - D17 value is "= $13,833.35"
  - D18 value is the string "= (Ending Balance - Beginning Balance) / Beginning Balance"
  - D19/D20/D21 are also strings/numeric strings
- Effect: These cells do not compute from the sheet values. They will not update if the data in the table changes (e.g., if Ending Balance or Beginning Balance changes).

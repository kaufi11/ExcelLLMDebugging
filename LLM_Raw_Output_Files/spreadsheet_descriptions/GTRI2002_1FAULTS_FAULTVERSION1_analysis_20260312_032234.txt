Summary of workbook structure and contents

Overall
- Purpose: Tracking Toxic Release Inventory (TRI) releases (air, land, water, offsite disposal) in millions of pounds by year and reporting transfers offsite (recycled, energy recovery, other treatment, incineration) and zip-code breakdowns for Silicon Valley / Joint Venture comparisons.
- Repeated pattern: yearly rows (late 1980s–2000) with numeric data and many formulas that (a) convert raw pounds to millions by dividing by 1,000,000, (b) sum columns (SUM ranges), and (c) auto-increment year (A cell = previous A + 1).

Sheet: 2003 Report
- Layout: Header (A3 title). Two main blocks:
  - Releases/Disposal block (columns A–F): A = Year, B = Air, C = Land, D = Water, E = Offsite Release/Disposal, F = Total Releases (sum of B:E).
  - Transfers Offsite block (columns J–O): J = Year, K = Recycled, L = Energy Recovery, M = Other Treatment, N = Incineration, O = total (sum K:N).
- Data: Years 1987–2000 in rows 7–20. Example key values:
  - 1987 total releases F7 = 13.239865 mil lbs; transfers total O7 = 14.847612.
  - 2000 total releases F20 = 4.790791 mil lbs; transfers total O20 = 23.790164.
- Notes: Row 2000 includes extra cells G20 and H20 (8.449074 and 0.6381540899) — likely auxiliary values or intermediate calculations.

Sheet: 2002 Report
- Header: Marked "IGNORE THIS SHEET" but contains full dataset and many formulas.
- Purpose: Converts raw pounds (columns F,G,H etc.) to millions (columns B,C,D) and computes subtotals and comparisons.
- Key formula patterns:
  - B11 = F11/1,000,000; C11 = G11/1,000,000; D11 = H11/1,000,000.
  - E11 = SUM(C11:D11); R11 = B11 + C11 + D11.
  - Year auto-increment: A12 = A11 + 1, etc.
  - R24 = (R11 - R23) / R11 (a percent change).
- Disposed / Recycled / Incinerated Offsite section with many NA placeholders for early years; numeric values begin later (e.g., F30 = 2,074,260).
- Joint Venture zip-code block (A43 onward): zip columns (B:K) with totals L = SUM(B:K)/1,000,000 and comparisons:
  - M = SVEP sum (references other sheet ranges), N = L - M (difference), O = N / L (fraction), and R = N * 1,000,000 in some rows.
- Several raw source figures in F/G/H columns used to derive million-lb values.

Sheet: 1999 Report
- Structure nearly identical to 2002 Report: raw pounds in F/G/H converted to millions in B/C/D, year auto-increment, and a Disposed/Recycled block.
- Contains year rows 1987–1999 and raw totals (e.g., F19 = 2,604,947; G19 = 2,127,676; H19 = 195,721).

Sheet: data
- Simple table for recycled/incinerated totals by year (1991–1999).
- Columns: A = Year, B = incinerated (raw pounds), C = recycled (formula = E - B), E = total recycled+incinerated (raw pounds).
- Example: 1999 E12 = 17,529,363; B12 (incinerated) = 4,002,596; C12 computed = 13,526,767.

Sheet: notes
- Duplicate / notes version of incinerated/recycled numbers for 1997–1999 (same values as in data).

Key formulas & patterns across workbook
- Converting pounds to millions: cell_in_millions = raw_pounds / 1,000,000 (consistent on 2002/1999 sheets).
- Year generation: A[row] = A[row-1] + 1.
- Totals: SUM across columns (SUM(B:K), SUM(C:D), etc.).
- Difference and percent comparisons: difference = total_A - total_B; percent = difference / total_A.
- Many cells are raw constants (source totals) while visible reporting columns are formula-driven conversions or sums.

Notable data points / trends
- Total releases (column F in 2003 Report) decline from ~13.24 mil lbs (1987) to ~4.79 mil lbs (2000).
- Offsite transfers (column O in 2003 Report) increase markedly: e.g., O7 = 14.85 (1987) → O19 = 26.28 (1999) and O20 = 23.79 (2000), driven by large recycling/energy-recovery figures in later years (K/L columns).
- 1999 shows especially large offsite recycling/energy recovery totals (K19 = 13.93 mil lbs; L19 = 5.73 mil lbs).

Key lookup tables
- R2:AG3 — weight thresholds and corresponding Agility Mod row (used by HLOOKUP for Agility Mod).
- R5:AW15 — a multi-row attribute table (Attr. Value, Str HP Mod, End HP Mod, End Heal Rate, Int HP Mod, Int Dmg Mod, Det Hidden, Det Danger, Agl HP Mod, Agl Dmg Mod, Agl Acc Mod, …). This table is referenced repeatedly via HLOOKUP to pull appropriate modifiers for given attribute values.
- R2:AG4 and similar ranges supply base HTH damage mappings.

Important formulas (cells called out)
- Basic Hits (E15) = ROUNDUP(B15/50,0) — determines basic hit units from Weight.
- Agility Mod (H15) = HLOOKUP(B15, R2:AG3, 2, TRUE) — picks agility modifier based on weight.
- React. Mod (G18) = HLOOKUP(B18, R17:AN18, 2, TRUE) — reaction modifier from Charisma table.
- Hit Mod breakdown (B19–E19) = HLOOKUP of Strength, Endurance, Intelligence, Agility against R5:AW15 with different row indices; F19 = PRODUCT(B19:E19) multiplies those factors; Hit Points (H19) = (F19 * E15) + 0.49 — HP derived from product of modifiers × Basic Hits (plus 0.49).
- Dmg. Mod. (B20) = SUM of two HLOOKUPs (Int and Agility damage modifiers pulled from R5:AW15).
- Healing Rate (G20) = E15 * HLOOKUP(G16, R5:AW15, 4, TRUE) — base healing scaled by Endurance.
- Accuracy (B21) = HLOOKUP(B17, R5:AW15, 11, TRUE) — agility-based accuracy.
- Power (G21) = SUM(B16:B17, G16:G17) — summed primary stats (Str + Agil + End + Int).
- Carrying Capacity (C22) = ((B16/10)^3 + (G16/10)) * (B15/2) — capacity uses Strength, Endurance and Weight.
- Base HTH Damage (H22) = HLOOKUP(C22, R2:AG4, 3, TRUE) — looks up hand-to-hand damage based on carrying capacity (or computed C22).
- Movement, detection and other derived fields use HLOOKUPs to extract values from the attribute tables (Det. Hidden E25, Det. Danger G25, Det. Hidden row A25 references HLOOKUP on G17).
- Inventing/skills: Inventing Points (C26) = (G17/10) * H4; Inventing (B27) = G17 * 3.

Sheet name and header
- Master Project - OSURF. Title in C1: "Project Summary". As-of date in C2: "As of November 24, 2003".

Layout and columns
- Columns A–J hold the record attributes:
  - A: Principal / Investigator (principal rows are header-like; project rows show investigator or blank under principal)
  - B: Org#
  - C: Project Title / description (often wrapped over multiple rows)
  - D: Sponsor
  - E: Project (ID)
  - F: End Date (dates/timestamps)
  - G: Budget (planned amount)
  - H: Expended / To Date
  - I: Commit (committed amounts)
  - J: Balance (calculated)

Rows and grouping
- Principals appear in column A on group header rows (e.g., HOJJAT ADELI at A7, TARUNJIT S BUTALIA at A12, etc.). Under each principal are one or more project rows with title fragments in column C.
- Many long titles use multiple wrapped rows in column C.

Formulas and calculations
- Balance column (J): majority use Jn = Gn - Hn - In (e.g., J7, J9, J12, …).
- Notable exception: J41 uses J41 = G41 + H41 + I41 (appears to be an error).
- Totals section near row 217:
  - A217 "Total Budget": G217 has =SUM(G7:G216)
  - A218 "Expended to-date": H218 has =SUM(H9:H216) (note range starts at H9, not H7)
  - A219 "Commitments": I219 has =SUM(I10:I217) (note starts at I10 and ends at I217)
  - A220 "Balance": J220 has =G217 - H218 - I219
- Inconsistent sum ranges (G sum starts at row 7; H and I sums start at row 9/10) may omit some rows or double-count depending on layout.

Data quality / notable values
- Several projects are over-expended (negative balances) — examples:
  - HOJJAT ADELI: G7 50,000; H7 50,247.60 → J7 ≈ -247.60
  - KEITH W BEDFORD: G24 65,484; H24 168,746.97 → large negative balance
  - C K SHUM: G124 101,972; H124 101,987.92 → small negative
- Some projects have zero budget (G59 = 0) or zero spend/commit.
- Commitments (column I) populated for some projects (e.g., I46 = 72,247; I145 = 89,401.54).

Issues to review
- Incorrect balance formula at J41 (uses addition instead of subtraction).
- Inconsistent SUM ranges for totals (G uses G7:G216; H uses H9:H216; I uses I10:I217). These should be aligned to ensure totals match row ranges.
- Wrapped/ multiline titles can shift visual alignment—verify that each project's numeric cells align with its title row.

Summary
- Single sheet listing projects by principal with columns for org, title, sponsor, project ID, end date, budget, expended, commitments and calculated balance. Main calculations are per-row balances (G-H-I) and sheet-level sums (total budget, expended, commitments, balance). Check the two formula issues above to ensure accurate totals.

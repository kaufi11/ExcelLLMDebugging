Spreadsheet overview (sheet: "Pricelist")
- Single sheet named "Pricelist" with a simple table in A1:E429 (row 1 = headers; rows 2–429 = 428 book records).
- Columns:
  - A: Subject (category text)
  - B: ISBN (text)
  - C: TitleName (book title text)
  - D: MEEC Price (calculated)
  - E: USPrice (numeric, retail price)

Formulas and logic
- Column D contains formulas that compute the MEEC Price from the USPrice in the same row:
  - Most rows use Dn = En * 0.7 (i.e., 70% of USPrice).
  - One notable exception: D6 is D6 = E6 * 1.49351631165241 (different multiplier / possible data entry error).
- USPrice values are explicit numbers (examples: E2 = 39.99, E28 = 249.99, E208/E209/E364 = 299.99).

Key values and ranges
- Record count: 428 items (rows 2–429).
- USPrice range: minimum typically 7.99 (several entries) up to maximum 299.99 (multiple entries).
- MEEC Price is derived, so ranges accordingly (normally 70% of USPrice).

Layout and consistency notes
- Tabular, one row per book; consistent headers in row 1.
- ISBNs stored as text strings.
- Column D formulas are consistent except the D6 anomaly—verify whether that multiplier is intentional.
- Recommend formatting columns D and E as currency for readability and auditing formulas to ensure consistent multipliers if needed.

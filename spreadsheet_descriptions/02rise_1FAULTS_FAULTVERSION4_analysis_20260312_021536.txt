Summary of workbook (concise)

Sheets
- "Sheet4 | Jan.-Dec. 2002" — the main, detailed class roll-up by class and by participating organization for the year 2002.
- "Companies" — summary table that aggregates participants, public cost, RISE cost and RISE savings per company.
- "Financial Info" — year‑by‑year high‑level financial and activity summary (1997–2002) with derived averages and explanatory notes.

Layout and key columns (Sheet4)
- Column A: CLASS TITLE
- Column B: CLASS DATE
- Column C: NUMBER OF CLASS DAYS
- Column D: PUBLIC COST (per class; used to compute what the public would have paid)
- Column E: RISE COST (per class; the RISE tuition)
- Columns F:BG: one column per participating company/agency (participant counts for that class)
- Column BK: "Number of Participants Per Class" — formula: =SUM(F[row]:BG[row]) (total participants in the class)
- Column BL: "Total Number of Student Days Trained" — formula: =C[row]*BK[row]
- Columns F..BG (a "Total" row under each class): per-company dollar totals computed as participant_count * $E$[class_row] (example F4 = G? pattern: each company cell in the Total row uses =SUM(company_count * $E$<class row>))
- Column BO: "Total Actual Costs and Savings Per Class" — per-class RISE revenue total: =SUM(F[TotalRow]:BG[TotalRow]) (sum of all per-company RISE charges for that class)
- Column BM / BN (and per-class "Total" / "Total Savings" rows): BM = D[row]*BK[row] (public cost × participants = public cost allocation for that class); BN = BM - BO (public cost minus actual RISE cost) — i.e. per-class public cost minus RISE receipts = savings
- Many repeating three-row blocks per class:
  - class data row (participants),
  - "Total" row (per-company costs = participants × RISE cost),
  - "Total Savings" row (per-company saving = public allocation − RISE totals).
- Final summary block (near bottom of Sheet4):
  - C268: =SUM(C3:C264) (sum of class days / count of classes)
  - BK total participants (sum of BK3:BK264)
  - BL total student days (sum BL3:BL264)
  - F268:BG268, F269:BG269, F270:BG270 etc. aggregate columns across all classes:
    - F269:BG269 collect aggregated RISE cost contributions per company (sum of per-class totals).
    - F270:BG270 collect aggregated RISE savings per company (sum of per-class savings rows).
  - BI268 (Public Cost Grand Total) = SUM(F268:BG268)
  - BI269 (RISE Cost Grand Total) = SUM(F269:BG269)
  - BI270 (RISE Savings Grand Total) = SUM(F270:BG270)
  - BI271 (Participant Grand Total) = SUM(F271:BG271) where F271:BG271 contain per-company participant totals.

Companies sheet (aggregated by company)
- Columns:
  - A: Company Name
  - B: Class Participants (aggregate participants for 2002 per company)
  - C: Total Public Cost (aggregate public cost assigned to that company)
  - D: Total RISE Cost (aggregate RISE tuition charged)
  - E: Total RISE Savings = C − D (formula in each company row)
- Final rows:
  - B58, C58, D58, E58 compute Grand Totals (SUM of the company columns).

Financial Info sheet (summary & comparisons)
- Header identifies the sheet as "2002 RISE Financial Information".
- Table comparing years 1997–2002 for:
  - Number of classes held, cancelled, rescheduled
  - Number of students trained
  - Number of student days
  - Average class length, cost per day, cost per class
  - Total RISE tuition, total RISE savings, total public cost
- Notes at bottom document how averages and totals are calculated (e.g., average class length = student days ÷ students; total public cost = tuition + savings).

Key formulas & mechanics (high level)
- Participant totals per class: BK = SUM(participant columns F:BG).
- Student-days: BL = Number_of_class_days * BK.
- Per-company RISE charge for a class: company_count * RISE_cost (E is anchored in the Total row formulas).
- Total RISE revenue per class: BO = SUM(per-company RISE charges across F:BG).
- Public cost allocation per class: BM = PUBLIC_COST * BK.
- Per-class savings: BN = BM − BO.
- Company-level summary (Companies sheet) computes totals by aggregating the per-class contributions into per-company C (public) and D (RISE) and E = C − D (savings).
- Sheet4 final summary aggregates across all classes; Companies sheet aggregates across all companies.

Important places to look for totals

  - BI268 = public cost grand total; BI269 = RISE cost grand total; BI270 = RISE savings grand total; BI271 = participant grand total.
- Companies:
  - Row "Grand Totals" (B58..E58) — company-level grand totals and overall RISE savings.
- Financial Info:
  - Row G13 and G14 (Total RISE tuition and Total RISE savings for 2002) and G15 (Total Public cost) give year summary (2002 column).

Overall purpose

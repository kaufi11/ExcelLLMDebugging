Short answer — two problems found:

1) Simple formula typo on Sheet2
- Error: Cell C24 contains =COUNT(B20:B23). COUNT returns 1 (or number of numeric cells), not the sum of asset amounts, so Total Assets (2001) is wrong.
- Fix: Change C24 to =SUM(B20:B23).
- Why: Total Assets must add the asset dollar amounts, not count them.

2) Balance-sheet imbalance because prior-year inventory errors were not closed to retained earnings
- Symptom: Changing ending inventory changes COGS, net income and the Inventory asset — but Total Assets ≠ Total Liabilities + Equity (difference equals the inventory error).
- Explanation: An inventory misstatement affects that year’s income. If that prior-year error was never corrected (closed) into retained earnings, the balance sheet’s retained earnings (equity) does not reflect the cumulative error and the accounting equation won’t balance. Sheet2 shows the error causing the mismatch; Sheet3 shows the correct fix (retain earnings adjusted).
- Fix: Adjust retained earnings to include the prior-year inventory error (i.e., close the prior-year income effect into retained earnings). Example (as done on Sheet3):
  - For 2002 retained earnings, add the 2001 inventory error (D35 = E17 + 17000).
  - For 2003 retained earnings, subtract the 2002 inventory error (F35 = G17 - 25000).
- Why: Net income flows into retained earnings. Correcting retained earnings removes the imbalance because Equity then reflects the cumulative income (including prior-year corrections), restoring Assets = Liabilities + Equity.

If you want a single automatic approach: keep beginning inventory linked to prior year ending inventory (already done), and make retained earnings equal prior retained earnings + current net income + any prior-period adjustment. That way any inventory error is reflected in retained earnings and the balance sheet will balance.

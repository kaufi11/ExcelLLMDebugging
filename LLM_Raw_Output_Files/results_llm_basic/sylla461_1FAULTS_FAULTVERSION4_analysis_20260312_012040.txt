Summary — two real problems I see:

1) Wrong sign in a date formula (sheet "2000")
- Error: Cell B11 contains =B7-2. That subtracts 2 days from the start date.
- Why it’s wrong: Row A7 is "T" (Tuesday) and A11 is "TH" (Thursday). To get Thursday from Tuesday you must add 2 days, not subtract 2.
- Effect: Every date later in the sheet that is built from B11 (B15, B19, …) is shifted/incorrect.
- Fix: Change B11 to =B7+2. After that Excel will recalc all dependent cells and the schedule will line up with the day labels.

2) Wrong base date (sheet "1999")
- Error: Cell B7 on the 1999 sheet is 2000-08-22 (a 2000 date) — clearly the wrong year for the 1999 sheet.
- Why it’s wrong: All later date formulas on that sheet depend on B7. If the base date is in the wrong year, the whole schedule is wrong.
- Fix: Replace B7 with the correct semester start date in 1999 (for example, enter the actual start date for that term — e.g., 1999-08-24 if that is the correct Tuesday). Excel will recalc the rest because the row formulas are relative (B11 = B7+2, etc).

Quick extra checks and tips
- Show all formulas: press Ctrl+` (grave) to quickly spot accidental minus signs or other typos in formulas.
- Ensure date formatting: if you don’t want the time part shown, format the date cells as Date (Home → Number → Short Date or Custom e.g. yyyy-mm-dd).
- Protect the template: after fixing, consider locking those formula cells to avoid accidental edits (Review → Protect Sheet).

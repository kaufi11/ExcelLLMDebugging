Findings
- There is one formula error on the Exams sheet: cell G19 contains =SUM(E19:F19). That omits Exam 1 (D19) for Mittelstadt, so her exam total is wrong.

Why it's wrong
- SUM(E19:F19) adds only Exam 2 and Exam 3. It should add Exam 1, Exam 2 and Exam 3.

Fix (quick)
- In Exams!G19 replace the formula with =SUM(D19:F19).

Fix (faster, consistent)
- On Exams sheet, copy G6 (which correctly has =SUM(D6:F6)) and paste it down G6:G28 (or drag the fill handle). That ensures the same correct formula is used for every row.

Result after fix
- Mittelstadt's exam total becomes 99+101+93 = 293 (was 194).

Tip to avoid this in future
- When a column uses the same formula pattern, fill the whole column (fill down) instead of typing one cell manually. You can also use error-checking (Formulas → Error Checking) or look for inconsistent formulas (Home → Find & Select → Go To Special → Row differences) to catch mismatches.

General advice / how to apply broadly
- Wherever you see a weighted average currently using only C and E (or a single column), replace it with the pattern:
  =IF((Cnn+Enn+Gnn)<>0, ((Cnn*Cmm)+(Enn*Emm)+(Gnn*Gmm))/(Cnn+Enn+Gnn), "")

- Fix obvious typos like C2 that should be C20.

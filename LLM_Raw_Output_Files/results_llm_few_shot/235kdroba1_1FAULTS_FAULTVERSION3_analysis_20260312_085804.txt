I found two problems (one definite error, one risky/inconsistent formula). Short fixes and why:

1) Faulty cell: O11
- Error: formula uses subtraction (=L11-M11-N11) but O (Other) should be the sum of L, M and N like every other row.
- Fix: change to =L11+M11+N11
- Why: O is intended to total three component scores. The minus signs make O11 wrong and that error cascades to P11, R11, AI11, AJ11, etc.

2) Risky/inconsistent cell: AF33
- Current formula: =SUM(Z33:AD33)
- Problem: student AF cells use AFn = Z n + AB n + AD n (non‑contiguous columns). SUM(Z33:AD33) works now only because AA33 and AC33 are blank/zero. If those intermediate cells ever get values you’ll accidentally include them.
- Fix (safer/consistent): =Z33+AB33+AD33   or   =SUM(Z33,AB33,AD33)
- Why: matches the per‑student AF formula and avoids accidentally summing unwanted columns.

Quick suggestion: search formulas for unexpected subtraction in sum/total columns (Find “-” in formulas or review any O-row formulas) — that will catch similar typos.

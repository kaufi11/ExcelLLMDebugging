O20 — Wrong cell reference in formula.

What's wrong: formula is =SUM(100-(E20+G20+I20+K20+M21)) but it uses M21 (next row) instead of M20, so the "Other" percentage is calculated with the wrong value.

Fix: change to =100-(E20+G20+I20+K20+M20)  (or =100-SUM(E20,G20,I20,K20,M20)).

Why: the formula should subtract the five same-row grade percentages from 100; referencing M21 pulls a value from the next row and gives an incorrect result. After fixing, verify similar O-column formulas for the same off-by-one error.

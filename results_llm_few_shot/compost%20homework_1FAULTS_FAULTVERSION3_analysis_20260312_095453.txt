Summary of error
- Faulty cell: Bulk compost!C19 (column C, row 19).
- Problem: Its formula subtracts -2.00188636389994 instead of the -1.48 used everywhere else in that sheet.
- Effect: C19 is 0.52188636389994 days (~12.53 hours) earlier than it should be — this will shift that row’s "start" time and misalign time-series analysis.

Quick fix
- Replace the formula in C19 with the consistent formula used elsewhere:
  =A19 + (B19/24) - 1.48

Better (robust) solution — avoid this error repeating
1. Put the offset value in one cell (example: put 1.48 in Bulk compost!$Z$1).
2. Change all start formulas in Bulk compost to reference that cell, e.g.:
   =A4 + (B4/24) - $Z$1
3. Do the same on Treatments (use e.g. $Z$1 on that sheet with value 1.71) so all rows use the same single source for the offset.

Why this is better
- Single-source offset prevents typos/inconsistent constants.
- Easier to update if offset needs changing.
- Clearer, auditable formulas.

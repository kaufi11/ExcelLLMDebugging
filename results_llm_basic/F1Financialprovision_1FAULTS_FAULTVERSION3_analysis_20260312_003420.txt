Summary — one clear error found

1) Error
- Cell I15 on sheet "B Direct Payments" uses =COUNT(I10:I14). That returns how many cells in I10:I14 contain numbers (a count), not the total value of payments.

2) Why it's wrong (short)
- You want the total monetary value of payments (sum), but COUNT gives a count of numeric cells (e.g., returns 5), so the reported "Total" will be wrong.

3) Fix (step-by-step)
- Select cell I15.
- Replace the formula =COUNT(I10:I14) with either:
  - =SUM(I10:I14) — (recommended), or
  - =SUM(F15:H15) — equivalently sums the age-group totals.
- Press Enter.

4) Quick explanation of the fix
- SUM adds the payment values together and gives the total amount; COUNT just counts how many numeric entries exist.

5) Extra check (recommended)
- Make sure the cells I10:I14 (and F10:H14) are actually numbers (not text). If a number is stored as text the SUM will ignore it. If you see a missing amount, convert text-to-number (Text to Columns or multiply by 1) or check formatting.

That is all — change I15 to SUM and verify numeric formatting.

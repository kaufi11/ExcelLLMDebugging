Fix

Explanation (short)

- After fixing G25, re-evaluate the totals (Excel will auto-recalculate if Automatic calculation is on).

How to find any similar errors (quick)
- Check column G for constants (cells that are not formulas):
  - Select column G → Home → Find & Select → Go To Special → Constants (or use a helper column with =ISFORMULA(G2) copied down).
  - Any TRUE for "not formula" are suspect — convert them to =Erow*Frow.
- Also ensure column F (number of subscriptions) contains numeric values; blanks/ text will make E*F return 0 or an error.

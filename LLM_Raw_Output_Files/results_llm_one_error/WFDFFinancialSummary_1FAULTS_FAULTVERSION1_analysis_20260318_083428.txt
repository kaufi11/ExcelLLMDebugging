Cell: Balance Sheet!H14

Error: Formula uses COUNT instead of SUM (=ROUND(COUNT(H10:H13),5)), so it returns the number of numeric cells (a small integer) instead of the total equity amount — causing the Balance Sheet's equity/total to be wrong and the sheet to not balance.

Solution: Replace with =ROUND(SUM(H10:H13),5).

Why: SUM adds the equity line items; COUNT only counts how many numeric cells exist. Using SUM restores the correct Total Equity and makes TOTAL LIABILITIES & EQUITY match TOTAL ASSETS.

I inspected the workbook formulas and found several clear formula errors (wrong functions and wrong ranges) that propagate incorrect numbers into the final grades. Summary of the problems and recommended fixes below.

Main problem (causes inflated D values and wrong final grades)
- Sheet3!P4
  - Current formula: =SUM(D4,F4,H4,J4,L40)
  - Problems: uses SUM instead of AVERAGE, and references L40 (out of the student rows) instead of L4.
  - Effect: P4 becomes ~3.59 (sum of five ~0.9 figures) instead of ~0.9; that wrong number flows to Sheet1 column D via =Sheet3!P4 and inflates final numeric grades.
  - Fix: change to =AVERAGE(D4,F4,H4,J4,L4)

Systematic offset errors in Sheet3 P column (all student rows)
- Sheet3!P5..P18
  - Current pattern: =AVERAGE(D5,F5,H5,J5,L41), =AVERAGE(D6,F6,H6,J6,L42), ..., =AVERAGE(D18,F18,H18,J18,L54)
  - Problem: the last argument uses L41..L54 (row-offset) instead of the corresponding L5..L18 cells. This produces wrong student average values in column P.
  - Fixes: update each P row so it averages the five percentage cells in that same row. For example:
    - P5 => =AVERAGE(D5,F5,H5,J5,L5)
    - P6 => =AVERAGE(D6,F6,H6,J6,L6)
    - ...
    - P18 => =AVERAGE(D18,F18,H18,J18,L18)

Downstream effect
- Sheet1 column D (Test percentage) is populated from Sheet3 P4:P17. Because P4:P17 are wrong, Sheet1 D3:D17 are wrong and therefore Sheet1 I3:I17 (final numeric grade calculation) and the VLOOKUP letter grades are also wrong (e.g., I3 = 157.255 is a symptom of the incorrect D3 value of ~3.59).

Other inconsistent / incorrect formulas (summary rows on Sheet1)

- Sheet1!C21
  - Current formula: =MAX(C5:C19)
  - Problems: wrong function (MAX) and wrong range. This cell is labeled "Class Average" but uses MAX and wrong rows.

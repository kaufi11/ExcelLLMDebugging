- Correcting B7 on 1999 sets the base date for that sheet; all dependent +n formulas then produce the intended weekdays.

Better (robust) solution — avoid fragile +n offsets
Replace the fixed +n formulas with a "next weekday" formula so the schedule always lands on the day code in column A regardless of the start date. Example pattern:

- Map weekday numbers (with WEEKDAY(…,2): Mon=1, Tue=2, Wed=3, Thu=4, Fri=5).
- To get the next date after prevDate (in B7) that falls on a specific weekday (e.g., Thursday = 4):
  =prevDate + MOD(4 - WEEKDAY(prevDate,2), 7)

Practical example you can paste (assuming previous date is in B7 and A11 contains "T","W","TH" or "F"):
- If you want next Tuesday (T): =B7 + MOD(2 - WEEKDAY(B7,2), 7)
- If you want next Thursday (TH): =B7 + MOD(4 - WEEKDAY(B7,2), 7)
- If you want next Friday (F): =B7 + MOD(5 - WEEKDAY(B7,2), 7)
(Replace B7 with the relevant previous-row cell.)

If you want a single formula that reads the day code in column A, use SWITCH/CHOOSE or MATCH to map the text to numbers and then use the same MOD(...) approach. Example (newer Excel with SWITCH):
=LET(prev,B7, t, SWITCH(A11,"T",2,"W",3,"TH",4,"F",5), prev + MOD(t - WEEKDAY(prev,2),7))

Found one clear error.

Problem
- Bulk compost cell C19 has =A19+(B19*24)-1.48. All other rows use B/24. C19 multiplies hours by 24 instead of converting hours to fractional days.

Why it’s wrong (quick example)
- With A19=15 and B19=12.75:
  - Wrong formula gives: 15 + (12.75*24) - 1.48 = 319.52 (nonsense)
  - Correct logic gives: 15 + (12.75/24) - 1.48 = 14.05125 (consistent with other rows)

Fix
- Replace C19 with the correct formula:
  =A19 + (B19/24) - 1.48

Better / safer practice
- Make column C consistent by re-applying the correct formula to the whole column (fill down): =A4 + B4/24 - 1.48 (use the row numbers as appropriate).
- Put the constant (1.48 or 1.71) in a single cell (e.g. $Z$1) and use an absolute reference so it’s easier to change:
  =A4 + B4/24 - $Z$1
- If you want readable date/time output, format column C as Date/Time (or Number with a few decimals if you want the serial day value).

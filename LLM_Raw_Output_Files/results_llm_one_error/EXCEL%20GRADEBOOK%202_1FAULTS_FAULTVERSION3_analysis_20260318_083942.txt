X5: Formula uses COUNT instead of SUM.  
Fix: change X5 to =(SUM(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)  
Why: COUNT(R5:V5) returns the number of quiz cells (not their total), so the numerator is wrong. SUM gives the total points so dropping the min and dividing by (COUNT-1) yields the correct average.

Cell: Page 2!A83

Error: formula =A$9 references A9 on the same sheet (Page 2) instead of the header on Page 1, so the copied header will be blank/wrong when Page 2's A9 differs.

Solution: change the formula to explicitly reference the master header cell on Page 1, for example:
='Page 1'!$A$9

Why this fixes it: an explicit sheet reference ensures every page pulls the same header cell from Page 1 (even if A9 on the current sheet is different or blank), keeping headers consistent across sheets.

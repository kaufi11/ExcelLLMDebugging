Summary of the error
- Sheet: In-kind contribution template — cell F5 has the wrong formula. It is =571*E5 instead of using the hourly rate in D5. That makes F5 = 571 * 100 = 57,100 instead of the correct 50 * 100 = 5,000. This inflates all downstream totals (F16, G16, G29, G41, F43) by 52,100.

Why it’s wrong (short)
- F column should multiply Hourly Rate (column D) by Projected Hours (column E) on the same row. F5 uses the constant 571 instead of D5.

How to fix (quick steps)
1. Go to the In-kind contribution template sheet, cell F5.
2. Replace the formula =571*E5 with =D5*E5.
3. Press Enter and confirm the value becomes 5,000.
4. Recalculate (F9 or press F9) if automatic calc is off.

Optional checks to prevent similar errors
- Scan column F for formulas that do not use D of the same row. Example quick test: in a spare column put =F5<>D5*E5 and copy down to spot mismatches (TRUE = problem).
- Use Formula Auditing (Formulas tab → Show Formulas or Trace Precedents) or conditional formatting to highlight F cells whose formulas don’t reference D of the same row.
- Protect formula cells or lock them to avoid accidental overwrite.

Impact (one-line)
- Fixing F5 reduces totals by 52,100 (57,100 → 5,000), giving correct grand totals throughout the In-kind sheet.

Spreadsheet: Sheet "1-24"
Purpose / overall layout
- Single sheet with title in A1: "Table 1-27: Condition of U.S. Highway Bridges".
- Column B:N are years 1990 through 2002 (B2 = 1990 ... N2 = 2002). Column A contains row labels (categories, notes, sources).
- Main blocks run down the sheet: TOTAL bridges, Urban bridges (and urban deficient breakdown), Rural bridges (and rural deficient breakdown), All deficient bridges (and structural/functionally totals), then explanatory NOTES and SOURCES (rows A16–A25).

Rows and formula patterns (summary)
- Row 3 (A3 "TOTAL bridges"): each year = Urban (row 4) + Rural (row 5). Most years use direct addition (e.g., B3 = B4+B5); L3 and M3 use SUM(L4:L5)/SUM(M4:M5); N3 contains a hard value 590,877.
- Row 4 (A4 "Urban bridges"): for years B–I many cells are formulas summing several subcomponents (explicit adds, e.g., B4 = 24435+11890+21403+18580+11876+20586). For J–N the values are hard numbers: J4=128,312; K4=130,339; L4=133,384; M4=133,401; N4=135,337.
- Row 5 (A5 "Rural bridges"): similar pattern — complex sum expressions for early years, then J–N contain values: J5=454,664; K5=455,203; L5=456,290; M5=456,284; N5=455,540.
- Rows 6–8: Urban deficient (A6 = A7 + A8). Row 7 = Structurally (sum of components), row 8 = Functionally (sum of components). Many years built by adding component numbers; J–N often contain explicit totals (e.g., J7=14,073; N7=12,504; J8=27,588; N8=29,675). L6/M6 use SUM(L7:L8), SUM(M7:M8).
- Rows 9–11: Rural deficient (A9 = A10 + A11). Row 10 = Structurally; Row 11 = Functionally. Pattern like urban: component sums for some years, explicit totals for J–N (e.g., J10=78,999; N10=68,762; J11=51,912; N11=51,850). L9/M9 use SUM(L10:L11), SUM(M10:M11).
- Rows 12–14: All deficient (A12 = A6 + A9). Row 13 = Structurally (some years hard-coded B13..M13), Row 14 = Functionally (hard-coded some years). L12 and M12 reference SUMs in rows 13–14 (but in sheet given L12 formula looks inconsistent: SUM(L13:L14) and M12 = SUM(M13:M14)). N12 is hard value 162,791; N13 = 81,266; N14 = 81,525.
- Notes and source text occupy rows A16–A25 (definitions of "structurally deficient" and "functionally deficient", scope of table, and source citations).

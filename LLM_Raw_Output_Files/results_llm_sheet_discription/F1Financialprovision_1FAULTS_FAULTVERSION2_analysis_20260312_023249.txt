Findings — error
- Sheet "B Direct Payments": cell I15 contains =COUNT(I10:I14). This is wrong. COUNT returns the number of numeric cells (0–5), not the total value of payments.

Why it's wrong (short)
- I15 is labelled "Total" for Value of Payments. It should add the payment amounts in I10:I14. COUNT only counts cells and so will produce a small integer instead of the money total.

Fix (easy steps)
1. Select I15.
2. Replace the formula with:
   =SUM(I10:I14)
3. Press Enter.

Extra checks / recommendations
- Ensure I10:I14 are numeric (not text). If they look like numbers but SUM returns 0, convert them to numbers (Format → Number, or use Text to Columns / multiply by 1).
- Quick audit: search the workbook for any other uses of COUNT where a SUM is expected (totals). Replace similarly if found.

Result
- I15 will now show the correct total of the "Value of Payments" column (sum of I10:I14) and reporting will be accurate.

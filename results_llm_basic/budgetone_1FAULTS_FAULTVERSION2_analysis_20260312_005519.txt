Summary of the problem
- Cell F7 (Total Sales Revenue) has the wrong SUM range: it is =SUM(B7:D7). That omits the 4th quarter (E7).
- Because F7 is wrong, any formulas that use F7 (F5 = ASP/Unit, the %‑Sales column denominators $F$7, etc.) are producing incorrect results.

Why this matters (short)
- Total annual sales is too low (missing Q4). That makes the annual ASP (F5 = F7/F4) wrong and inflates all percent‑of‑sales calculations that use $F$7 as the denominator.

Fixes (exact formulas)
1. Fix total sales:
   - Change F7 to: =SUM(B7:E7)

2. (Optional) Percent of sales logic
   - If column G is intended to show each line’s percent of TOTAL SALES (annual), the formulas currently using =F#/ $F$7 are fine (e.g., G8 =F8/$F$7). After step 1 these will be correct.
   - If instead you wanted to show each QUARTER’s percent of total sales, use e.g. in a helper row or in column H:
     - Q1 % = B7/$F$7, Q2 % = C7/$F$7, Q3 % = D7/$F$7, Q4 % = E7/$F$7.

Quick check list
- After fixing F7, verify:
  - F5 (ASP/Unit) = F7 / F4 now gives the correct average selling price.
  - All G‑column % formulas that use $F$7 give sensible percentages that add to 100% where appropriate.
- Scan other SUM formulas for the same off‑by‑one omission (make sure all totals use B: E ranges where applicable).

That single correction (F7 =SUM(B7:E7)) should fix the cascading errors.

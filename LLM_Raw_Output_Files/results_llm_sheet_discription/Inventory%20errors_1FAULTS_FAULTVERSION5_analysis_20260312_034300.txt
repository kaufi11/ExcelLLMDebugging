Short answer — two problems and how to fix them:

1) Why Sheets 1 & 2 don’t balance
- Problem: a prior-year inventory error was left in the ending inventory (an asset) but was NOT closed to retained earnings (equity). That leaves Assets changed by the error but Equity unchanged, so Assets ≠ Liabilities + Equity.
- Fix: close the prior-period inventory error into retained earnings for the year the error is discovered. In practice add (or subtract) the inventory error amount to the Retained Earnings cell for the year that should absorb the prior-period correction.

Example changes you can make (match Sheet3 logic):
- For the 2002 column add +17,000 to retained earnings (the 2001 error): set the retained earnings cell for 2002 to =E17 + 17000 (or to the current retained earnings cell +17,000).
- For the 2003 column subtract 25,000 for that prior-year error: set retained earnings for 2003 to =G17 - 25000.
- After doing this the balance sheet will balance because the asset change (inventory) is offset by the equity correction.

Why this works: inventory overstating/understating in a prior period changed net income in that prior period and so should have been reflected in opening retained earnings. Adjusting retained earnings restores the missing equity offset.

2) A formula typo on Sheet3
- Problem: C17 (Net Income, 2001) shows a wrong formula: =734 - C16. That is a literal-number typo; it should use this year’s Gross Profit cell.
- Fix: change C17 to =C15 - C16 (Net Income = Gross Profit - Operating Expenses).

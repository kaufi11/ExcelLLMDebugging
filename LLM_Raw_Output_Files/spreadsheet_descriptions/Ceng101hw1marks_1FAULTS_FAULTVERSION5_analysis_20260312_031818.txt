High-level structure
- Workbook has 3 sheets; only Sheet1 contains data (Sheet2 and Sheet3 are empty).
- Rows 1–68: student records; row 1 is headers. Rows 69–72 are summary/statistics.

Columns and meaning (Sheet1)
- A: Stdt ID (numeric IDs; some outliers e.g. A16 = 98,227,577; A68 = 99,650,185).

Header examples: B "HW1 (/30)", C "HW2 (/40)", D "Exam1", E "exam2 bonus", F "14/4/2003" (date), G "HW3 (/30)", H "22/4/2003", I "28/4/2003", J "lecture 4", K "lecture 5", L–M various lecture/lec columns, N "HW4 (/25)", O "lec Q 5 (/10)", P "rep 2 (/10)", Q "Final exam".

Formulas inside data rows

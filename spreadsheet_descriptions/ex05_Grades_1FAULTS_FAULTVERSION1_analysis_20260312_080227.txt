Overview
- Workbook has five sheets: Exams, Quizzes, Projects, Homework, Summary. Each student appears on rows 6:28 (ID, Last, First) — 23 students, same roster on each sheet.

Sheets and layout
- Exams (C2 title "Exams"): Columns A–C = ID #, Last Name, First Name; D–F = Exam 1–3; G = Total. Row totals G6:G28 use =SUM(Dx:Fx) (one exception noted below).
- Quizzes (D2 title "Quizzes"): A–C = ID/Last/First; D–I = Quiz 1–6; J = Total. J6:J28 use =SUM(Dx:Ix).
- Projects (C2 title "Projects"): A–C = ID/Last/First; D–F = Project 1–3; G = Total. G6:G28 use =SUM(Dx:Fx).
- Homework (C2 title "Homework"): A–C = ID/Last/First; D–G = Summary, Report, Survey, Service; H = Total. H6:H28 use =SUM(Dx:Gx).
- Summary (C2 title "Grade Book for Spring 2003"): A–C = ID/Last/First; D–G headed Homework, Projects, Quizzes, Exams; H = Total; I = Percent; J = Letter. Row 30 (Total Points Possible) shows: D30=120, E30=345, F30=60, G30=300 and H30 = =SUM(D30:G30) → 825 total possible points.

Formulas and key values
- Category totals on each per-student sheet are simple SUM formulas across that row (e.g., Exams G6 =SUM(D6:F6), Quizzes J6 =SUM(D6:I6), etc.).
- Summary includes headers for pulling category totals per student but no explicit per-student formulas are listed in the provided data (i.e., D6:H28 on Summary are present as labels but the formulas linking to the other sheets are not shown).
- Error/irregularity: Exams G19 for Mittelstadt is =SUM(E19:F19) — it omits Exam 1 (D19) and should likely be =SUM(D19:F19).
- Totals row on Summary shows overall possible points per category and H30 sums them to 825.

Notes
- Consistent roster across sheets makes it straightforward to link Summary D–G to each sheet’s Total column (e.g., Summary!D6 =Homework!H6, E6 =Projects!G6, F6 =Quizzes!J6, G6 =Exams!G6).

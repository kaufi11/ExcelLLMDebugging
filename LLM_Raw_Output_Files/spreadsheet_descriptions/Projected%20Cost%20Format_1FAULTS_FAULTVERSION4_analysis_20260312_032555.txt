Overview: The workbook contains three related cost-projection sheets (one per grant-year form) capturing sequence production, technology development, and bioinformatics costs. Most numeric inputs are entered in non-gray cells; many summary/ratio cells are pre-set with formulas to compute totals and percent contributions.

1) Sheet: Seq Production Costs (Sheet 1 — Sequence production projected costs)
- Purpose: Breaks costs into Shotgun reads (B), Q20 bases (C), Finished basepairs (D) and Other Activities (G). Also computes cost-per-read and cost-per-finished-base.
- Key input rows (per category): Amortized Equipment (3 yrs), Labor, Materials & Reagents, Sub-contracts, Travel, Other.
- Direct cost totals: B19 = SUM(B13:B18), D19 = SUM(D13:D18), G19 = SUM(G13:G18).
- Indirects and grand totals: B21 = SUM(B19:B20), D21 = SUM(D19:D20). G21 = COUNT(G19:G20) (note: sheet uses COUNT here).
- Percent columns: C13..C18 = B13/$B$21 etc.; E13..E18 = D13/$D$21; H13..H18 = G13/$G$21.
- Cost-per-unit calculations: B11 = B21 / B10 (avg cost per shotgun read); C11 = B21 / C10 (cost per Q20 base using B21); D11 = D21 / D10 (cost per finished basepair).
- Year total: B23 = B21 + D21 + G21 (one-year total request summed across columns).
- Notes: A6–A7 explain amortization (3 yrs) and indirect costs; gray cells perform the calculations.

2) Sheet: Tech Dev Costs (Sheet 2 — Technology development projected costs)
- Purpose: Itemizes technology-development spending (equipment, labor, supplies, subcontracts, travel, other).
- Key totals: B17 = SUM(B9:B16) (Total Tech Dev direct+other); B17 shown as "Total Tech Dev Costs".
- Relative share: B18 = B17 / B6 (percent of TOTAL COSTS; B6 is TOTAL COSTS and defaults to 0 in the template).
- Guidance cells C9, C10, C12 expect breakdown notes (equipment amortization and labor breakdown).
- A19 requests "Contribution to total read cost ($)" (input).

3) Sheet: Bioinformatics Costs (Sheet 3 — Bioinformatics projected costs)
- Structure: Three main subsections:
  - Infrastructure costs (rows B9:B16; B17 = SUM(B9:B16))
  - Assembly costs (rows B22:B29; B30 = SUM(B22:B29))
  - Annotation costs (rows B35:B42; B43 = SUM(B35:B42))
- Totals and percentages: For each subsection, percent of total = subsection total / B6 (e.g., B18 = B17 / $B$6; B31 = B30 / $B$6; B44 = B43 / $B$6). B6 is TOTAL COSTS (defaults to 0).
- Each subsection contains same category rows: Equipment (amort 3 yrs), Labor, Supplies, Sub-contracts, Travel, Other, Indirects.
- C-cells are for explanatory breakdowns (labor/equipment details).
- A33..A45 are the annotation section; A20..A32 are the assembly section.
- A19, A32, A45 request contribution to total read costs ($) for each subsection.

General/layout/flow notes
- Common categories across sheets: Amortized equipment (3-year amortization), Labor, Supplies/Materials, Sub-contracts, Travel, Other, Indirect costs.
- Most numeric inputs are expected in the $$ columns (B, D, G). Percentage columns auto-compute as category / grand total for that column.
- Several explanatory text cells (C9, C10, C12, etc.) are placeholders for breakdowns and notes.
- The template enforces consistent aggregation: per-category sums → direct costs → add indirects → grand totals → percent contributions and cost-per-unit calculations (in Sheet 1).
- Watch for template defaults (B6 = 0, G21 using COUNT) that may need adjustment when populating real numbers.

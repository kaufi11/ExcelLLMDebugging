Summary of workbook structure and contents

Sheets
- EER SITE — titled "L3 Government Services In. GSA AUTHORIZED PRICING - L3 GSI SITE". Contains labor categories (rows 6–68) with Year1 rates and annual escalations across columns B–H.
- GOVT SITE — titled "EER SYSTEMS INC. GSA AUTHORIZED PRICING - GOVERNMENT SITE". Same layout as EER SITE but with different base rates for many categories.
- Sheet3 — present but no data/detail provided.

Layout and headers
- Column A: Labor Category Description (A4 header).
- Column B: Year 1 rates (7/28/01–7/27/02).
- Column C: Escalation (Per Year) — consistently 0.04 (4%) in most rows.
- Columns D–H: Years 2–5 (7/28/02 through 7/27/07). Year2 is in D, Year5 in H.

Formulas / calculation pattern
- Standard formula pattern: Year2 (D) = B * 1.04; Year3 (E) = D * 1.04; Year4 (F) = E * 1.04; Year5 (G) = F * 1.04; final column H = G * 1.04 (i.e., 4% compounded annually).
- Many rows follow the pattern exactly. Example: Administrative/Graphics Specialist (EER SITE) B6 = 29.99; D6 = B6*1.04; E6 = D6*1.04, etc.
- Several rows were "added 8/02": these have no Year1 (B) value and start with a Year2 value in D (e.g., Secure Information Technologist One in EER SITE: D38 = 72.1). Their subsequent years use the same year-to-year *1.04 pattern.

Key values / observations
- Highest base rates (examples): Enterprise Network Engineer (EER SITE) B18 = 175.55; Systems Engineer, Expert (EER SITE) D57 = 180.74 (entered as Year2 value).
- Lower rates (examples): Training Analyst (GOVT SITE) B64 = 29.48; Administrative/Graphics Specialist (B6) = 29.99 (EER SITE).
- Column C consistently shows 0.04 for escalation across most rows.

Anomalies / data issues to note
- G21 (EER SITE, Functional Analyst, Senior) is set to =F21*1.56935229331187 instead of the expected *1.04 — this is a large, likely accidental multiplier and is inconsistent with the rest of the sheet.
- Several "added 8/02" rows have blank B (Year1) values; their D (Year2) values are entered directly. This is intentional but differs from the rows that compound from Year1.
- A few rows swap the ordering of Programmer/Program Analyst vs Programmer/Program Analyst, Senior between sheets (lookups by row number differ between sheets).

Overall
- Two parallel sheets with the same structure: base-year hourly rates in B and a uniform 4% annual escalation implemented via chained multiplication across columns D–H. Most rows are consistent; check the one anomalous multiplier (G21) and any blanks in Year1 for the "added 8/02" entries if you want uniformity.

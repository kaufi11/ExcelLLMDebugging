Cell J4 — Error: ATAN2 arguments are reversed. Formula uses =ATAN2(G4,H4)*180/PI() but G4 is the real part and H4 is the imaginary part, so the arguments are swapped.

Fix: change to =ATAN2(H4,G4)*180/PI()

Why: Excel’s ATAN2(x_num,y_num) returns arctan(x_num/y_num). To get the complex phasor angle (atan2(imag,real)) you must pass the imaginary part first and the real part second. Apply the same swap to all other angle formulas using ATAN2 (e.g. J4:J50 and N4:N50).

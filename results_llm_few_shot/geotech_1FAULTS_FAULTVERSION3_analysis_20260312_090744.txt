I found two clear errors and one class of suspicious formulas. Short fixes below.

1) Major bug (wrong hard-coded value)
- Faulty cell: Data Analysis!AU32
- Error: It contains IF(D$2="x",345,0) — the 345 is a hard-coded number instead of the cell value for that course. This will add 345 to the totals whenever that course is included and wildly distort all results.

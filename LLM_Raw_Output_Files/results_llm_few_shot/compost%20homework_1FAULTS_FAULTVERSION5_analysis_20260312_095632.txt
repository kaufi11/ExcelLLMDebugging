Faulty cell
- Bulk compost!C19

Error type
- Mechanical/typo (wrong operator).

What's wrong (short)
- C19 uses B19*24 instead of B19/24, so the hour value is multiplied by 24 instead of converted to a fraction of a day. That makes the "start" value huge and incorrect.

Correct formula
- =A19 + (B19/24) - 1.48

Why this fixes it (short)
- Dividing B (hours) by 24 converts hours to days so A (day) + hours gives the correct datetime. Multiplying by 24 returns an enormous number of days and shifts the timestamp.

How to fix quickly
- Edit C19 to the correct formula, then fill/copy that corrected formula down the C column to keep consistency.
- Alternatively, in C4:C22 replace the formula with =A4+(B4/24)-1.48 (adjust row) and drag down.

Check
- I scanned the other C formulas in both sheets — the Treatments sheet and the rest of Bulk compost use B/24 correctly. Only C19 had the *24 typo.

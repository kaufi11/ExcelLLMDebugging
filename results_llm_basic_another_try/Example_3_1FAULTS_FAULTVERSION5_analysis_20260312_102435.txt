Errors found:

1. In "Q4 Homework" sheet, cell H5 uses =COUNT(B5:G5) instead of =SUM(B5:G5). This causes the total points for Evans to be the count of cells with numbers, not the sum of points.

2. In "Q4 Summary" sheet, cells F3 to F12 have incorrect formulas for weighted scores. They use D3*0.4 + D3*0.2 instead of D3*0.4 + E3*0.2. The E column is homework, but the formula uses D column twice, ignoring homework scores.

Solutions:

1. Change cell H5 formula in "Q4 Homework" from =COUNT(B5:G5) to =SUM(B5:G5). This correctly sums Evans' points.

2. Change formulas in F3:F12 in "Q4 Summary" from B3*0.3 + C3*0.1 + D3*0.4 + D3*0.2 to B3*0.3 + C3*0.1 + D3*0.4 + E3*0.2 (and similarly for rows 4 to 12). This correctly weights homework scores.

Explanation:

- SUM adds all numeric values; COUNT counts cells with numbers. Using COUNT for total points is wrong.

- Weighted score must multiply each category by its weight. Using D3 twice ignores homework (E3), causing wrong totals.

Fixing these will give correct totals and weighted scores.

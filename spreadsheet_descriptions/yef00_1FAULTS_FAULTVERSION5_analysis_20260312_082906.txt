Overview
The workbook documents Iowa community college revenues, expenditures, state aid and FTEE for Fiscal Year 1999–2000. It is organized by topic into six sheets with consistent layouts: headers, line-item dollar amounts in a C column, percent-of-total formulas in an adjacent B (or I) column, totals and footnotes. Source and rounding notes appear on each sheet.

General/formula patterns and key points
- Percent-of-total formulas consistently reference the sheet total with absolute references: Revenues use $C$15; Expenditures by Function use $C$19; Categories use $C$15 (their local total).
- Totals are computed by SUM ranges (e.g., SUM(C9:C14), SUM(C9:C18), SUM(B9:B23), etc.).
- The workbook contains a few manual adjustments/oddities (several "+1" additions, a literal "3684419+1" and one cell set to 4) that affect totals and should be checked if strict reconciliation is required.
- Expenditure totals (by function and by category) match at 309,825,920. General State Financial Aid total is 141,577,403; total revenues listed sum to 311,917,185.
- Each sheet contains source citations to Iowa Department of Education reports and a rounding/footnote block.

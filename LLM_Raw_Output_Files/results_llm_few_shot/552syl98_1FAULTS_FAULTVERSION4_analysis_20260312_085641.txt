Summary of problems
- Faulty cell: B33. Formula =B30+904 is a typo.  
  - Error type: Mechanical/typo.  
  - Why it’s wrong: adding 904 days jumps the calendar ~2.5 years ahead; every date after B33 is therefore wrong.

Fix
- Replace B33 with: =B30+7

Why this fixes it
- The sheet uses a repeating pattern (class days: +2 days, +2 days, then +7 days to the next week). B33 should be the next-week (Monday) date after B30, so +7 (not +904). Once B33 is corrected, B34:B59 (and formulas that reference B33) will return the intended dates.

Other notes (short)
- The leading + in formulas like =+B16+2 is harmless but unnecessary — you can remove it for clarity.  
- B60 currently contains “? April/May” rather than a date. If you want a final-date formula, set it relative to the prior class (for example =B59+7 or whatever the correct offset is).

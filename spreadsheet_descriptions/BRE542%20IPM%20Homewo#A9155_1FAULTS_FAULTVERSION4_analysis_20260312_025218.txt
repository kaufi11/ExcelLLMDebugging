Key formulas & workflow encoded
- Van Genuchten model in D-column of fit sheet: theta(h) = theta_r + (theta_s - theta_r) / [1 + alpha * (-h)^n]^(1 - 1/n).
- Residual squared errors per point: (observed - predicted)^2; SSE is sum of those errors.
- Ln P column computes ln(-pressure) for plotting/weighting if needed.
- Data sheet intended to be used to compute fluxes from changes in profile water content, compute hydraulic gradients, then apparent K from Darcy’s law; solver is to be used to fit Ksat using the retention parameters fixed from the theta-h fit.

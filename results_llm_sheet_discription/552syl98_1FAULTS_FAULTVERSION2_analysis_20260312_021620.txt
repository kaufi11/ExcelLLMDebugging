Summary of the errors I found and how to fix them (short & simple)

1) Big typo in B18
- Problem: B18 =B15+663. That adds 663 days and breaks the whole schedule because many later date formulas refer to B18 (B19, B21, B24, B27, B30, B33, …).
- Why it's wrong: the pattern in the sheet is +2 (next class day) and +7 (next week). B18 should be the next Monday/week, not 663 days later.
- Fix: change B18 to =B15+7 (or to =B17+2 if you prefer building from the previous class). After that the cascade of dependent dates will correct automatically.
- How to do it: double‑click B18 (or click and edit formula bar), replace 663 with 7, press Enter. Optionally use Trace Dependents (Formulas → Trace Dependents) to confirm which cells update.

2) Placeholder in B60
- Problem: B60 contains text "? April/May" instead of a date, so formulas that expect a date might fail if you try to use B60 in calculations.
- Fix / Recommendation: Replace the placeholder with the actual final exam date when known, or with a computed date if you want to follow the pattern (for example =B59+2 or whatever offset is correct). If it is intentionally unknown, leave the text but don’t reference B60 from formulas.

3) Cosmetic / maintenance suggestions
- Cells like =+B15+2 (leading +) are harmless (Excel accepts them) but unnecessary. You can clean them to =B15+2 for clarity.
- The very large block of repeated identical URLs across row 6 and 7 is redundant. If they’re only for printing or header, move the URL to a single cell and use Header/Footer or a merged cell to keep the sheet tidy.
- If you want to find similar typos quickly, use Edit → Find to search for "+663" (or other large unexpected numbers).

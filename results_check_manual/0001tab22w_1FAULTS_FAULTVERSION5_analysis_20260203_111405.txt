Summary of problems I found
- Wrong operator in O14 (Hospitality, lower level). The formula uses subtraction inside the parentheses: =SUM(100-(E14-G14-I14-K14-M14)). That produces 100 - (E14 - G14 - I14 - K14 - M14) instead of 100 - (E14 + G14 + I14 + K14 + M14). Result: O14 ≈ 164.7 instead of ≈ 4.1.
- Misplaced / stray values in intermediary columns (F, H, Z, etc.). Example cells with unexpected numbers: F18, H18, F19, H19, F20, H20, Z22, Z23, Z24 (and H10, H14 present). These extra entries shift or duplicate data and can cause confusion and wrong totals if formulas ever change to use contiguous ranges.
- Inconsistent/fragile formula style: many "Other" formulas use =SUM(100-(...)) which is harmless but odd; also many cells contain the text "n/appl" — these are text and will break aggregate formulas if included in a SUM range.

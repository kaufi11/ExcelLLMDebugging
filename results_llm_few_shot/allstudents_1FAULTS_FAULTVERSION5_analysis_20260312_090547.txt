Faulty cells
- D74 (All Grades Tested for Spring 1997)
- F74 (All Grades Percent for Spring 1997) — wrong only because D74 is wrong

Error type
- Range/omission error (inconsistent sum ranges)

What’s wrong (short)
- D74 =SUM(D68:D73) omits Grade 3 tested (D67). E74 =SUM(E67:E73) DOES include Grade 3 passing. That makes the numerator and denominator inconsistent, so F74 (=E74/D74) is wrong.

Fix
- Change D74 to: =SUM(D67:D73)
- Keep E74 =SUM(E67:E73) and F74 =E74/D74

Why this fixes it (short)
- Both Tested (D74) and Passing (E74) will sum the same set of grades (Grade 3 → Grade 10). The percent F74 will then be the correct total passing divided by total tested.

Quick tip
- Scan other “All Grades” rows to ensure the Tested and Passing sums use the exact same start/end rows.

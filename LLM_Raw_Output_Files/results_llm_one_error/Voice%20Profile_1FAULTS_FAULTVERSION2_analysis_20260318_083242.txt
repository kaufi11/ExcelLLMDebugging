I22 — Wrong cell reference causing incorrect/blank result

Error: Formula uses C2+E20 instead of C20+E20, so the weighted average duration is wrong or returns divide-by-zero.

Solution: Replace formula in I22 with:
=IF((C20+E20)<>0, ((C20*C22)+(E20*E22))/(C20+E20), "")

Why this fixes it: The formula must weight average durations by the number of calls in C20 and E20. Using C2 (a header) breaks the calculation; C20+E20 correctly sums call counts so the weighted average computes properly.

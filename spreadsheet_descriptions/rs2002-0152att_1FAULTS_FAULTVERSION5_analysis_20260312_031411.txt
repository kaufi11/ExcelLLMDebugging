Overview
- Workbook contains multiple grant-program sheets recording Region, RFP#, Applicant, Project Title, Result (Fund/Not Fund/Non-Competitive), Project Funds Requested (G), Approved/Awarded Funds (H), and a running Cumulative Total Awarded (I).
- Column layout repeated on all sheets: B=Region or program code, C=RFP#, D=Applicant, E=Project Title, F=Result, G=Requested Funds, H=Awarded/Approved Funds, I=Cumulative total (running sum down the sheet).
- Typical formula pattern in column I: first awarded row I7 = H7, subsequent rows I(n) = I(n-1)+Hn (running cumulative of Awarded Funds). This column is intended to track spending against a sheet-level funding limit shown in the header.
- Each sheet header shows program/funding limit and approving resolution; several sheets include a note when the funding ceiling is reached or remaining balance for a specific project.

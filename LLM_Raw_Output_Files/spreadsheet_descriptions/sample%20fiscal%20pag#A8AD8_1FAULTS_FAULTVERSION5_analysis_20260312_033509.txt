High-level summary of the workbook (concise)

Workbook purpose and overall layout
- Purpose: Consolidated fiscal/forecast report for several non‑sponsored funds (TFAC), showing resources, transfers, expenditures (by function and by object class) and ending balances by year/period.
- Columns: years/periods across D:I (D=1999-00, E=2000-01, F=2001-02, G=2002-03 Period 14, H=2003-04 Projected, I=2004-05 Budget).
- Repeating block structure on every sheet: RESOURCES (a: Carry Forward + Revenues b), Transfers (c), Net Transfers, TOTAL NET RESOURCES (d = a+b+c), EXPENDITURES by Function, EXPENDITURES by Object Class, TOTAL EXPENDITURES (e), ENDING BALANCE (d - e).
- Every fund sheet has the same template; Summary sheet aggregates across funds.

Sheets
1. Summary

   - Many 3‑D SUM formulas over sheets '1003:1856'!<cell> for totals (e.g., A6 carry forward row D6 uses =SUM('1003:1856'!D6)).
   - Also pulls specific fund totals with direct references (e.g., Summary D9:D13 pull each fund's Total Revenues via '1003'!D25 etc.).
   - Key computed outputs: TOTAL NET RESOURCES in row 26 (D26:I26) and ENDING BALANCE row 63 (D63:I63) where Ending = TOTAL NET RESOURCES - TOTAL EXPENDITURES.

Key formula patterns and links
- Aggregation pattern:
  - Summary uses 3‑D SUM('1003:1856'!<cell>) to aggregate the same cell across all fund sheets.
  - Summary also references individual fund totals directly (e.g., Summary D9 = '1003'!D25).
- Carry Forward calculation on each fund sheet: D6 = E6 + D72 - D25 - D35 (uses prior period carryforward + prior period total expenditures - revenues - net transfers).
- Revenues totals: C25 computed with SUM across revenue rows D9:D24.
- Net Transfers: C35 = SUM(D28:D34) (central + mandated + non‑mand transfers).
- TOTAL NET RESOURCES: D37 = D35 + D25 + D6 (c + b + a).
- TOTAL EXPENDITURES by object class: row 72 = SUM(D56:D71).
- ENDING BALANCE: row 74 (or Summary D63) = TOTAL NET RESOURCES - TOTAL EXPENDITURES.

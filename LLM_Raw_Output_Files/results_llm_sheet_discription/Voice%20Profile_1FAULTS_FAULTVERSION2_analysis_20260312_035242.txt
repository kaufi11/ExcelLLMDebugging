General recommendation (cleaner, less error‑prone)
- Use the same pattern for every weighted average: IF(SUM(counts)=0,"", (C*dur + E*dur + G*dur) / SUM(counts)).
- Or, if your source columns are contiguous in the future, use SUMPRODUCT for compactness. Example (if counts in C14:E14 and durations in C16:E16): =IF(SUM(C14:E14)=0,"",SUMPRODUCT(C14:E14,C16:E16)/SUM(C14:E14))
  (Note: in your file columns are non-contiguous (C,E,G), so the explicit C* + E* + G* approach is easiest and clearer.)

Why these fixes matter (one-line)

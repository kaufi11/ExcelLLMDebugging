C19: formula uses COUNT(D19:CC19) but should SUM the scores — change to =SUM(D19:I19)

Why: COUNT returns the number of numeric cells (e.g., 6), not the total points, so the student's points and resulting % (B19) are wrong. Also the range CC19 is incorrect; use the actual columns D:I. After fixing to =SUM(D19:I19) the percent formula (C19/$C$6) will compute correctly.

Cell: Year 1!B6

Error: Circular reference — B6 =SUM(B55) while B55 depends on B15 (which uses B6). This makes Excel iterate or return an error and breaks the cash-flow logic.

Solution (and why):
- Replace the formula in Year 1!B6 with the actual opening cash balance (manual input) or a link to a dedicated "Starting Cash" cell (e.g., =StartUp!A1). 
- Keep month-to-month rollforward correct by setting each subsequent month’s beginning cash to the prior month’s ending cash (C6 = B55, D6 = C55, E6 = D55, etc.).

Explanation: The beginning-of-month cash for the first month must be an input (initial cash), not calculated from the same month’s ending cash — otherwise formulas reference each other and create a circular loop. Changing B6 to a fixed starting value removes the circular reference and lets the month-to-month links (C6 = B55, etc.) correctly roll cash forward.

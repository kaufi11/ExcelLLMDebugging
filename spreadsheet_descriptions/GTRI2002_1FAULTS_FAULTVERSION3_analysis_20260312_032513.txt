Summary of workbook structure and contents

1) Sheet: 2003 Report
- Purpose/layout: Year-by-year TRI summary (1987–2000) in two side-by-side tables:
  - Left table (columns A–F): Year (A), Releases/Disposal categories — Air (B), Land (C), Water (D), Offsite release/disposal (E), Total releases (F).
  - Right table (columns J–O): Year (J) repeated, Transfers offsite broken into Recycled (K), Energy recovery (L), Other treatment (M), Incineration (N), Offsite total (O).
- Data: Numeric values per year. Example totals:
  - 1987: Total releases F7 = 13.239865 (million lbs), Offsite total O7 = 14.847612.
  - 1999: F19 = 4.255724, O19 = 26.282264 (large offsite transfers).
  - 2000 row includes extra values in G20 (8.449074) and H20 (0.6381540899) — likely intermediate/calculated numbers.
- No explicit formulas shown in supplied extract (values only).

2) Sheet: 2002 Report (header: "IGNORE THIS SHEET")
- Purpose/layout: TRI by year with conversions and comparisons (pounds → millions of pounds), plus joint-venture / SVEP comparisons by ZIP.
- Common formulas/patterns:
  - Pounds → millions: many cells use =<pounds cell>/1000000 (e.g., B11 = F11/1000000).
  - Year increment: A12 = A11+1 pattern to enumerate years.
  - Media totals: E11 = SUM(C11:D11) or R11 = B11+C11+D11 etc.
  - Comparison metrics: L = SUM(B:K)/1000000 (total pounds across ZIP columns → millions); M = SUM(B14:D14) (SVEP subset); N = L-M (difference); O = N/L (proportion).
  - R24 = (R11-R23)/R11 (percentage change between two totals).
- Key raw pound values (examples): F11 = 1,293,429; G11 = 9,709,396; H11 = 250 (these get converted to millions in B11–D11).
- Sections: annual TRI conversion table, "Disposed Offsite / Recycled / Incinerated Offsite" table (many NA entries until later years), JV vs SVEP ZIP-code breakdown (rows 43–56), ZIP-level breakdown for 1999 data (rows 58–71) with sums (B71:F71).
- Note: sheet title suggests ignore, but it contains conversion formulas and JV/SVEP comparison calculations.

3) Sheet: data
- Purpose: reconciliation of "incinerated" vs "recycled" for several years.
- Layout: columns: incinerated (B), recycled (C), total recycled+incinerated (E).
- Formulas: Cn = En - Bn (recycled = total - incinerated).
- Values: Years 1991–1999. Examples:
  - 1991: incinerated B4 = 291,636; total E4 = 6,659,217 → recycled C4 computed.
  - 1999: B12 = 4,002,596; C12 = 13,526,767; E12 = 17,529,363.

4) Sheet: notes
- Short reference of incinerated/recycled numbers for 1997–1999 duplicate of data sheet entries (B2:C4).

5) Sheet: 1999 Report
- Very similar structure to 2002 Report: year series with pounds converted to millions (cells B9:D19 = F9/1000000 etc.), and a "Disposed Offsite / Recycled or Incinerated Offsite" table (F29:G35 etc).
- Uses same conversion and year-increment formulas.
- Raw pound values F9..F19 and G9..G19 mirror those in 2002 sheet.

Cell: FTEE Calculation!H14

Error: Formula uses the wrong row — it is =+C15+F14 (references C15) so ILCC-03’s FTEE is calculated with another college’s credit FTEE.

Solution: Change the formula to =C14+F14 (or =C14+F14+0.01 if you need the same small adjustment used elsewhere). Then recalc totals (H28) and percent shares (I12:I27) so they reflect the corrected value.

Why: Using C15 instead of C14 pulls the wrong credit FTEE into ILCC-03, which skews that row’s FTEE, the H28 total, and any percentage of total derived from H values. Correcting the reference fixes the chain of dependent calculations.

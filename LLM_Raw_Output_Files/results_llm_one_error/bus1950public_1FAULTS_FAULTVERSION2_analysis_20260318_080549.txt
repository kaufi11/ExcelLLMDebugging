M Noon!P12 — Error: formula compares O12 to 'F Noon'!O$38 (wrong sheet), so this student’s grade uses another sheet’s total points (inconsistent threshold) and may be misgraded.

Solution: point the comparison to the correct anchored total on the same sheet. Example fix:
=IF(O12>$O$38*0.9,"A",IF(O12>$O$38*0.8,"B",IF(O12>$O$38*0.7,"C",IF(O12>$O$38*0.6,"D","F"))))

Why: using O$38 (or 'M Noon'!$O$38) ensures the grade cutoffs come from this sheet’s total points; $ locks the row when you copy the formula so all rows use the same threshold.

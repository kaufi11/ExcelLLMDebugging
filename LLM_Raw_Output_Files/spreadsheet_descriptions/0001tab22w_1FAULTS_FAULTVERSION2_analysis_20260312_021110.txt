Sheet name and purpose
- Sheet: "Table 22" — Percentage distribution of Fall-term undergraduate grades by College, Grade Category, Level (Lower/Upper) and Year.

Overall layout
- Title rows: A1–A2 contain the sheet title/subtitle.
- Column groups:
  - Left identifiers: A = College, C = Year.
  - Lower-level grade % columns: E = A, G = B, I = C, K = D, M = F/WF, O = Other (calculated), Q = Total Number of Grades (count).
  - A blank column R separates lower and upper blocks.
  - Upper-level grade % columns: S = A, U = B, W = C, Y = D, AA = F/WF, AC = Other (calculated), AF = Total Number of Grades (count).
- Rows are repeated blocks per college (Arts & Sciences, Business Administration, Hospitality Management, Education, Engineering, Health & Public Affairs) with subrows for years (2000, 1999, 1998). Final block A29–A32 contains TOTAL UNIVERSITY and totals by year. Source line in A33.

Key formula patterns
- "Other" percentage columns (O and AC) are computed as 100 minus the sum of the listed grade-category percentages (e.g., O6 = 100 - (E6+G6+I6+K6+M6)). The sheet uses SUM(100 - (...)) around the single result — functional but unnecessary.
- TOTAL UNIVERSITY totals:
  - Lower-level count Q30 = Q26+Q22+Q18+Q14+Q10+Q6 (sum of lower-level counts by college).
  - Upper-level count AF30 = AF26+AF22+AF18+AF14+AF10+AF6 (sum of upper-level counts by college).
  - Year totals use analogous SUM formulas across college rows (e.g., Q31, AF31).
- Cell formulas occasionally include minor errors/typos (see "Data/Formula issues" below).

Key values (TOTAL UNIVERSITY)
- Year 2000, Lower-level percentages: A 30.7%, B 29.3%, C 17.8%, D 5.3%, F/WF 6.6%, Other (computed) = 10.3%.
- Year 2000, Upper-level percentages: A 36.6%, B 31.3%, C 16.0%, D 3.9%, F/WF 4.8%, Other (computed) = 7.4%.
- Year 2000 totals: Lower-level count Q30 = 47,993; Upper-level count AF30 = 57,857.

Data / formula issues to watch
- Several rows contain extraneous columns or misplaced entries (e.g., Education rows list values in F18 and H18 which break the primary column mapping).
- AC22 (Engineering upper-level Other) shows a formula using subtraction operators inside the parentheses: SUM(100-(S22-U22-W22-Y22-AA22)) — this is incorrect; it should be SUM(100-(S22+U22+W22+Y22+AA22)) like the others.
- Hospitality Management 1999 and 1998 entries are "n/appl" (not applicable), so percentages are absent for those years.
- Some cells include a zero in intermediate columns (e.g., H10 = 0) that may indicate misaligned inputs.

Notes
- Percentages are per-level distributions by grade category; totals in Q and AF are counts used to compute university totals.
- The "Other" columns ensure each row sums to 100% (unless data are n/appl or formulas are wrong).

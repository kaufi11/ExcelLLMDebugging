Cell: 'Table 2.0 BS - Annex E'!H59

Error: H59 (Endowments) contains =H15 (Total Income) — endowment balance is incorrectly set equal to income, which inflates Total Funds (H69) and breaks the balance sheet.

Solution: Replace =H15 with the correct endowment closing balance (a manual value or a reference to the proper endowment source cell). Example: set H59 = 0 if there are no endowments, or set H59 = 'CorrectSheet'!Cell where the endowment closing balance is stored.

Why: Total Funds (H69) = H56 + H59 + H67, so linking H59 to total income incorrectly adds income into funds. Using the actual endowment value fixes totals and allows the balance sheet to reconcile.

Faulty cell(s)
- Your Company!C41

Error type
- Hard-coded value / Aggregation error (logical)

What's wrong (short)
- C41 contains the number 7 instead of summing the calculated savings lines (Cash flow, Obsolete, Waste, Exports, Returns, Zone transfers, Inverted duty). That gives a wrong total and makes the "Text Only" sheet show 7 too.

Fix (exact formula)
- Replace C41 with:
  =SUM(C22,C24,C27,C30,C33,C36,C39)
  or simply:
  =SUM(C22:C39)

Why this fixes it (short)
- The SUM collects each component of the possible savings instead of a fixed number, so the total updates correctly when any input changes. Using the explicit list avoids accidentally including unrelated cells; the range version is fine if you keep the layout.

Optional note (quick)
- If Inventory Turnovers (Your Company!C14) might be zero, avoid divide-by-zero in C22 by wrapping it, e.g.:
  =IF(C14=0,0,C13/C14*C15*C17)

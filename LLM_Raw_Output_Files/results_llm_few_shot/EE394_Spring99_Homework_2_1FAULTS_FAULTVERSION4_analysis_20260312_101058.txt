   - Where you compute angles you used ATAN2(Gx,Hx) and ATAN2(Kx,Lx).
   - Excel's ATAN2 expects ATAN2(y, x) (first argument = imaginary/vertical, second = real/horizontal). Your G = Real and H = Imag, so those are swapped.

   Fix (example):
   - J4 currently: =ATAN2(G4,H4)*180/PI()
   - Correct J4: =ATAN2(H4,G4)*180/PI()
   - N4 currently: =ATAN2(K4,L4)*180/PI()
   - Correct N4: =ATAN2(L4,K4)*180/PI()
   - Apply the same swap for every row in J and N (fill down).

   Why: Using ATAN2(real,imag) returns atan(real/imag) and gives wrong quadrant/angle. Use ATAN2(imag,real) to get the proper phasor angle.

Summary of workbook: 3 sheets — "Summary Info", "Participant List", and "Speakers & Mentors".  Below is a concise description of each sheet, the layout, key values and formulas, and a note of a couple of formula/data quirks.

1) Sheet: Summary Info
- Purpose: high-level recruitment/response summary and Carnegie-class breakdown of CAREER awardees invited to a workshop.
- Top narrative (A1:A9): key summary numbers and context:
  - 1,387 (~85%) of CAREER awards (2000–2003) identified as PIs at Research‑Extensive (RE) institutions.
  - 238 (~15%) identified as awardees at HBCUs and/or non‑RE institutions — these 238 formed the candidate pool for self‑nomination.
  - Invitations were sent to ~193 eligible awardees (~80% of 238); expected 45–50 participants; actual interest was higher: 66 interested, plus 21 who "would like to attend but can't."
- Directorate table (rows 11–20):
  - Columns: Directorate (A), "would like to attend" (B), invited (C), eligible invitees (D).
  - Totals row (row 20): B20 uses =COUNT(B12:B19) (counts number of numeric cells — returns 8), C20 =SUM(C12:C19), D20 =SUM(D12:D19).
  - Example per‑directorate values (B12:B19): OPP 0, EHR 4, SBE 1, GEO 8, BIO 14, CISE 12, ENG 15, MPS 12 (sum = 66 interested).
- Carnegie classification table (rows 22–30):
  - Columns: classification (A), would like to attend (E), invited (F), eligible invitees (G).
  - Totals row (row 30): E30 =SUM(E23:E29), F30 =SUM(F23:F29), G30 =SUM(G23:G29).
  - Example values: Doctoral/Research Intensive (DRU‑i) — E27=38, F27=112, G27=149; HBCUs row (E29–G29) = 2,2,2.
- Notes/links: Carnegie classification footnote (A33) and comments in column J clarifying invitation limitations (e.g., only 61 participants invited, 5 on waiting list).

2) Sheet: Participant List
- Purpose: detailed roster of invited/confirmed participants, organized (mostly) by directorate and institutional classification.
- Header row (row 2) columns:
  - A: sequence/# by directorate, B: total count (running index), C: Institution Type (e.g., BCLA, DRU‑i, MCU‑1, R1), D: Project Title, E: NSF Program, F: Directorate, G: NSF PD & Program, H: start date, I: end date, J: Amount (grant $), K: PI name, L: PI email & address info, M: more address info. Some rows include extra metadata in N and O (discipline codes/keywords).
- Data types:
  - Dates in H/I appear as full datetime values (e.g., 2001-06-01 00:00:00).
  - Amounts (J) numeric (grant totals).
  - Contact fields (K–M) include PI names, email addresses, institutional mailing addresses and phone numbers.
- Range / scope:
  - Detailed entry rows for many participants (rows 3–63 shown), spanning directorates BIO, CSE, ENG, GEO, MPS, SBE, EHR, etc.
  - Waiting list block begins around row 68–74 with labeled "ON WAITING LIST" entries.
- Notable values:
  - Participant count by directorate encoded in column B (the running counts correspond to directorate grouping).
  - Example entries: Rebecca Kohn (Ursinus College) — CAREER start 2001‑06‑01, amount 541,158; many PIs from DRU‑i and MCU‑1 institutions.
- Misc: bottom of sheet includes Carnegie classification legend (rows ~80–86).

3) Sheet: Speakers & Mentors
- Purpose: list of organizing committee, dinner speaker, administrators, education speakers, and mentors.
- Columns: A Name, B Affiliation, C Position, D e‑mail (headers in row 2).
- Sections:
  - Organizing Committee (rows 4–6): names, affiliations, roles and emails (e.g., Alison Flatau, UMD).
  - Dinner Speaker (row 9): Cesar Malavé.
  - Administration and Education Speakers (rows 11–20): names and roles.
  - Mentors (rows 22–34): list of mentors with affiliations and contact emails (e.g., Makola Abdullah, Florida A&M).
- Clean, simple contact list format.

General notes / data quirks
- Summary Info B20 uses COUNT instead of SUM, so it returns the count of numeric cells (8) rather than the numeric sum (66) of "would like to attend" values — the actual interested total is 66 (sum of B12:B19).
- Dates are stored as full datetimes; amounts are stored as numeric currency-like values (no explicit currency formatting shown).
- Participant List contains some extra discipline codes in columns N/O for selected rows.
- Waiting list entries are included in Participant List (labeled) and are separate from the main invited list.

Findings
- Faulty cell: Bulk compost sheet — C5.
- Error type: Copy/paste / sign error (mechanical). All other rows use "-1.48" but C5 uses "+1.48".

Why it's wrong (short)
- C5 = A5 + (B5/24) + 1.48 adds 1.48 days instead of subtracting it, shifting that row by 2.96 days compared with the others (difference = +1.48 − (−1.48) = +2.96).

Quick fix
- Replace the formula in C5 with:
  =A5 + (B5/24) - 1.48

Better practice (recommended)
- Put the offset (1.48) in one cell (e.g. put 1.48 in F1 on that sheet and name it StartOffset or use $F$1), then use:
  =A5 + (B5/24) - $F$1
  Copy that formula down the column. This avoids future sign/copy errors and makes the offset easy to change.

Note
- The Treatments sheet consistently uses −1.71; change only if that is also an accidental mismatch.

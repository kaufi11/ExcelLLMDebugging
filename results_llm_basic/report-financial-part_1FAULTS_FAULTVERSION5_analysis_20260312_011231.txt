Why these are wrong (brief)
- Column D is the unit rate, column C is # units, columns F and G are spent by source. The logical total is either:
  - H = F + G (actual spent split by source), or
  - H = C * D (if you compute amount from units and unit rate).
  Multiplying C * G mixes a unit count with an absolute money value and produces nonsense.
- Fixed values (H12 = 4) create wrong totals.
- Subtotals that start on the wrong row include headers/empties so totals are incorrect or inconsistent.

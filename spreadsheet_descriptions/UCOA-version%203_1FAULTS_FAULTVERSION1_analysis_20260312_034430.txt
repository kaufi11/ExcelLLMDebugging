Overview
- Workbook maps chart-of-accounts ranges to presentation/ reporting line items (Balance Sheet, Income Statement) and to multiple reporting frameworks (GAAP exhibits, Form 990/990‑EZ, OMB A‑122 cost principles, United Way accounting guide, Tennessee templates).
- Main content is an account-code dictionary: column A = primary account code, B = alternate code, C = account description. Columns D–I provide cross‑references: GAAP exhibit (D), Form 990 line item (E), Form 990‑EZ line (F), Cost Principles / matching rules (G), Accounting Guide ranges (H), Templates/notes (I).

Sheets summary

Pages 1–2 (Balance sheet coding)
- Page 1: Assets section (1000–1999). Rows list cash, receivables, investments, fixed assets, accumulated depreciation, other long‑term assets; includes "Total assets" row (D72 = 15; E72 = 59; F72 = 25 — these are mapping numbers, not calculated totals).
- Page 2: Liabilities (2000–2999) and Equity (3000–3999). Rows include payables, accrued liabilities, deferred revenue, short/long‑term debt, split‑interest liabilities, net asset classes and totals (Total liabilities D124 = 28; Total net assets D141 = 36; Totals cross‑reference cells in E/F columns).
- Repeated header cells (version, standards, titles) are stored near row 9–12 (A9,B9,G9,H9,I9; A10,B10; A11,B11; I12) and are referenced elsewhere.

Pages 3–6 (Income statement / Revenue & Expense coding)
- Page 3–4: Contributions & Support (4000 series) and Earned Revenues / Government contracts (5000 series). Detailed breakdown of contribution types, donated goods/services, grants, indirect contributions, government vs. non‑government, and mapping to GAAP/Form 990.
- Pages 4–5–6: Program/Operating revenues, investment income, other revenue, special events, unrealized gains, net assets released (6900s), and "Total revenue, gains & other support" (D262 = 66; E262 = 12; F262 = 9).
- Pages 5–6: Expenses split into personnel-related (7000s) and non‑personnel (8100s etc.), contract services, salaries, benefits, travel, facility, other expenses, business expenses and totals (Total expenses D364 = 146; E364 = 44; F364 = 17).
- Page 6 includes non‑GAAP capitalization group (9800–9840) for fixed asset purchases and other non‑GAAP items.

Page 7 (Activity / Activity‑level coding)
- Provides activity code ranges (****-100 through ****-999) mapped to program departments, M&G, fundraising, cost pools, capital pools. Columns show OMB A‑122 cost principle numbers, United Way function codes, Policy/Template line item numbers, and federal/state Form 990 line item references.
- Column B lists activity descriptions; column E shows UWAACS function code ranges (e.g., 001–899 for program), column G shows Form 990 line item groupings (IIIa, IIId, etc.).

Sheet A-122 (OMB Circular A‑122 index)
- Enumerates OMB A‑122 cost principle topics. Column A contains sequential numeric indexes (many cells use incremental formulas like =A5+1), column B the cost descriptor, column C additional index numbers, column D related descriptors.
- Several formula patterns: many header/text cells are set to refer to anchor cells (see examples on Page 2 rows 83–86).

Key formulas & structural conventions
- Consistent use of absolute references for repeating header values: e.g., =A$9, =B$9, =E$10, =I$12 appear across multiple pages (examples shown on Page 2 rows 83–86 and repeated on pages 3–6). This centralizes the version/standards text.
- Sequential numbering in A‑122 sheet uses formulas like =A5+1, =C5+1 to auto‑increment index numbers.
- No aggregation/financial math formulas shown in the provided excerpt; "Total" rows contain mapping numbers (D/E/F) rather than SUM formulas — workbook is primarily a mapping/dictionary rather than a transactional ledger.

Notable constants / flags
- Many entries in columns D–I are numeric cross‑reference codes (GAAP exhibit numbers, Form 990 line numbers, internal activity/cost principle codes).
- Text flags include "n/a", "contra", "contra (…)", "capitalized", and "req" to indicate required mapping, contra accounts, or capitalization treatment.
- Account number ranges are listed in column H for accounting guide grouping (e.g., H16 = 1000‑1099, H22 = 1200‑1299).

Overall layout and purpose
- Purpose: standardized chart-of-accounts to external reporting and internal activity/cost allocation frameworks.
- Layout: rows = individual account mappings; columns provide multiple cross‑walks to reporting frameworks and policy guidance; top rows contain global header values referenced across sheets.
- Use case: look up an account (A/B/C), find how it should be reported on GAAP exhibits, Form 990/990‑EZ, which OMB A‑122 cost principle applies, United Way/accounting guide group, and template/line item references.

Spreadsheet overview — concise by sheet

Revenues by Source (Sheet)
- Purpose: Revenues as percent of the Unrestricted General Operating Fund (FY 1999–2000).
- Layout: Title rows in A1–A4, column headers in B6 ("Percentage of Total") and C6 ("Dollar Revenue"); revenue source names in A9:A13; footnotes and source lines in A17–A46.
- Key formulas:
  - Percent column (B9:B13): each = Cn / $C$15 (e.g., B9 = C9 / $C$15).
  - Dollar rows: C9 = 9,432,730 + 111,410,104 (-> 120,842,834), C10 = 18,185,022, C11 = 141,577,402 + 1 (-> 141,577,403), C12 = 10,599,091, C13 = 20,712,835.
  - Total (row 15): B15 = SUM(B9:B14), C15 = SUM(C9:C14).
- Key values: Total Revenues C15 = 311,917,185 (sum of provided revenue lines).
- Notes: Some cells use inline arithmetic (e.g., C9, C11); footnotes state rounding may cause small differences.

Expenditures by Function (Sheet)
- Purpose: Expenditures by program/function (FY 1999–2000).
- Layout: Titles A1–A4, headers B6 ("Percent of Total") and C6 ("Dollar Expenditures"), functions in A9:A17, footnotes A20–A49.
- Key formulas:
  - Percent column (B9:B17): each = Cn / $C$19 (e.g., B9 = C9 / $C$19).
  - Total (row 19): B19 = SUM(B9:B18), C19 = SUM(C9:C18).
- Key values: Dollar totals C9–C17 sum to C19 = 309,825,920.
- Notes: standard percent-of-total pattern; rounding footnote present.

Expenditures by Category (Sheet)
- Purpose: Expenditures by category (salaries, services, etc.) for the same fund/year.
- Layout: Titles A1–A4, headers B6 and C6, categories in A9:A13.
- Key formulas:
  - Percent column (B9:B13): each = Cn / $C$15.
  - Total (row 15): B15 = SUM(B9:B14), C15 = SUM(C9:C14).
- Key values: Category totals sum to C15 = 309,825,920 (Salaries = 229,427,498; Services = 42,607,188; etc.).
- Notes: C15 matches Expenditures by Function total.

General State Financial Aid (Sheet)
- Purpose: Breakdown of General State Financial Aid by community college (FY 1999–2000).
- Layout: Titles A1–A4, headers B6:D6 (B = General State Financial Aid $; C = % of total; D = % of Total FTEE), college rows A9:A23, totals in row 25.
- Key formulas:
  - Percent of total (C9:C23): each = Bn / $B$25.
  - Total (row 25): B25 = SUM(B9:B23) → B25 = 141,577,403; C25 = SUM(C9:C23); D25 = SUM(D9:D23).
- Notes: B21 implemented as 3,684,419 + 1 (-> 3,684,420). Footnote explains FY1997–98 FTEE used to calculate FY1999–2000 aid.

FTEE Calculation (Sheet)
- Purpose: Convert credit and non‑credit activity into FTEE and compute each college’s share.
- Layout: Titles A1–A4, detailed column headings in B7:I8; college rows A11:A25; totals in row 27; footnotes A29–A33.
- Key formulas / conversion factors:
  - Credit FTEE: Cn = Bn / 24 (credit hours divided by 24).
  - Non‑credit FTEE: Fn = En / 600 (non‑credit contact hours divided by 600).
  - College total FTEE Hn = Cn + Fn (for most rows).
  - Percent of total I n = Hn / H27.
  - Totals: B27 = SUM(B11:B26), C27 = SUM(C11:C26), F27 = SUM(F11:F26), H27 = SUM(H11:H26).
- Key values:
  - Total FTEE (H27) computed from rows = 72,409.4683 (sum of H11:H25 using the provided data; note one row formula affects this — see anomalies).
- Notable anomalies:
  - Row NCC-04 (row 14) has H14 = -C14 - F14 (negative sign) — likely an error.
  - E27 = SUM(E11:E26) + 1 (extra +1 adjustment).
  - Several formulas use leading “+” (e.g., =+C11/H11) — stylistic but not functional.
  - Footnote warns percentages may not equal 100% due to rounding; another note about using FY1997–98 FTEE.

FTEE Graph (Sheet)
- Purpose: Figure/chart for FTEE Calculation by college (title and source lines only).
- Layout: Simple chart/figure sheet with title A1–A4 and source in A28:A29.

General observations and data sources
- All tables follow the same design: top title rows, two main columns (percent and dollar/hours), percent column computed as item ÷ sheet total (absolute reference to the total cell).
- Totals are at the bottom of each table and used as denominators for percent calculations.
- Data source repeated in footers: Iowa Department of Education, Annual Report Fiscal Year 2000 (Unrestricted General Fund, AS-15E / Fund 1) and Bureau of Community Colleges / MIS pro‑rate sheets.
- Rounding and a few manual adjustments (+1 or arithmetic inline) are present; one formula yields a negative FTEE (probable data/formula error) and should be reviewed.

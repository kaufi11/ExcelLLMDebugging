Summary of workbook structure and contents (concise)

Overall layout
- Workbook organized as a multi-page coding reference: Sheets titled Page 1 through Page 6 (presented as "pages 1-6" in the data), plus "page 7" and "A-122".
- Each page is a structured chart of accounts / coding cross‑reference for a nonprofit: Balance Sheet (1000–3999), Income/Activity codes (4000–9999), Expenses and Non‑GAAP items.
- Consistent column layout across pages (columns A–I): A/B = account number(s), C = description/line item, D–G = internal cross‑references (GAAP exhibit, Form 990/990‑EZ line, cost principles, activity codes), H = internal accounting guide ranges, I = templates/notes.

Sheets and primary content
- Page 1 (Balance Sheet — Assets): Lists asset account ranges (1000–1999), detailed accounts (Cash, Receivables, Investments, Fixed operating assets, Accumulated depreciation), with reference columns D (GAAP exhibit/line), E/F (Form 990 line numbers), H (internal 4‑digit ranges such as 1000–1099), many G/I cells marked "n/a".
  - Key totals: "Total assets" at row C72 with references in D72, E72, F72.
  - Header cells include version/metadata in row 9 (A9 = version 3.0, B9 = version 2.2, G9 = OMB A‑122, H9 = United Way, I9 = Tennessee).
- Page 2 (Balance Sheet — Liabilities & Equity): Continues liabilities (2000–2999) and equity (3000–3999); rows for Total liabilities (C124), Total net assets (C141) and "Total liabilities & net assets" (C143).
  - Page 2 contains explicit formulas that pull header/metadata from Page 1 top cells (e.g., A83 = A$9, B83 = B$9, etc.), demonstrating reuse of top‑of‑sheet metadata across pages.
- Pages 3–6 (Income Statement / Financial Activities and Expenses): Detailed mapping of revenue accounts (4000–5999), earned revenues, other revenue, unrealized gains, net assets releases (Page 3 & 4), and expense categories (7000–9999) including personnel vs non‑personnel, business/non‑GAAP items, special events, capital purchases (Pages 4–6).
  - Key totals: "Total revenue, gains, & other support" (C262, row D262 = 66 etc.), and "Total expenses" (C364).
  - Pages 3–6 repeat header formulas (e.g., A154/A204/A274/A314 = A$9 etc.) to echo the same metadata across sheets.
- Page 7 (Revenue & Expense Coding by Activity): Maps activity code ranges (****-100 to -999) to program/administrative/fundraising/cost‑pool labels and shows activity coding rules (UWAACS function codes, OMB A‑122 cost principles, Form templates references).
  - Contains a column heading index and mapping of activity ranges to interpretation (program departments, M&G, fundraising, cost pools), plus summary rows (Total program services, etc.).
- A-122 sheet: Extract of OMB Circular A‑122 cost principle index. Left column uses simple increment formulas to build paragraph numbers (A6=A5+1 etc.). Lists cost principle descriptors and cross‑references.

Key formulas and mechanics
- Header propagation: Many cells on Pages 2–6 use direct formulas referencing top metadata on Page 1 (e.g., =A$9, =B$9, =G$9, =H$9, =I$9 and similarly for row 10/11), ensuring consistent version and guide labels across sheets.
- Sequential numbering on A-122: Rows use arithmetic formulas (e.g., =A5+1, =C5+1) to generate a sequence of paragraph numbers.
- Totals are represented as labeled rows (Total assets, liabilities, net assets, revenue, expenses) with numeric cross‑reference codes in D/E/F columns (not summation formulas shown in supplied extract).
- Many mapping fields contain static lookups (hardcoded account ranges, GAAP exhibit numbers, Form 990 line ids, "n/a" flags) — the sheet is primarily a reference table rather than a calculation workbook.

Notable values / conventions
- Account numbering:
  - Balance sheet: 1000–3999 (Assets 1000–1999, Liabilities 2000–2999, Equity 3000–3999).
  - Financial activities (revenues/expenses): 4000–9999, with detailed subranges (e.g., 4000–4199 contributions, 5000–5499 earned revenues, 7000–8999 expenses).
- Cross‑references included for each line: GAAP exhibit numbers (col D), Form 990/990‑EZ line numbers (cols E/F), UWAACS/internal ranges (col H), cost principles or activity coding (col G/I).
- Metadata/versioning in row 9–11 (versions, policy references, jurisdiction: United Way, Tennessee, OMB A‑122).

Overall purpose
- This workbook is a standardized coding dictionary / chart of accounts and crosswalk for nonprofit financial reporting: maps account numbers to descriptions and to multiple external/internal reporting references (GAAP exhibits, Form 990 lines, OMB A‑122 cost principles, United Way accounting guide, internal activity codes). It’s designed primarily for lookups and consistency across reports rather than heavy numeric calculation.

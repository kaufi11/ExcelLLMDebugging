Overall layout

- Each annual block uses columns:

  - D: Number tested (raw counts).
  - E: Number passing (raw counts).
  - F: Percent passing (formula-driven).

Key formulas and patterns
- Percent by grade: F# = E#/D# (percent passing = passing / tested) for each grade row.

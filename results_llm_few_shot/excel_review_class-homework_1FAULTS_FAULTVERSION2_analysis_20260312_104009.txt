1) Sheet Q'back — wrong completion percentage for Bryant
- Faulty cell: H5
- Error: H5 uses =D5/C5*180 (multiplies by 180 instead of 100). This inflates the completion % and therefore the Efficiency (L5).
- Fix (repair formula): =D5/C5*100
- Why: Completion % = Completions / Attempts × 100. Use 100 (percent), not 180. After fixing H5, L5 (which uses H5) will compute correctly.

(If H5 was a copy-paste mistake, copy the corrected H5 down or re-enter the formula for each row: Hn = Dn/Cn*100.)

2) Sheet lookup — missing/absent formulas for totals, averages and grade lookup

- Fix formulas:

  - G4 (Average /100 = average per 100): =F4/4  — copy down G4:G9
    - (Alternative if you want percent-of-400: =F4/400*100 — but F4/4 returns the average score out of 100 which matches your header.)

    - Make sure the grade table in F12:F18 is sorted ascending and represents the lower-bound points for each letter (VLOOKUP with TRUE needs ascending order).
    - Alternative robust formula: =INDEX($G$12:$G$18, MATCH(G4,$F$12:$F$18,1))

- Why: Totals and averages must be calculated from the four course scores. The VLOOKUP (approximate match) returns the letter based on the numeric average and the threshold table.

Summary — main errors
1. Overtime column (G3:G8) uses the wrong E-row in the last term (it points to the next row or to E10). So overtime is being calculated with the wrong Week 4 value.
2. Smith's Gross Pay (C16) references the wrong total (uses F5 instead of F4).

Why this is wrong (short)
- G3:G8 should calculate MAX(EcurrentRow-40,0), but they reference E of the next row (or E10), so overtime for each employee is using the wrong Week‑4 hours.
- C16 mixes Smith’s overtime/total with Jones’ F5, so Smith’s gross pay is wrong.

Correct formulas (replace the current ones)

- G3:
  =MAX(B3-40,0)+MAX(C3-40,0)+MAX(D3-40,0)+MAX(E3-40,0)

- G4:
  =MAX(B4-40,0)+MAX(C4-40,0)+MAX(D4-40,0)+MAX(E4-40,0)

- G5:
  =MAX(B5-40,0)+MAX(C5-40,0)+MAX(D5-40,0)+MAX(E5-40,0)

- G6:
  =MAX(B6-40,0)+MAX(C6-40,0)+MAX(D6-40,0)+MAX(E6-40,0)

- G7:
  =MAX(B7-40,0)+MAX(C7-40,0)+MAX(D7-40,0)+MAX(E7-40,0)

- G8:
  =MAX(B8-40,0)+MAX(C8-40,0)+MAX(D8-40,0)+MAX(E8-40,0)

- C16 (Smith Gross Pay):
  =(F4+G4)*B16

Explanation of fix (short)
- Each overtime formula must use that same row’s Week‑4 cell (E row for that employee). Fixing the row reference corrects the calculation.
- Smith’s gross pay should use Smith’s totals (F4 and G4), not F5. Changing F5 to F4 fixes his pay.

Optional tip
- If you copy G3 down correctly once (with relative references to E3), Excel will fill the right row references automatically.

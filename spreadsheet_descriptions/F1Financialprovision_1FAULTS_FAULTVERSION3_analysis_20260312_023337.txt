Summary of workbook structure and contents

General
- Purpose: Collection of local authority financial and direct payment data for the period 1 April 2002 to 31 March 2003 (cash or vouchers), with contact details and two data sections: A: Financial Provision and B: Direct Payments.
- Layout style: each sheet uses labeled rows for categories and columns for counts/amounts; totals calculated with SUM formulas.

Sheet: Contact details
- Cells:
  - B2: Name of Local Authority
  - B3: Contact name
  - B4: Telephone number
  - B9: Submission e‑mail: SWStat@scotland.gsi.gov.uk
- Purpose: metadata for the form submitter.

Sheet: A Financial Provision
- Title: A: FINANCIAL PROVISION.
- Applies to payments made in FY 2002/03 under:
  - Section 12 of the Social Work (Scotland) Act 1968 (Table 1).
  - Sections 22 and 25 of the Children (Scotland) Act 1995 (Table 2).
- Table 1 (payments under Section 12)
  - Columns: B = Number of Payments, C = Grant/Loan (amount), D = Recovered (amount).
  - Rows (A8–A13): Energy, Rent, Travel, Food, Asylum Seekers, Other.
  - Row A14: TOTAL
    - B14 = SUM(B8:B13)
    - C14 = SUM(C8:C13)
    - D14 = SUM(D8:D13)
- Table 2 (payments under Sections 22 & 25)
  - Header: B19 indicates Number of Payments.
  - Columns: B = Section 22 (counts), C = Section 25 (counts), D = Grant/Loan (amount), E = Recovered (amount).
  - Rows (A22–A28): Energy, Rent, Travel, Food, Day Care for Children in need, Asylum Seekers, Other.
  - Row A29: TOTAL
    - B29 = SUM(B22:B28)
    - C29 = SUM(C22:C28)
    - D29 = SUM(D22:D28)
    - E29 = SUM(E22:E28)

Sheet: B Direct Payments
- Title: B: DIRECT PAYMENTS. Describes direct payments under Section 12B (2002/03).
- Structure: two parallel groupings by client age groups:
  - Columns B–E: Number of clients by age (B=16–17, C=18–64, D=65+, E=Total).
  - Columns F–I: Value of payments by age (F=16–17, G=18–64, H=65+, I=Total).
- Client rows (A10–A14): People with Physical Disabilities; Learning Disabilities; Mental Health Problems; Disabled Parents*; Other (specify).
- Row A15: Total
  - B15 = SUM(B10:B14)
  - C15 = SUM(C10:C14)
  - D15 = SUM(D10:D14)
  - E10–E14 each = SUM(Bx:Dx) (e.g., E10 = SUM(B10:D10)); E15 = SUM(E10:E14)
  - F10–F14 etc. with I10–I14 each = SUM(Fx:Hx) (e.g., I10 = SUM(F10:H10)); F15 = SUM(F10:F14), G15 = SUM(G10:G14), H15 = SUM(H10:H14)
  - Note: I15 is defined as =COUNT(I10:I14) in the sheet (appears to be an anomaly — likely intended to be SUM(I10:I14)).

Other notes
- All monetary totals use SUM over the category ranges.
- The sheet includes explanatory footnotes (e.g., disabled parents reference CCD9/2001) and a prompt to specify any entries in the “Other” category.
- Suggested review: correct I15 from COUNT to SUM if total value of payments is desired.

Summary of the errors
- Column I in the FRF sheet contains literal text like "FORMULA =SUM(H5-J5)" instead of a working formula. Those cells are stored as text, so no subtraction is being calculated.
- Row 20 (I20) uses =COUNT(H20-J20) (also shown as text) — COUNT is the wrong function here; it counts values, it does not compute a difference.
- Result: the "Height off" (I) values are not computed; some rows show N/A or text instead of numeric differences.

Why this is wrong (short)
- Excel treats "FORMULA =SUM(H5-J5)" as text (or as a user string). You need an actual formula starting with = followed by the expression. COUNT(...) will not give the numeric difference H - J.

Simple fix (step‑by‑step)
1. Change the cell contents to an actual subtraction formula. For the first computed row (I5) enter:
   =H5 - J5
2. Press Enter, then drag (or double‑click the fill handle) down the I column for all rows that need the calculation (I5:I23 or as far as your data goes). This will produce numeric results for each row.
3. For the mistaken COUNT (I20), replace it with:
   =H20 - J20
4. Make sure the cells are formatted as Number (Home → Number) so results show as numbers not text.
5. If many cells show the literal "FORMULA ..." text, a quicker method: remove the leading text/quote and convert to formulas
   - If cells literally start with an apostrophe (') remove it (Edit cell or use Find & Replace to remove leading apostrophes).
   - Or overwrite the column with the proper formula and fill down.

Optional cleaner formula
- You can also use =SUM(H5,-J5) but =H5-J5 is clearer and recommended.

Quick check

O46: Wrong function used in numerator — formula uses COUNT instead of SUM.

Error (current):
=(COUNT(B46:F46)+COUNT(G46:M46)+N46)/(COUNT(B46:F46)*100+COUNT(G46:J46)*15+COUNT(K46:M46)*30+COUNT(N46)*150)*100

Fix (corrected):
=(SUM(B46:F46)+SUM(G46:M46)+N46)/(COUNT(B46:F46)*100+COUNT(G46:J46)*15+COUNT(K46:M46)*30+COUNT(N46)*150)*100

Why:
COUNT returns the number of numeric cells (e.g. 5), not the total points. Using COUNT in the numerator makes the overall grade far too small. Replace COUNT with SUM so the numerator adds the students' scores. Then copy the corrected formula to any other row(s) with the same mistake.

Summary — what’s wrong
- The Balance Sheet has manual / wrong cash numbers instead of linking to the Financial summary. That causes a $500 mismatch for Dec 31, 2001:
  - Balance Sheet G4 = 12323.99 (hard‑typed) but Financial summary I35 (the 2001 Checking account) = 12823.99. Balance Sheet is using a wrong value, so assets/totals are incorrect.
- Several Balance Sheet cash cells are inconsistent (some are hard numbers, some are formulas). This invites transcription errors.

Why it’s an error
- The Financial summary is the source of the checking/money‑market balances. The Balance Sheet should reference those cells. Hard‑typing numbers in the Balance Sheet breaks the live link and lets values diverge (here by $500).

Fix (easy)
1. Replace the hard numbers on the Balance Sheet with formulas that point to the Financial summary:
   - E4  → ='Financial summary'!E35  (Dec 31, 1999 checking)
   - F4  → ='Financial summary'!G35  (Dec 31, 2000 checking)
   - G4  → ='Financial summary'!I35  (Dec 31, 2001 checking) ← fixes the $500 mismatch
   - H4  is already ='Financial summary'!K35 (keep)
   - E5  → ='Financial summary'!E36  (1999 money market)
   - F5  → ='Financial summary'!G36  (2000 money market)
   - G5  → ='Financial summary'!I36  (2001 money market)
   - H5  → ='Financial summary'!K36  (already set)

2. After changing those cells, recalc and confirm:
   - Balance Sheet Total Current Assets (E6:F6:G6:H6) and TOTAL ASSETS (E7:F7:G7:H7) will update automatically.
   - The $500 difference will disappear if the Financial summary figure is correct.

Extra check you should do
- If the $500 remains unexplained after linking, review the detailed receipts/expenditures (Stork sheets) for any missing/misposted $500 transaction (search for 500, deposits/checks around the 2001 year end). If Financial summary I35 is wrong, correct the entry in the Financial summary/detail sheet instead of the Balance Sheet.

Why this solution works (short)
- Keeping the Balance Sheet cells linked to the Financial summary ensures single source of truth. Any correction in the Financial summary automatically flows to the Balance Sheet and prevents manual transcription errors.

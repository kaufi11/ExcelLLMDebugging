Findings (short)
- Error: On the In-kind contribution template, cell F5 has the wrong formula: =571*E5. It should be =D5*E5.
- Effect: F5 shows 57,100 instead of 5,000, inflating the In-kind totals.

Fix (do this)
1. Go to the In-kind contribution template.
2. Select cell F5.
3. Replace the formula with: =D5*E5
4. Press Enter.

What changes after fix (numbers)
- F5 becomes 50 * 100 = 5,000 (was 57,100).
- Labor total F16 becomes 9,500 (was 61,600).
- Running totals:
  - G16 = 9,500
  - G29 = 18,500
  - G41 = 83,600
  - Grand total F43 = 83,600
(Previous grand total was 135,700.)

Why this fixes it (simple)
- All per-line contribution/funding rows should multiply the rate (D) by quantity/hours (E). F5 used a hard-coded constant (571) instead of the D5 cell reference, producing a wrong value and wrong totals. Using =D5*E5 makes the sheet consistent and totals correct.

Prevention tips
- Use Excel’s Formulas > Show Formulas to scan for hard-coded numbers in formulas.
- Search (Ctrl+F) for “=571” or numeric literals if you suspect typos.
- Keep a consistent formula pattern (copy F6 to F5 if needed) so per-line formulas match.

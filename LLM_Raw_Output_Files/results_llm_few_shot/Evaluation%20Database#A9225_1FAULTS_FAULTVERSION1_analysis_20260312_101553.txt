Faulty cell
- I19

Error type
- Range / transcription error (off-by-one)

What’s wrong (short)
- I19 is =SUM(I8:I17) and therefore omits the value in I7 (52). All other column totals use row 7 as the first row (e.g. G19 uses G7:G17). So the I column total is too small.

Fix (direct)
- Replace formula in I19 with:
  =SUM(I7:I17)

Why this fixes it
- Including I7 ensures the total sums the same set of rows as the other rating columns (rows 7–17), so column totals match the per-row totals in L7:L17.

Quick extra checks (recommended)
- Scan G19:K19 to confirm consistent ranges (they should all be X7:X17).
- Consider summing the row totals (L7:L17) as a cross-check or convert the data to an Excel Table so totals auto-adjust if you add/remove rows.

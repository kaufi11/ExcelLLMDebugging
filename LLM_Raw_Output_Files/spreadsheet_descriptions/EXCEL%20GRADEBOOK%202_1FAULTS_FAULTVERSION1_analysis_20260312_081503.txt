Sheet1 — layout and key values
- Row 1: headers for categories and weights. Category weights are stored in C1 = 0.2 (Homework), K1 = 0.15 (Classwork), S1 = 0.25 (Quizzes), Z1 = 0.4 (Tests).
- Columns (grouping):
  - A: Student names (A2:A6).
  - Homework: B:G (6 scores), H = Homework Total (SUM B:G), I = Homework Average that drops the lowest ((SUM(B:G)-MIN(B:G))/(COUNT(B:G)-1)).
  - Classwork: J:O (6 scores), P = Classwork Total (SUM J:O), Q = Classwork Average that drops the lowest (same pattern as I).
  - Quizzes: R:V (5 scores), W = Quiz Total (SUM R:V), X = Quiz Average that drops the lowest (same pattern as I/Q except one anomaly noted below).
  - Tests: Y, AA, AB (3 scores), AC = Test Total (SUM Y:AB), AD = Test Average (AVERAGE(Y:AB)) — tests use a normal average (no drop).
  - AE = Final Grade computed as weighted sum: (HomeworkAverage*C$1) + (ClassworkAverage*K$1) + (QuizAverage*S$1) + (TestAverage*Z$1). The formulas use row-anchored weight references (e.g., C$1) so they copy down correctly.

Formulas and patterns
- Totals: H/P/W/AC use SUM of their respective score ranges.
- Category averages: I/Q/X generally use the pattern (SUM(range)-MIN(range))/(COUNT(range)-1) to drop the lowest score.
- Tests average AD uses AVERAGE(range) (no lowest-score drop).
- Final grade AE = I*C$1 + Q*K$1 + X*S$1 + AD*Z$1 (weights anchored to row 1).

Found one clear error.

What’s wrong
- Sheet "B Direct Payments", cell I15 contains: =COUNT(I10:I14)
  - COUNT counts how many numeric cells are in I10:I14 (e.g. returns 3), not the total value of payments.
  - You need the total money paid, so COUNT gives a wrong result.

Fix
- Change the formula in I15 to:
  =SUM(I10:I14)

Why this fixes it (short)
- SUM adds the amounts in I10:I14 (the total value of payments). COUNT only returns the number of numeric cells, not the sum.

Quick steps to correct
1. Select cell I15.
2. Replace =COUNT(I10:I14) with =SUM(I10:I14) and press Enter.
3. Optionally format I15 as Currency/Number to match other totals.

Extra check (recommended)
- Search the workbook for any other COUNT(...) used in a totals row and replace with SUM(...) where you mean to add values. Use Ctrl+F for "COUNT(" or Formulas → Show Formulas.

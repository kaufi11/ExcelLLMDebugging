Findings (short)
- One error: cell F44 (Grade 8, Spring 1999) contains =E44/D45. It divides Grade 8 passing by Grade 10 tested, so the percent is wrong.
- All other F formulas appear correct (each should be =Erow/Drow).

Why it's wrong (simple)
- Numerator is E44 (Grade 8 passing = 208,412). Denominator is D45 (Grade 10 tested = 233,657). That gives ~89.2%, but the correct percent is 208,412 / D44 (274,873) = ~75.8%. So the value is inflated.

Quick fix (exact steps)
1. Select cell F44.
2. Replace the formula with:
   =E44/D44
3. Press Enter. Format F44 as Percentage (same format as the other percent cells).

Optional improvements (recommended)
- Prevent divide-by-zero showing errors:
   =IF(D44=0,"",E44/D44)
  or
   =IFERROR(E44/D44,"")
- To ensure no other mistakes, re-fill the percent column for each year: in the first percent cell of each block type =E# / D# and drag/fill down for that block (or select the whole F column percent rows and Fill Down).
- Use Formula Auditing → Trace Precedents or Home → Find → search “/D” in column F to locate any other mis‑references.

Why this solution
- =E44/D44 matches the intended pattern (passing / tested) so percent will reflect the correct base population.
- IF/IFERROR avoids #DIV/0! if a tested count is zero.
- Re-filling or auditing ensures consistency across all grade rows so no other row refers to the wrong denominator.

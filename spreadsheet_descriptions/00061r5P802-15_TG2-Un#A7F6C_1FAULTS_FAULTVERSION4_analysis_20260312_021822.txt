Overview
- Workbook contains 2 sheets: "Profs" and "URLs".
- "Profs" is a contact directory of academics and students (rows ~1–75). "URLs" is a one‑column list of project / institution names with their web addresses in the next row.

Sheet: Profs
- Header row (row 1) defines columns:
  A: # (row number) | B: sent | C: Name | D: Position | E: University | F: Email address | G: Suggested by | H: Personal or Related Web Page | I: Address (line 1) | J: Address (line 2) | K: City | L: S/P (State/Province) | M: CNTY (Country) | N: Zip | O: Phone | P: FAX
- Data range: starts at row 2 and runs through row 75 (some blank rows in between).
- Row numbering (col A): mostly auto-increment formulas (e.g., A3 = A2 + 1, A4 = A3 + 1, etc.). A2 is the seed value (1). There are a few anomalies: A38 shows a manual value/formula =6 (breaks sequence) and there are skipped row numbers (rows 50–51 are blank while A52 references A49+1).
- Column B ("sent") uses markers like "x" and "u".
- Core contact fields:
  - Column C: full names (professors, researchers, students).
  - Column D: titles/positions (Professor, Assistant Professor, Graduate Student, Director, etc.).
  - Column E: institution/department.
  - Column F: email addresses; some entries are missing or marked "?".
  - Column G: who suggested/contact source.
  - Column H: personal or project web pages for selected people.
  - Columns I–P: postal address, city, state, country, zip, phone and fax when available.
- Examples / key entries:
  - Row 2: Roy D. Yates — Director, Rutgers — ryates@winlab.rutgers.edu (suggested by Kevin Marquess).
  - Rows 3–4: Andy Ogielski, Christopher Rose — Rutgers.
  - Row 17: Dr. Kaveh Pahlavan (WPI) includes full address, web page, phone.
  - Rows 36–37: MIT Media Lab contacts (Joseph Paradiso, Nicholas Negroponte) include web pages.
  - Several entries include phone/fax (e.g., row 35: Kevin Karcz phones in O35/P35).
- Data quality notes:
  - Mostly consistent columns but many blank cells and occasional multi-line addresses split across I/J.
  - Some emails and other fields are missing or set to "?".
  - Numbering breaks and blank rows introduce gaps in the sequence.

Sheet: URLs
- Single-column list in column A (label in odd rows, URL in the following row).
- Contains project and institution names with their URLs, e.g.:
  - Worcester Polytechnic Institute — http://www.wpi.edu/
  - MIT LCS — http://www.lcs.mit.edu/
  - MIT Media Lab — http://www.media.mit.edu/
  - WIND, CM, CAG, COMET/CellularIP, CMU Monarch, MosquitoNet, etc.
- Useful as a quick reference of related projects/sites referenced from the Profs sheet.

Overall layout & usage
- A straightforward contact list with a header row and one contact per row; numbering is intended to be automatic but has some manual overrides/gaps.
- Primary uses: emailing or tracking suggested academics and project web links.
- Recommend: normalize numbering (fix A38 and gaps), consolidate multi-line addresses into consistent fields, and fill missing emails where available.

High-level overview
- Workbook contains 3 sheets: "Summary Info", "Participant List", and "Speakers & Mentors".
- Layout is a mix of narrative text, tabular summaries, and detailed participant/contact lists. Dates are stored as full date‑time strings; award amounts are numeric.

Sheet: Summary Info
- Top (A1:A9) — narrative summary of selection process and response counts (e.g., "1387 or ~85% ..." and "238 or ~15% ...", 66 interested, 21 unable to attend).
- Directorate table (rows 11–20): columns:
  - A = Directorate name (OPP, EHR, SBE, GEO, BIO, CISE, ENG, MPS)
  - B = "would like to attend" (counts by directorate)
  - C = "invited" (counts by directorate)
  - D = "eligible invitees"
  - J contains explanatory notes (e.g., “Information available from the NSF website”, space/waiting‑list notes).
- Key formulas:
  - B20 = SUM(B12:B18) → total "would like to attend" (sheet value = 54).
  - C20 = SUM(C12:C19) → total invited (sheet value = 193).
  - D20 = SUM(D12:D19) → total eligible invitees (sheet value = 238).
- Carnegie classification table (rows 22–30): columns E–G give counts by institution type (Specialized school, museums, Baccalaureate/Lib Arts, Master's, Doctoral/Research Intensive, Doctoral/Research Extensive, HBCUs). 
  - E30 = SUM(E23:E29) → interested (sheet value = 66).
  - F30 = SUM(F23:F29) → invited (sheet value = 193).
  - G30 = SUM(G23:G29) → eligible invitees (sheet value = 238).
- Notes: Asterisked Carnegie link and some blank rows used for spacing.

Sheet: Participant List
- A header row (row 1–2) defines column meanings. Key columns include:
  - A: sequence / index
  - B: "# by Directorate" (an incremental count)
  - C: Institution Type (BCLA, MCU-1, DRU‑i, R1, etc.)
  - D: Project Title/Short description
  - E: NSF Program (e.g., IBN, MCB)
  - F: Directorate (BIO, CSE, ENG, GEO, MPS, etc.)
  - G: NSF PD & Program (program officer text)
  - H / I: start date / end date (date/time strings)
  - J: Award amount (numeric)
  - K: PI name
  - L: PI email & contact
  - M: Address / institution details
  - N / O: occasional discipline codes or classification notes (e.g., "1136 SIGNAL TRANSDCTN/CELL REGULATN")
- Rows 3–63: detailed participant records (one per awarded PI), with mixed text and numeric fields; amounts vary by row (examples J3=541,158; J4=549,995, etc.).
- Rows ~68–74: "ON WAITING LIST" block with additional candidate records.
- Bottom (rows ~80–86): legend/definitions for Carnegie abbreviations (BCLA, MCU‑1, DRU‑i, M, R1s, R1).
- No explicit formulas shown in this sheet (mostly raw data).

Sheet: Speakers & Mentors
- Simple contact list of organizing committee, dinner speaker, administration, education speakers, and mentors.
- Columns:
  - A: Name (group headings embedded as plain text rows)
  - B: Affiliation
  - C: Position / role
  - D: e‑mail
- No formulas; grouped by role headings (e.g., "Organizing Committee", "Mentors").

Notable/Key values (from formulas and summary)
- Eligible invitees (total): 238 (D20 / G30).
- Invitations sent: 193 (C20 / F30).
- Interested / would attend (by Carnegie sum): 66 (E30). Note: the Directorate "would like to attend" sum (B20) = 54 — the workbook contains both counts (different aggregations) and the narrative rows that cite 66 interested and 21 "would like to attend but can't."

Overall structure and use
- Summary Info: high‑level narrative and two summary tables with SUM formulas to aggregate counts.
- Participant List: master data table for invited/interested PIs (detailed contact, award, program info), plus waiting list.
- Speakers & Mentors: contact roster for event staff/speakers/mentors.
- The workbook supports selection/invitation tracking (eligible → invited → responses) and provides the participant/contact data needed for workshop logistics.

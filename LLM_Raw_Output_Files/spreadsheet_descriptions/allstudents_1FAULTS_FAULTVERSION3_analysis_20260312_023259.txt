Spreadsheet: single sheet named "2001 - 1994"

Purpose
- Shows Texas Assessment of Academic Skills (TAAS) counts and percent passing by grade for spring administrations 1994–2001 (All Students). Includes totals by year and two footnotes about Grade 8 tests and 2001 being preliminary.

Overall layout
- Data is arranged in repeating year blocks (one block per spring year): headings at A6 (Spring 2001), A21 (Spring 2000), A35 (Spring 1999), A49 (Spring 1998), A63 (Spring 1997), A77 (Spring 1996), A91 (Spring 1995), A105 (Spring 1994).
- Each year block uses three column header rows (e.g., D8:E8:F8 = Number/Number/Percent and D9:E9:F9 = Tested/Passing/Passing).
- Grades listed in column C (Grade 3, Grade 4, Grade 5, Grade 6, Grade 7, Grade 8*, Grade 10, All Grades).
- Numeric counts: Tested = column D, Passing = column E.
- Percent passing: column F, calculated as Passing / Tested (displayed as a percent).

Formulas and totals
- Grade percent cells use F = E / D (e.g., F10 = E10/D10, F11 = E11/D11, etc.).
- Year totals use SUM across the Tested and Passing ranges (e.g., D17 = SUM(D10:D16); E17 = SUM(E10:E16)) and F17 = E17/D17.
- This pattern repeats for each year:
  - 2001 totals: D17 = SUM(D10:D16), E17 = SUM(E10:E16), F17 = E17/D17
  - 2000 totals: D32 = SUM(D25:D31), E32 = SUM(E25:E31), F32 = E32/D32
  - 1999 totals: D46/E46/F46, 1998: D60/E60/F60, 1997: D74/E74/F74, 1996: D88/E88/F88, 1995: D102/E102/F102, 1994: D116/E116/F116.

Key sample values (2001)
- Grade 3: Tested 265,357; Passing 205,320; Percent = E10/D10.
- Grade 10: Tested 244,482; Passing 194,475; Percent = E16/D16.
- All Grades (2001): totals computed from D10:D16 and E10:E16; percent = E17/D17.

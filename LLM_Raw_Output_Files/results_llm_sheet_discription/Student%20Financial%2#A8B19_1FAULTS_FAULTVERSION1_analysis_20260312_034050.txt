1) F8 (Room and Board On‑Campus, 2003–2004) is wrong
- Problem: F8 contains the hard value 4892 but the other years use formulas (G8 = G9+G10, etc.). F9 (Room) + F10 (Board) = 2466 + 1806 = 4272, not 4892. So F8 is inconsistent with its components.

- Fix: Replace the 4892 with the formula =F9+F10.
  - In Excel: select F8 → type =F9+F10 → Enter.
- Result: F8 will always equal the sum of Room and Board for 2003–2004 and any dependent totals update correctly.

  - In Excel: select H12 → type =H6+H8+H7 → Enter.

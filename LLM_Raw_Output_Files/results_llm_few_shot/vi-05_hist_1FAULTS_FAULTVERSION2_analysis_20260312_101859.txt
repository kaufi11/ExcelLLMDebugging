Findings (short)
- Faulty cell: 2003-04 sheet — H14.
- Error: wrong function used. H14 is =COUNT(H10:H13) but should sum the four dollar amounts.
- Effect: H14 returns 4 (count of cells) instead of the subtotal (~5435.23). That makes H21 (Total resident per term) and H25 (Total non‑resident per term) and any other dependent totals wrong.

Fix (one-line)
- Replace H14 with: =SUM(H10:H13)

Why this fixes it (short)
- COUNT returns how many cells contain numbers (4). SUM adds the monetary values. Changing to SUM gives the correct subtotal; dependent cells that add H14 (H21, H25) will then recalculate correctly.

Quick verification steps
1. Edit H14 → enter =SUM(H10:H13).
2. Check H14 (should be ~5435.23). 
3. Check H21 and H25 update to expected totals (H21 ≈ H14 + H20; H25 ≈ H21+H23+H24).

Optional recommendation
- Scan other subtotal cells for accidental use of COUNT instead of SUM (common slip).

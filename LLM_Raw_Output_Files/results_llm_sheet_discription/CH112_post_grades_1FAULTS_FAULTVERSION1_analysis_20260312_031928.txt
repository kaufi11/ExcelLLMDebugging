Findings (short)
- The overall-grade formula in O13 is wrong. It uses division (/) in the denominator where multiplication (*) is required. That makes the denominator a tiny number and O13 comes out completely incorrect.
- Everything else appears intentionally designed (COUNT used so missing items reduce the possible points). Still: use the correct O4 formula for all rows to avoid copy/typing errors.

What’s wrong (simple)
- Wrong piece in O13:
  (COUNT(B13:F13)/100 + COUNT(G13:J13)/15 + COUNT(K13:M13)/30 + COUNT(N13)/150)/100
  should be:
  (COUNT(B13:F13)*100 + COUNT(G13:J13)*15 + COUNT(K13:M13)*30 + COUNT(N13)*150)
- In short: you divided the counts by the max-points instead of multiplying counts by the max-points.

Correct formula
Replace O13 with the same formula used elsewhere (for example O4):
=(SUM(B13:F13)+SUM(G13:M13)+N13)/(COUNT(B13:F13)*100+COUNT(G13:J13)*15+COUNT(K13:M13)*30+COUNT(N13)*150)*100

How to fix (steps)
1. Select cell O13.
2. Paste the corrected formula above and press Enter.
3. To avoid repeating mistakes, copy O4 (which is correct) and drag / paste it down the O column (O4:O46) so every row uses identical formula.

Why this fixes it (one line)
- The overall percent = earned points ÷ possible points × 100. Multiplying the counts by each assignment’s max gives the possible points; dividing by counts made the denominator wrong, inflating the percent.

Optional improvements
- Use the O4 formula and fill down (instead of editing single cells) so one typo doesn’t reappear.
- Consider using named constants (ExamMax=100, HWsmallMax=15, HWbigMax=30, FinalMax=150) or a single weighted calculation to make the formula clearer and less error-prone.

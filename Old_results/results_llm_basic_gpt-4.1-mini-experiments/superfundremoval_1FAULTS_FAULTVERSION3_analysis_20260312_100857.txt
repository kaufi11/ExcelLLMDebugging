### Solutions:

1. **Fix rounding error in total weighting checks (F24, F36):**

   Replace the exact equality check with a tolerance check, for example:

   ```excel
   =IF(ABS(SUM(F16:F22)-1)>0.001,"ERROR","100%")
   ```

   and similarly for F36:

   ```excel
   =IF(ABS(SUM(F28:F34)-1)>0.001,"ERROR","100%")
   ```

   This allows small rounding differences and prevents false errors.

2. **Correct formula in G21:**

   Change from:

   ```excel
   =IF(C21<>"yes",(1*F21),IF(C21<>"no",(0*F21),""))
   ```

   To the consistent pattern used elsewhere:

   ```excel
   =IF(C21="yes",(1*F21),IF(C21="no",(0*F21),""))
   ```

### Explanation:

- The total weighting checks compare the sum of weights to 100% (which is 1 in Excel). Due to decimal rounding, sums like 0.9996 are not exactly 1, causing the formula to return "ERROR". Using `ABS` and a small tolerance (e.g., 0.001) allows for minor rounding differences and avoids false errors.

- The formula in G21 is intended to assign the weighted score based on the answer in C21. The logic should be: if answer is "yes", assign full weight; if "no", assign zero; else blank. The original formula reversed this logic, giving scores when answer is NOT "yes", which is incorrect. Fixing the formula aligns it with other similar formulas in the sheet.

- Many weighted-average and total formulas on the FINANCIAL DATA & ANALYSIS sheet omit the third data column (column G) or reference the wrong cells (copy/paste errors). That produces wrong averages/totals.
- A similar single-cell error exists on the Example sheet (H68 just points to D68).

General rule and quick solution
- Pattern to follow for totals: Total = ColC + ColE + ColG.
- Pattern for weighted average (average duration/cost-per-call): =IF((Ccount+Ecount+Gcount)<>0, (Ccount*Cavg + Ecount*Eavg + Gcount*Gavg) / (Ccount+Ecount+Gcount), "")
- Search the FINANCIAL sheet for formulas that use only C+E (or only one column) where the pattern should be C+E+G and correct them with the general pattern above. Use Find (Ctrl+F) for occurrences of formulas like "C14+E14" or "C32+E32" and update.

Why this fixes it (short)
- Missing the G column means you’re ignoring one whole data source → totals and weighted averages are understated/wrong.
- Wrong row references (e.g., C2) are copy-paste mistakes that make formulas use the wrong values.
- The corrected formulas include all sources and multiply counts by average values correctly, so totals and weighted averages reflect all data.

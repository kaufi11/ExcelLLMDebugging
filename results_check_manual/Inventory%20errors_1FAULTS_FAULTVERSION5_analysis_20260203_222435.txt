Summary — what's wrong
- The Balance Sheet doesn't balance because Retained Earnings are wrong. Retained earnings (cells B33, D33, F33) are set equal to the current year Net Income only. They should be cumulative (prior retained earnings + current net income). Because of that the change in inventory (a prior-period inventory error that carries into beginning inventory) changes Assets but not Equity, so Assets ≠ Liabilities + Equity. The difference equals the inventory error amount.

Where the incorrect formulas are
- Sheet1 (and Sheet2): B33 = C15, D33 = E15, F33 = G15 — these set retained earnings = that year’s net income (not cumulative).
- That is why the balance sheet is off by the inventory error.

How to fix it (easy)
Option A — roll retained earnings forward (recommended)
- Keep B33 as the starting retained earnings for 2001 (if there were no prior retained earnings, B33 = C15 is OK).
- Change the later-year retained earnings to add the prior year balance:
  - D33 (2002) = B33 + E15
  - F33 (2003) = D33 + G15
This makes retained earnings cumulative and will restore Balance Sheet equality.

Option B — if you want to correct a prior-period inventory error directly
- Adjust the opening retained earnings by the inventory error amount. Example: if 2001 ending inventory was understated by 17,000, add +17,000 to opening retained earnings (B33 or the appropriate opening cell). That will also make Assets = Liabilities + Equity.

Why this works (short)
- Inventory errors affect both assets and prior period equity. If you change beginning inventory (which is last year’s ending), assets change now but equity must also be adjusted for the prior-period error (via retained earnings). Rolling retained earnings forward or adjusting opening retained earnings keeps the accounting equation in balance.

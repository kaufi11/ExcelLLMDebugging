Spreadsheet Summary — Sheet "1-24"

Purpose and layout
- Title (A1): "Table 1-27: Condition of U.S. Highway Bridges".
- Columns B:N (row 2) are years 1990 through 2002.
- Main data block occupies rows 3–14, grouping bridge counts by location (Urban/Rural) and condition (Structurally / Functionally deficient), plus totals.
- Rows 16–25 contain NOTES and SOURCES text.

Key rows (what each row contains)
- Row 3: TOTAL bridges (calculated as Urban + Rural for each year).
- Row 4: Urban bridges (subtotals computed by summing multiple urban-category counts; many cells use explicit addition expressions).
- Row 5: Rural bridges (subtotals from multiple rural-category counts).
- Rows 6–8: Urban deficient bridges total (row 6 = structurally [row 7] + functionally [row 8]).
- Rows 9–11: Rural deficient bridges total (row 9 = structurally [row 10] + functionally [row 11]).
- Row 12: All deficient bridges total (row 12 = urban deficient [row 6] + rural deficient [row 9]).
- Rows 13–14: Aggregate deficient totals by type across systems (Structurally and Functionally) — some years are given as constants, L13 and L14 are formulas summing L10+L7 and L11+L8 respectively.
- Rows 16–25: Definitions, data notes, and source citations.

Formula patterns and notable formulas
- TOTAL (row 3): typically =row4+row5 (e.g., B3 = B4+B5). For some columns SUM(...) is used (e.g., L3 = SUM(L4:L5), M3 = SUM(M4:M5)).
- Urban/Rural subtotals (rows 4–5) frequently use explicit addition strings (e.g., B4 = 24435+11890+21403+...).
- Deficient totals: each deficient total cell is computed by adding the structurally and functionally components for that year (e.g., B6 = B7+B8; B9 = B10+B11).
- All deficient (row 12): usually =row6+row9 for each year. For columns L and M row 12 is computed with SUM(L13:L14)/SUM(M13:M14) (note: different referencing style).
- A small possible formula inconsistency: D6 = 607 + D8 (appears to be a typo — expected D7 + D8).
- Several cells are direct constants rather than formulas (example N3 = 590877, N12 = 162791, many J/K/L/M/N cells have fixed values).

Quick extra note about HLOOKUP ranges
- When using HLOOKUP(range, …, row_index, TRUE) the top row of the range must contain the lookup breakpoints. Check each HLOOKUP so the lookup-value column and the returned row are inside the same range, and that row_index points to the correct relative row. If top/return rows are inverted (like damage row above the attribute-value row) use INDEX/MATCH as above.

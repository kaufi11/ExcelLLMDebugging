Sheet name and purpose
- Sheet: "2001 - 1994"
- Purpose: Year-by-year counts and percent passing on the Texas Assessment of Academic Skills (TAAS) for All Students, by grade (3,4,5,6,7,8*,10) for springs 2001 down to 1994.

Overall layout
- Title block (A1–A4) with test name, population note and exclusions.
- Each year is a separate block with a year heading in column A (e.g., A6 = Spring 2001, A21 = Spring 2000, … A105 = Spring 1994).
- For each year the data columns are:
  - C: Grade labels (Grade 3, Grade 4, … Grade 10, All Grades)
  - D: Tested (counts)
  - E: Passing (counts)
  - F: Passing (percent)
- Header rows for each block repeat labels (e.g., D8/E8/F8 = Number/Number/Percent; D9 = Tested; E9 = Passing; F9 = Passing).

Key formulas and patterns
- Percent for each grade: column F uses E(row)/D(row). Example: F10 = E10/D10 for Grade 3 (Spring 2001).
- "All Grades" totals:
  - Tested total uses SUM over that year’s tested cells (example: D17 = SUM(D10:D16) for 2001).
  - Passing total uses SUM over that year’s passing cells (example: E17 = SUM(E10:E16) for 2001).
  - All Grades percent divides the totals: F17 = E17/D17.
- This pattern (grade rows, F = E/D, All Grades = SUM ranges and E/D) repeats for each year:
  - 2001 uses D10:D16 / E10:E16 / F17 = E17/D17
  - 2000 uses D25:D31 / E25:E31 / F32 = E32/D32
  - 1999 uses D39:D45 / E39:E45 / F46 = E46/D46
  - 1998 uses D53:D58 / E53:E59 / F60 = E60/D60 (note E59 included for Grade 10)
  - 1997 uses D67:D73 / E67:E73 / F74 = E74/D74
  - 1996 uses D81:D87 / E81:E87 / F88 = E88/D88
  - 1995 uses D95:D101 / E95:E101 / F102 = E102/D102
  - 1994 uses D109:D115 / E109:E115 / F116 = E116/D116

Representative key values (examples)
- Spring 2001 Grade 3: Tested = 265,357; Passing = 205,320; F10 = E10/D10.
- Spring 2000 Grade 10: Tested = 235,489; Passing = 187,457.
- Spring 1994 Grade 10: Tested = 201,396; Passing = 100,813.

Notes and annotations
- A118 explains: Grade 8* includes reading, mathematics, and writing only.
- A119 notes: Spring 2001 results are preliminary.

Formatting expectations
- D and E contain integer counts; F contains calculated ratios (E/D), presumably formatted as percentages.
- The sheet is a repeated, consistent block structure for each year, enabling straightforward year-to-year comparisons or aggregate calculations.

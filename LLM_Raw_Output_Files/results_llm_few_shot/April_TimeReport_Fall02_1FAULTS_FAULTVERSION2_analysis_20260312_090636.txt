Findings
- Faulty cell: H19 (Studio I header row)
- Symptom: H19 uses =COUNT(H20:H34) so it returns how many numeric cells exist in H20:H34 (a count), not the total hours.

Why it's wrong (short)
- You want total hours for Studio I in column H, but COUNT returns the number of numeric entries (e.g. 10), not their sum (e.g. 45.5). Any totals that use H19 (for example H35) will therefore be wrong.

Fix
- Replace H19 with: =SUM(H20:H34)

Quick checks and small improvements
- After fixing H19, H35 (and any other dependent totals) will update correctly.
- To reduce manual-errors, use row totals like N4..N34 as: =SUM(B4:M4) instead of long explicit additions.
- Scan for other accidental COUNTs (only H19 was COUNT in the sheet you gave).

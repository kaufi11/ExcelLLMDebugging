F53 — Error: formula uses COUNT(F7:F51) but should SUM(F7:F51).

Why: COUNT returns the number of numeric cells (not the total entries for 2002), so the subtotal for 2002 and any downstream totals/percent changes (e.g. G53, F63, G63) are wrong.

Fix: replace the formula in F53 with:
=SUM(F7:F51)

Explanation: SUM adds the 2002 entry values to give the correct subtotal; dependent formulas will then produce correct totals and percentage changes.

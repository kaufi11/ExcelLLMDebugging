Summary of workbook structure and contents

High-level
- Purpose: chart-of-accounts / coding guide mapping account numbers to financial statement line items, GAAP exhibits, Form 990/990‑EZ lines, OMB A‑122 cost principle references, activity codes and templates.
- Main organization: Balance sheet coding (pages 1–2), Income/Revenue & Expense coding (pages 3–6), Activity coding summary (page 7), OMB Circular A‑122 cost principles (sheet "A-122").

Sheet-by-sheet (concise)

Pages 1–2 — Balance Sheet Account Coding
- Header/metadata: version fields in row 9 (A9, B9, G9, H9, I9), descriptors in A10..I11 (Account / Illustrative / Form 990 / Cost Principles / Templates).
- Primary columns:
  - A: internal Account number (1000–3999 ranges shown)
  - B: mapped Account number (alternate coding)
  - C: description / account line text (e.g., Cash in bank‑operating, Accounts receivable)
  - D: Line item code (numeric, used for GAAP mapping)
  - E: GAAP/Exhibit or Form 990 line references
  - F: Form 990 EZ line references
  - G: Cost principle or matching indicator (often "n/a")
  - H: Activity / account range grouping (e.g., 1000–1099)
  - I: Template / notes (often "n/a")
- Structure: grouped by asset categories (Assets: cash, receivables, investments, fixed operating assets, accum. depreciation), liabilities (payables, accrued liabilities, deferred revenue, short/long‑term loans), equity/net assets (unrestricted, temporarily/permanently restricted).
- Key total cells: C72 “Total assets” (D72 = 15, E72/F72 mapping), C124 “Total liabilities”, C141 “Total net assets”, C143 “Total liabilities & net assets”.
- Page 2 duplicates header metadata using formulas to pull values from page 1 (examples: A83 =A$9, B83 =B$9, etc.) — this pattern is repeated on later pages to maintain consistent header values.

Pages 3–6 — Income Statement / Revenue & Expense Account Coding
- Header: same metadata scheme (cells reference $9/$10/$11 on base header rows via formulas).
- Content divided across pages into revenue/support categories and expense categories:
  - Pages 3–4: Contributions & Support (4000–4999), Revenue from grants, donated goods/services, indirect contributions, government grants, earned revenues (5000–5499) and program fees.
  - Pages 4–5: Continued earned revenues, investment and other revenue (5300–5490), special events, unrealized gains, net assets released from restrictions.
  - Pages 5–6: Expenses — personnel related (7000–7590) and non‑personnel related (8100–; facility, travel, other expenses). Also non‑GAAP items (9800+ fixed asset purchases etc.).
- Primary columns same as balance sheet (A = account, B = alternate, C = description, D/E/F = GAAP/990 mappings, G = cost/principle or match type, H = activity code ranges, I = template/notes).
- Key totals: C262 “Total revenue, gains, & other support” (D262 = 66 etc.), C364 “Total expenses”.
- Notes: Many account ranges use placeholders with "-***" to indicate series (e.g., 4010-***), and contra accounts marked as "(contra)". Several cells use text markers like "n/a", "match/in‑kind", "grant/match".

Page 7 — Revenue & Expense Coding by Activity
- Purpose: maps activity code ranges (****-100 to 999) to program/department names and how to treat costs (UWAACS function ranges, allowed/unallowed under A‑122, activity code columns).
- Key columns: A = activity code ranges, B = activity description, D = OMB A‑122 cost principle reference (or n/a), E = UWAACS function codes, G = Form 990 line/group.
- Includes column heading index clarifying (a) cost principle, (b) function codes, (c) Form 990 line item, (d) templates.

Sheet "A-122" — OMB Circular A‑122 Cost Principles
- Lists cost principle topics and numeric codes from A‑122.
- Rows A5..A33 use auto-increment formulas (e.g., A6 = A5+1, A7 = A6+1) to sequentially number paragraphs; corresponding C and other columns use similar increment formulas (C6 = C5+1, etc.).
- Provides descriptors (Advertising, Bad debts, Depreciation, Fringe benefits, Travel, Trustees, etc.) — used as reference links in other sheets (column D/E links).

Notable recurring patterns & formulas
- Header propagation: many cells on pages 2–6 use absolute-referenced formulas back to the primary header cells (e.g., =A$9, =B$9, =E$10, =I$12) so header/version info is consistent across sheets.
- Account layout is tabular and repeated: (Account # / Mapped # / Description / Line item / Form 990 / Form 990EZ / Cost principle / Activity code range / Template).
- Use of placeholders: "n/a", "XXXX", "-***", "match/in‑kind", "contra" and "capitalized" are used to indicate handling rules rather than numeric values.
- Totals and mapping codes are explicitly listed as line item numbers (useful to link into reporting).

Overall layout and use
- The workbook is a master coding dictionary: map GL account numbers to external reporting line items (GAAP exhibits, Form 990/990‑EZ), OMB A‑122 cost principles, and activity codes for allocation.
- Intended for consistent cross‑sheet header propagation and easy lookup by account number or description.
- "A-122" serves as authoritative reference for cost principle numbers used in G/D columns.

Faulty cell: Sheet "Summary Info" — B20

Error type: Wrong function (COUNT instead of SUM)

What’s wrong (short): COUNT(B12:B19) returns how many numeric cells are in B12:B19 (8), not the total number of interested people. The cells hold counts of people, so you need to add them, not count them.

Fix (formula): =SUM(B12:B19)

Why this fixes it (short): SUM adds the values (0+4+1+8+14+12+15+12 = 66), which matches the 66 interested awardees reported. COUNT only returns the number of numeric entries, so it gave 8 instead of 66.

Optional tip: If you expect filtered rows, use =SUBTOTAL(9,B12:B19) so totals update with filters.

1) Balance Sheet total assets (D100) missing Restricted Securities (D64)
- Problem: D100 = D50 + D59 + D98 + D79 (it does NOT include D64). Other columns (F/G/H/J) DO include their column64 in the total.

  =D50 + D59 + D98 + D79 + D64
  (or better: =SUM(D50,D59,D98,D79,D64))
- Why: D64 (restricted securities) is part of current/other assets and must be included so Total Assets balances with Liabilities & Equity.

2) Inconsistent / off-by-one SUM ranges between columns (causes mismatched totals)
- Examples:
  - D59 = SUM(D54:D58) but F59 = SUM(F53:F58) (F59 includes row 53 while D59 does not).
  - D21 = SUM(D12:D20) but F21 = SUM(F11:F20) (start row differs).

  =SUM(<column>54:<column>58)
  and for the other subtotal use the same fixed start/end rows in every column.
- Why: shifted ranges mean some columns include wrong lines (or omit lines), producing inconsistent subtotals and totals.

- Why: totals must include the same underlying rows; inconsistent ranges break the accounting identity.

- Why: mixing thousands and full-share counts changes EPS by a factor of 1,000.

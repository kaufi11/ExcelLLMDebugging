Fix: edit H6 to remove "FORMULA " so the cell begins with =56400000000*1.01097683452934/(1+F6*0.42*R6*298), ensure the cell format is General (not Text), then press Enter (or F2→Enter) to force recalculation. You can apply the same Replace (find "FORMULA =" → replace with "=") across the sheet to fix other occurrences.

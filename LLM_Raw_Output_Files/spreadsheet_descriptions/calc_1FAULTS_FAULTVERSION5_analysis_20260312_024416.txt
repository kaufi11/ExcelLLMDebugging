Overview
- Workbook contains 3 sheets: "Your Company" (data input / calculator), "Text Only Version" (print-friendly mirror), and "Sample Company" (example values).

Sheet: Your Company (input / primary calculator)
- Purpose: Enter company background and percentages to calculate potential INzone savings.
- Key input cells (highlighted required per A11):
  - C13: Annual Value of Inventory Importations (default 0)
  - C14: Inventory Turnovers per year (1)
  - C15: Average Duty Rate for Parts (0.10)
  - C16: Average Duty Rate for Finished Product (0.075)
  - C17: Interest Rate (0.12)
  - B25, B28, B31, B34, B37: user % for obsolete/surplus, waste/scrap, exports, international returns, zone-to-zone transfers (all defaults 0)
- Calculated lines (column C):
  - C22 Cash Flow / Inventory Cost of Money = C13 / C14 * C15 * C17
  - C24 Obsolete/Surplus Inventory = C13 * B25 * C15
  - C27 Waste/Scrap/Engineering Changes = C13 * B28 * C15
  - C30 Exports = C13 * B31 * C15
  - C33 International Returns = C13 * B34 * C15
  - C36 Zone-to-Zone Transfers = C13 * B37 * C15
  - C39 Inverted Duty = C13 * ((1 - B31) - B37 - B25 - B28 - B34) * (C15 - C16)
  - C41 Total Possible INzone Savings = SUM(C22:C39)
- Other cells for annual customs/interest expense (C18, C19, C20) present but default to 0.

Sheet: Text Only Version (print-friendly mirror)
- Copies key inputs and results from "Your Company" for printing.
- Many cells reference 'Your Company' (e.g., C4 = 'Your Company'!C13; B16 = 'Your Company'!B25; C9 = 'Your Company'!C18; C32 = 'Your Company'!C41).
- Repeats the same calculated items using the mirrored input cells, e.g., C13 = C4 / C5 * C6 * C8 and other category formulas mapped to the Text Only sheet layout.

Sheet: Sample Company (example)
- Populated example to show expected outputs.
- Key example values:
  - C9 Annual imports = 50,000,000
  - C10 Inventory turnovers = 4
  - C11 Avg duty parts = 0.10; C12 finished product = 0.075
  - C13 Interest rate = 0.12
  - C14 Annual customs duty expense = 5,000,000
  - C15 Annual interest expense = 6,000,000; C16 avg interest expense = 150,000
  - Percentages: B21 obsolete 0.05; B24 waste 0.05; B27 exports 0.15; B30 returns 0.02; B33 zone-to-zone 0.10
- Calculated example formulas mirror the main sheet (e.g., C18 = C9 / C10 * C11 * C13; C35 inverted duty uses C9 * ((1-B27)-B33-B21-B24-B30)*(C11-C12); C37 totals SUM(C18:C35)).

Layout & flow
- Inputs are grouped at the top/left; calculated savings categories are listed below with formulas in column C.
- Total possible savings is the SUM of the category results.
- Contact and printing notes appear at the bottom of each sheet.

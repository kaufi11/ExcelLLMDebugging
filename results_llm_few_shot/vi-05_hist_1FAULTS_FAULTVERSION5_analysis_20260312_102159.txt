Findings
- Faulty cell: Fall 2002 sheet — H14 (Subtotal State Components)
- Error type: Wrong function (mechanical/formula error)

What’s wrong (short)
- H14 contains =COUNT(H10:H13). COUNT returns the number of numeric cells (4), not the sum of those fee amounts. That makes the Fall 2002 H14 subtotal (and any totals that reference it) incorrect.

Fix (exact)
- Replace the formula in Fall 2002!H14 with:
  =SUM(H10:H13)

Why that fixes it (short)
- SUM adds the fees together; COUNT only counts how many cells contain numbers. Changing to SUM returns the correct subtotal and automatically corrects H21 and H25 which depend on H14.

Corrected values (after fix)
- H14 = 4706.015 (4391.26 + 46.4 + 48.8 + 219.555)
- H20 = 398.0 (153.6 + 38 + 146.4 + 60) — already correct
- H21 = H14 + H20 = 5104.015
- H25 = SUM(H21:H24) = 13,890.295

Recommendation
- Scan other subtotal cells across sheets for accidental use of COUNT instead of SUM (same mistake can repeat).

Fehler (kurz):
- Zelle F27 hat den falschen Bereich: =AVERAGE(F7:F25) → F26 fehlt. Dadurch ist der Durchschnitt für Spalte F falsch.
- Möglicher weiterer Fehler: Einzelwerte mit 0 (z.B. H17 = 0) werden vom MITTELWERT als echte 0 gewertet. Falls 0 "fehlend" bedeutet, verzerrt das den Durchschnitt.

Lösung (Formeln im deutschen Format):

1) Korrigiere den Bereich in F27:
=MITTELWERT(F7:F26)

2) Optional — Wenn Nullen (0) ignoriert werden sollen (z.B. für Spalte H):
=MITTELWERTWENN(H7:H26;">0")

3) Optional — Wenn Zeilenmittelwerte (z.B. I7) Nullen ignorieren sollen (für Bereich B7:H7):
=SUMMEWENN(B7:H7;">0")/ZÄHLENWENN(B7:H7;">0")

Kurz erklärt:
- F27 musste F26 einschließen, sonst fehlt ein Schülerwert im Durchschnitt.
- MITTELWERTWENN bzw. SUMMEWENN/ZÄHLENWENN lassen Sie Nullen ausschließen, falls 0 als "nicht abgegeben" gemeint ist. Alternativ leere Zellen statt 0 eintragen, dann reicht MITTELWERT.

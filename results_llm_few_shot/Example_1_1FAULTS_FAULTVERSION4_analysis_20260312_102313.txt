1) Q4 Homework — H5
- Error: H5 =COUNT(B5:G5) — counts numeric cells (returns 6) instead of summing points.
- Fix: replace with =SUM(B5:G5)
- Why: COUNT returns number of numeric cells; SUM adds the scores (what you want).

- Error: F3 =B3*0.3+C3*0+D3*0.4+E3*0.2 — the quiz weight is 0 instead of 0.1.
- Fix: replace with a formula that uses the weight cells (robust): =SUMPRODUCT(B3:E3,$B$2:$E$2) or =B3*$B$2 + C3*$C$2 + D3*$D$2 + E3*$E$2

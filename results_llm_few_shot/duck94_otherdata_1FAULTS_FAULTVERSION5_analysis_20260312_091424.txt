Summary of problems (FRF sheet)
1) Many cells in column I show the literal text "FORMULA =SUM(Hx-Jx)" so they are not calculating — they are text, not formulas.  
2) One cell has a wrong reference/typo: I20 contains =SUM(H20-251) (subtracts the constant 251) instead of subtracting J20.  
3) Minor: using SUM around a simple subtraction (=SUM(Hx-Jx)) is unnecessary and less clear (but not wrong if entered as a formula).

Why these are errors
- If a cell contains the word "FORMULA " before the = sign (or is formatted as text) Excel treats it as text and does not evaluate it, so column I shows no computed results.  
- The I20 typo returns a meaningless number (H20−251) instead of the intended H20−J20.

Fixes (short & concrete)
1) Replace the text entries with actual formulas. Example corrections:
   - I5: =H5-J5
   - I6: =H6-J6
   - ...
   - I19: =H19-J19
   - I21: =H21-J21
   - I22: =H22-J22
   - I23: =H23-J23
2) Fix the typo in I20:
   - I20: =H20-J20  (replace =SUM(H20-251))
3) Optional cleanup: use simple subtraction (=H#-J#) not =SUM(H#-J#).

Quick way to apply fixes
- If many cells have the same problem, select I5:I23 (or the whole I column range), use Find & Replace:
   - Find: FORMULA =
   - Replace with: = 
  That will remove the extra word so the entries become real formulas (then press Enter / recalc).
- Or: enter =H5-J5 in I5 and drag the fill handle down to copy formulas.

Extra check
- After fixing, format column I as Number with appropriate decimals to see numeric results.  
- Also scan for other typos like wrong constants (e.g., any other =SUM(Hx-###)) and correct them to =Hx-Jx.

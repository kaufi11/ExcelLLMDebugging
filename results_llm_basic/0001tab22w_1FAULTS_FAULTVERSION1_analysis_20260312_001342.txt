1) Wrong cell reference (definite formula error)
- Problem: AC26 (Upper-level "Other" for Health & Public Affairs, 2000) formula is =SUM(100-(S26+U27+W26+Y26+AA26)). It incorrectly uses U27 instead of U26.
- Effect: AC26 shows the wrong percentage (uses the B% from the 1999 row), so the upper-level percentages for that row will not sum to 100 and the "Other" value is wrong.
- Fix: Change the formula to =100-(S26+U26+W26+Y26+AA26) (or =SUM(100-(S26+U26+W26+Y26+AA26)) if you prefer the original style).
- Why: the five upper-level percentage cells for that row are S26,U26,W26,Y26,AA26 — you must reference the same-row B% cell (U26), not U27.

- Effect: These stray values make the sheet look misaligned and can cause:
  - Wrong manual reading (you may read the wrong column).
  - Errors if someone later builds formulas that assume no values in those columns.
  - Potential calculation errors if those stray cells are accidentally referenced.
- Fixes (pick the appropriate one):
  a) If those numbers belong to the percentage series, move them into the correct column for that row (Cut the cell, Paste into E/G/I/K/M or S/U/W/Y/AA as appropriate).

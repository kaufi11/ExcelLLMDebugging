Summary of workbook structure and key calculations (concise)

Overall layout / common logic (applies to each populated sheet)
- Purpose: estimate nitrogen (N) removed in cereal production, partition N removed between soil vs fertilizer, compute estimated NUE (nitrogen use efficiency) and annual fertilizer (and value/energy) savings from incremental NUE improvements.
- Typical columns/fields:
  - Crop rows (wheat, corn, rice, barley, sorghum, millet, oats, rye) with production (MT), harvested area (ha) and yield (Mg/ha or converted bu/ac), and %N (grain N concentration).
  - N removed by crop = production * (%N / 100).
  - Total N removed in cereals = SUM(crop N removed).
  - Partition: 50% of N removal attributed to soil, 50% to fertilizer (E*0.5 and E*(1-0.5)).
  - Estimated NUE = fertilizer-derived N removed / cereal fertilizer N consumed.
  - N savings per 1% (and 20%) increase in NUE computed as difference between a baseline retained fraction of cereal fertilizer consumption and current fertilizer N removed; then monetized using $/metric ton and converted to energy/natural-gas savings in some sheets.
- Units: mT (metric tons) for mass.

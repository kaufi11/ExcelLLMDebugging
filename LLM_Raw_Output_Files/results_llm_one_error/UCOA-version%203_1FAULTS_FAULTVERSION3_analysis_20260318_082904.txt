Cell A83 — contains the literal text "FORMULA =A$9" instead of an actual formula (=A$9).  
Error: the cell is text (starts with "FORMULA ") so it doesn't pull the header value; many repeated "FORMULA =…" cells will not update.  
Fix: remove the leading "FORMULA " so the cell contains =A$9 (no quotes). To fix many at once: Use Find & Replace to replace FORMULA = with = (or remove "FORMULA " only), set format to General, then re-enter (or double‑click a cell and press Enter) so Excel evaluates the = formula.  
Why: Excel treats "FORMULA =A$9" as text, not a formula; changing to =A$9 makes the cell reference the header and update dynamically.

  2) Conceptual/accounting error: a prior‑period inventory error was never closed to retained earnings. Because the beginning inventory error affects prior net income, you must adjust retained earnings (a prior‑period adjustment) — otherwise Assets ≠ Liabilities + Equity by the amount of the unclosed error.

What to do (fixes):
- Fix the typo:
  - Sheet2 B10 should be =B8+B9 (cost of goods available for sale = beginning inventory + purchases).
- Close the prior‑period inventory error to retained earnings:
  - If last year’s ending inventory was understated by X, increase current beginning retained earnings by X.
  - If last year’s ending inventory was overstated by X, decrease retained earnings by X.
  - In your workbook that correction is shown on Sheet3: D35 = E17 + 17000 and F35 = G17 - 25000. Those add/subtract the prior-year inventory errors to retained earnings so the balance sheet balances.

Why this fixes it (simple explanation):
- An inventory misstatement changes COGS and net income. If it belonged to a prior year you must correct retained earnings (equity) for that prior‑period effect. Inventory (an asset) changes and retained earnings (equity) must change by the same amount (net of tax) so Assets = Liabilities + Equity stays true. If you only change inventory and current-year income but don’t adjust retained earnings for the prior error, the balance sheet will be out by the error amount.

How to verify:
- After making the two fixes above, check Total Assets = Total Liabilities + Total Equity for each year. If they match the worksheet is corrected.

Cell: Select Financial Data!J12

Error: Formula uses =J9-J11 (wrong) so J12 computes -J10 (a negative freight amount) instead of gross profit.

Fix: Change formula in J12 to =J8-J11 (Total revenues minus Cost of operations). Edit the cell (or formula bar), replace =J9-J11 with =J8-J11 and press Enter.

Why: J8 is Total revenues; J9 is Cost of sales. Using J9 caused a subtraction error that propagates down (Earnings from Operations, taxes, Net Earnings). Correcting J12 will update all dependent totals.

1) Incorrect function (actual error)
- Cell C19: formula is =COUNT(D19:CC19)
  - Problem: COUNT returns the number of numeric cells (6), not the sum of the points. That makes C19 = 6 and B19 = 6/100 = 0.06, while the student’s points should be 3+8+7+20+20+30 = 88.
  - Fix: change C19 to =SUM(D19:I19) (or =SUM(D19:CC19) if you really want a wide range, but D19:I19 is correct and explicit).
  - After fix B19 should be =C19/$C$6 (or keep the current =SUM(C19/$C$6) if you prefer), which yields 0.88.

- Cells C9:C18 and C20:C24 use formulas like =SUM(D9:CC9), =SUM(D10:CC10), etc.
  - Problem: these sum from column D all the way to column CC (very large range). Because columns beyond I are blank, the numeric result matches the intended D:I sum today, but this is poor practice and can hide data, cause maintenance/performance issues, or accidentally include future data. It is inconsistent with C6 which sums D6:I6.

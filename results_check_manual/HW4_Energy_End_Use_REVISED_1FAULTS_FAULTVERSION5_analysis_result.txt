3) Inconsistent / fragile logic to detect fuel type (logic issue)
- Sheets: All
- Cells: D38 and F38
  - D38 formula: =IF(C13<1, D37/100000, 0)
  - F38 formula: =IF(C13=G34, D37/F9, 0) with G34 = 1
- Explanation:
  - These formulas use furnace efficiency (C13) to decide which energy units to produce (therms vs kWh). Using C13 as a proxy for fuel type is fragile:
    - If you ever have an electric heater with efficiency slightly different than exactly 1 (floating point rounding) these IF checks may behave incorrectly.
    - The IF(C13<1, ...) assumes that any efficiency < 1 means combustion fuel (therms). That's OK for current inputs but not explicit.
    - The equality IF(C13=G34,...) uses an exact equality test on floating point numbers, which can be unreliable.
- Suggested improvement:
  - Add an explicit fuel-type or "fuel = gas/electric" flag cell (TRUE/FALSE or text), and base the IF on that flag rather than on the numeric C13 value.
  - Or use a tolerance test if you must compare floats (e.g. =IF(ABS(C13-G34)<0.0001, ...)).

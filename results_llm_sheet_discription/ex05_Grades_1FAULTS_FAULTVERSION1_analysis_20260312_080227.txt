Findings (short)
- Error found on Exams sheet: G19 has =SUM(E19:F19) and omits Exam 1 (D19).
- Impact: Mittelstadt’s exam total is 73+70 = 143 instead of 71+73+70 = 214. This will make Summary totals and percent/letter grades wrong for that student.

How to fix (quick)
1. On Exams sheet, select G19 and replace the formula with:
   =SUM(D19:F19)
   Press Enter. G19 will become 214.
2. To prevent similar mistakes, re-fill the whole Total column so every row uses the same relative formula:
   - Enter =SUM(D6:F6) in G6, then drag the fill handle down to G28 (or select G6:G28 and use Ctrl+D).

Explanation (simple)
- The formula in G19 only added columns E and F, so it left out Exam 1 (column D). Correcting to SUM(D19:F19) includes all three exam scores.
- Re-filling the column ensures no single-row typo remains. If Summary cells reference Exams!G6:G28, fixing G19 automatically corrects the Summary values.

Prevention tips (optional, one-line each)
- Use Excel Tables so totals auto-fill and maintain consistent formulas.
- Use Formula Auditing → Error Checking (Inconsistent formula) or add a check column like =SUM(D6:F6)=G6 to spot mismatches.

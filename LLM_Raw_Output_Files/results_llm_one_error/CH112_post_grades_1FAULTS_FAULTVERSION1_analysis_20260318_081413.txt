O13: Wrong formula — uses division (/) where multiplication (*) should be.

Problem: O13 is
=(SUM(B13:F13)+SUM(G13:M13)+N13)/(COUNT(B13:F13)/100+COUNT(G13:J13)/15+COUNT(K13:M13)/30+COUNT(N13)/150)/100
This divides the counts instead of multiplying them, so the denominator is tiny and O13 is incorrect.

Fix: replace with the same correct pattern as other rows:
=(SUM(B13:F13)+SUM(G13:M13)+N13)/(COUNT(B13:F13)*100+COUNT(G13:J13)*15+COUNT(K13:M13)*30+COUNT(N13)*150)*100

Why it works: COUNT(...) returns number of items; you must multiply by each item's max points to get total possible points (not divide). Then divide earned points by total possible and multiply by 100 to get a percentage.

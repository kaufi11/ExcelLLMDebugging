Layout and purpose (Calculations)
- Title and refs: A1 title "Points-Based Fee Calculator", A2 citation "20.2.75 NMAC".
- Column roles:
  - Column A — Permitting action / label text and section headings.
  - Column B — user inputs: number of units, number of applicable regs, and "x" or any symbol to mark presence/requirement.
  - Column C — calculated total points for each line (formulas shown below).
  - Column D — unit text (e.g., "per point").
  - Column E — a side table: "Non-applicable regulations" and mapping of NMAC parts/NSPS/NESHAP numbers.
  - Column F — final fee output cell (uses formula to choose full or reduced fee).

Key fixed value
- Fee per point: C5 = 315 ($/point).

Important input cells (examples)
- B9, B11, B12, B13, B16, B18, B19, B22, B23, B24, B27, B30, B31, B34, B35, B38 — users enter counts or place "x" to indicate applicability or small-business.

Notable formulas (location : formula — brief effect)
- C9: =IF(B9<>"",IF(B9<5,5,IF(B9>15,15,B9)),0) — points based on # of emission units, clamped 5–15.
- C11: =IF(B11<>"",5,0) — 5 points if fugitive emissions marked.
- C12: =IF(B12<>"",75,0) — 75 points if non-attainment area marked.
- C13: =IF(B13<>"",15,0) — 15 points if modeling review required.
- C16: =IF(B16<>"",8,0) — Level 1: 8 points if marked.
- C18: =IF(B18<>"",60,0) — BACT: 60 points if marked.
- C19: =IF(B19<>"",100,0) — Health assessment: 100 points if marked.
- C22: =IF(B22<>"",(3*B22),278) — 3 points per 20 NMAC 2.X reg; default 278 if B22 blank.
- C23 / C24: =IF(B23<>"",(5*B23),0) and =IF(B24<>"",(5*B24),0) — 5 points per NSPS/NESHAP if entered.
- C27: =IF(B27<>"",100,0) — 100 points for case-by-case MACT if marked.
- C30: =IF(B31="",(IF(B30<>"",20,0)),"") — PSD netting only: 20 points if B30 marked and B31 blank.
- C31: =IF(B31<>"",75,0) — PSD review including netting: 75 points if B31 marked.
- C34: =IF(B35="",(IF(B34<>"",10,0)),"") — General permit: 10 points if B34 marked and B35 blank.
- C35: =IF(B35<>"",(B35*10),0) — Streamlined: 10 points per site (B35).
- C36: =SUM(C34:C35) — total other actions points.
- C41: =IF(OR(C34>0,C35>0),C36,SUM(C9:C31)) — total points: if other-actions present uses C36, else sum of main action points.
- C42: =C41*C5 — computed fee ($) = total points × fee/point.
- C43: =IF(B34<>"","N/A",IF(B38<>"",C42/2,"N/A")) — half-fee for small business: returns half fee if B38 (small business) marked, otherwise "N/A" when general permit (B34) present or not small biz.
- F5: =IF(C43="N/A",C42,C43) — final fee displayed: uses half-fee if available, otherwise full fee.

Workflow / how to use
- Enter counts and mark applicable items in column B (numbers or "x").
- Column C auto-calculates points for each action.
- C41 shows total points; C42 shows full fee; C43 may show half fee for small businesses.
- F5 presents the applicable Total Fee (selecting half fee when enabled).

Miscellaneous
- E-column contains a numbered list of non-applicable regulation identifiers (NMAC parts, NSPS, NESHAP references) used for reference.
- Instructional notes in A47–A48 and revision note in A49 (rev. 02/06/01).

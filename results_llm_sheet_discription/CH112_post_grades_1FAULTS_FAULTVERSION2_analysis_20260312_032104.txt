2) Grade-count formulas (rows 49–53) can mis-count blanks as low scores:
- COUNTIF with comparisons like "<90" or "<60" treats empty cells as 0 for the numeric comparison, so blank cells get counted in the lower-grade bins (inflates B/C/D/F counts).
- Examples: B50 uses COUNTIF(...,"<90")-COUNTIF(...,"<80"); B53 uses COUNTIF(...,"<60"). These include blanks.

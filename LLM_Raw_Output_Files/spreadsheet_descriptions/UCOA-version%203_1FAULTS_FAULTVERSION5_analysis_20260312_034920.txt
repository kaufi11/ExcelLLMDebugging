Summary of workbook structure and contents (concise)

Overall design
- Master account-coding and cross-reference workbook that maps internal account numbers to financial-statement line items, GAAP exhibits, Form 990/990‑EZ line numbers, OMB A‑122 cost principles, United Way (UWAACS) function codes, activity codes and template references.
- Accounts follow a two-tier code style: 4-digit primary account (column A) and a mapped 2nd/alternate number (column B). Descriptions are in column C. Columns D–I contain multiple reporting cross‑references (GAAP exhibit, Form 990, Form 990‑EZ, cost principles, UWAACS ranges, templates/notes).

Sheets (pages 1–7 and A‑122)
- Page 1 (Balance sheet — Assets)
  - Header cells: H1/Page label; G5 “Balance Sheet Account Coding”; C6 “Balance Sheet or Statement of”.
  - Lists asset account ranges (1000–1999) with rows for Cash, Receivables, Investments, Fixed assets, Accumulated depreciation, Other long‑term assets.
  - Key total: “Total assets” row (C72) with D72 = line item code 15 (GAAP exhibit mapping).
  - Repeating header/version cells in row 9–11: A9/B9/G9/H9/I9 hold version/policy labels (e.g., version 3.0, OMB A‑122, United Way).

- Page 2 (Balance sheet — Liabilities & Equity)
  - Continues payables, accrued liabilities, deferred revenue, short/long‑term debt, custodial funds.
  - “Total liabilities” row (C124) mapped to line item D124 = 28.
  - Equity section (3000–3999): Unrestricted, temporarily & permanently restricted net assets and totals with D141 = 36 (total net assets).
  - Many cells on this sheet reproduce top header values via formulas (see Formulas below).

- Page 3 (Income statement — Contributions/Support)
  - Revenue account mappings for contributions, donated goods/services, grants, indirect contributions, split‑interest revenue.
  - Accounts are in the 4000 range. Total revenue/gains row (C262) mapped to D262 = 66.

- Page 4 (Income statement — Earned revenues)
  - Earned revenue mappings (government contracts, program fees, dues, investment income, special events).
  - Accounts primarily 5000–5999 and 5400 series for other sources. Includes “Total revenue, gains & other support” mapping.

- Page 5 (Expenses — personnel related)
  - Expense account mappings for grants/assistance, salaries & benefits, contract services, professional fees, donated services.
  - Accounts around 7000–7590. “Total expenses” mapping on Page 6/7 (see below).

- Page 6 (Expenses — non‑personnel, business, fixed assets)
  - Non‑personnel expense groups: supplies, facility & equipment, travel & meetings, other expenses, business expenses, non‑GAAP/capital purchases (9800+).
  - Total expenses mapping (C364) to D364 = 146 and other non‑GAAP items (capitalized accounts 9810–9840).

- Page 7 (Activity coding)
  - Maps activity code ranges (****‑100 to ****‑999) to program departments, M&G (management & general), fundraising, cost pools, capital purchases pool.
  - Columns include activity code, UWAACS function code ranges (column E), cost principle notes (column D), and Form 990/Policy mapping (column G).
  - Intended to be used for activity‑level revenue & expense coding.

- Sheet "A-122" (OMB Circular A‑122 listing)
  - Enumerated cost principle titles with an automated numbering pattern.
  - Many cells use sequential formulas: column A entries like A6 = A5+1, and column C similarly auto‑incremented. This creates the A‑122 paragraph numbering automatically.
  - Also includes links/metadata (B3 web link).

Key formulas & automation patterns
- Header propagation: many sheets replicate top metadata using absolute references, e.g.:
  - =A$9, =B$9, =G$9, =H$9, =I$9 (seen on rows A83/B83/G83/H83/I83, repeated on other pages). This ensures version/policy headers are consistent.
  - Similar for row 10/11 references: =A$10, =E$10, =F$10, etc., to bring form/column headers across sheets.
- Page 2 (and others) shows explicit formula cells copying the header values to rows used for printing or export.
- A‑122 sheet uses sequential formulas to auto‑number paragraphs: A6 = A5+1, A7 = A6+1, etc., and corresponding formulas in C and other columns to keep numbering consistent.
- Mostly static mapping rows; very few arithmetic accounting formulas — primarily lookup/propagation style formulas and auto‑increment sequencing.

Numbering conventions and mappings
- Balance sheet accounts: 1000–3999 (Assets 1000–1999; Liabilities 2000–2999; Equity 3000–3999).
- Income/expense accounts: 4000–9999 (4000–4999 contributions/support; 5000–5999 earned revenues; 6000–6999 other; 7000–8999 personnel/expenses; 9800+ capital).
- Columns mapping:
  - A = Primary account number (internal)
  - B = Alternate/mapped account number
  - C = Account description
  - D = GAAP exhibit / line item code (or other internal line numbers)
  - E = Form 990 line reference
  - F = Form 990‑EZ line reference
  - G = OMB A‑122 cost principle or matching rules/allowability notes
  - H = UWAACS / United Way function code ranges or activity code
  - I = Policy/templates/other references
- Activity codes are four‑digit patterns ****‑XXX; page 7 maps ranges (****‑100..199 etc.) to functional buckets and UWAACS codes.

Notable values to watch
- “n/a” and “contra” flags appear frequently in G/H/I — used to indicate nonapplicability or contra‑accounts.
- Many rows include specific mapping references (e.g., D72=15, D124=28, D141=36, D262=66, D364=146) — these are key line‑item IDs for aggregated totals.
- Capitalization flags for 9810–9840 (marked "capitalized").

Overall layout & intended use
- The workbook is a master coding table to standardize chart of accounts mapping to external reporting frameworks and cost allocation rules. It supports:
  - Financial statement construction (balance sheet & income statement),
  - External reporting (Form 990 / GAAP exhibits),
  - Federal cost principle compliance (OMB A‑122),
  - United Way accounting mappings (UWAACS),
  - Activity‑based allocations via activity codes.
- Designed for print/export across multiple “pages” (1–7) with header propagation to ensure consistency.

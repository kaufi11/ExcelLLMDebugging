Error: formula uses the wrong cell in the IF test (C2 instead of C20), so the zero-check doesn't match the denominator used in the calculation.

Wrong formula:
=IF(C2+E20<>0,((C20*C22)+(E20*E22))/(C20+E20),"")

Fix (correct formula):
=IF((C20+E20)<>0,((C20*C22)+(E20*E22))/(C20+E20),"")

Why: the IF must test the same sum that is used as the denominator (C20+E20). Using C2 can return an incorrect blank or allow a division-by-zero. Changing to C20 ensures the condition and calculation match.

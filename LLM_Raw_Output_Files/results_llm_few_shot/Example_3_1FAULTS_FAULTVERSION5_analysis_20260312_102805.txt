I found two problems. Short fixes and why they work below.

1) Faulty cell: Q4 Homework!H5
- Error: Wrong function (mechanical). H5 uses =COUNT(B5:G5) so it returns the count of cells (6), not the sum of points.
- Fix: Replace with =SUM(B5:G5)
- Why: Net score should be the sum of the assignment columns, not a count.

2) Faulty cells: Q4 Summary!F3:F12 (all student Weighted Score rows)
- Error: Wrong reference (logic/copy error). Each formula uses D#*0.2 (duplicating the Projects column) instead of using the Homework column E# for the 0.2 weight.
- Fix (one example for row 3): change F3 to =B3*0.3 + C3*0.1 + D3*0.4 + E3*0.2
  Then copy that corrected formula down F4:F12.
- Better (more robust) formula using the weight cells: =B3*$B$2 + C3*$C$2 + D3*$D$2 + E3*$E$2  (copy down)
- Why: The last term must be Homework (E) × 0.2. Using absolute references to the weight row ($B$2 etc.) makes the formula easier to maintain.

How to apply:
- Replace H5 with =SUM(B5:G5).
- Replace F3 with one of the corrected formulas above, then fill down F4:F12.

That will fix the incorrect totals and the weighted scores.

Summary of workbook structure and contents

Sheets
- Profs — main contact list (rows ~1–75).
- URLs — one-column list of project/organization names and their URLs (alternating description / link).

Profs sheet — layout and key columns
- Header row (row 1): A: #, B: sent, C: Name, D: Position, E: University, F: Email address, G: Suggested by, H: Personal or Related Web Page, I: Address, (J used inconsistently for second address line), K: City, L: S/P (state/province), M: CNTY (country), N: Zip, O: Phone, P: FAX.
- Numbering: column A is sequential numbering. A2 = 1, most subsequent rows use formulas like A3 = A2 + 1 and so on. Notable exceptions: A42 uses =A41-1 (likely a mistake) and A52 references =A49+1 (skipping A50/A51), so numbering has at least one anomaly/gap.
- Sent status (col B): markers like "x" (sent), "u" (unknown/unsubscribed?) and blanks.
- Data content: names (col C), roles (col D), affiliations (col E), emails (col F). Many rows include personal web pages in H and postal/phone details in I/O/P/K/L/M/N. Some email fields contain "?" where unknown.
- Examples: Row 2 — Roy D. Yates, Director, Rutgers, ryates@winlab.rutgers.edu; Row 17 — Dr. Kaveh Pahlavan with full address and phone; multiple MIT Media Lab and Columbia contacts included.
- Inconsistencies: J header is missing (address line 2 appears in J on some rows), occasional missing emails, variable use of address columns, and the single anomalous formula noted above.

URLs sheet — layout and key values
- Single-column list (column A) with alternating description and URL format: e.g., A3 = "Worcester Polytechnic Institute, Center for Wireless Information Studies", A4 = "http://www.wpi.edu/"; A10/A11 for MIT LCS, A13/A14 for MIT Media Lab, etc.
- Contains links for projects and groups such as WIND, Congestion Manager, CAG, CellularIP, IETF Mobile IP WG, CMU Monarch, MosquitoNet, MEDIA (KTH), INRIA hierarchical mobile IPv6, etc.

Overall notes
- Workbook is a contact directory of academics/technical contacts with sequential numbering, status tracking, and relevant URLs.
- Main formulas are simple incrementing formulas in column A; check A42 and the A49→A52 jump for numbering errors. Column layout is mostly consistent but address components are split across I–P with occasional use of J.

Summary — main error found
- Angel's Curse sheet, cell E69 is wrong: it has =SUM(I69:I69) so it only adds the Limited column (I69) and ignores Unlimited (J69). That makes the "Have" total in E69 (and therefore H69) incorrect.

Why it's wrong
- The formula should add both Limited (column I) and Unlimited (column J). =SUM(I69:I69) only includes I69; =SUM(I69:J69) (or I69+J69) is correct.

How to fix (quick)
1. Put this in E69: =SUM(I69:J69)  (or =I69+J69)
2. Press Enter.
3. If you want consistency, copy that corrected formula down column E for all rows (select E69, double‑click the fill handle or drag down).

How to prevent / check for other errors
- Run a Find (Ctrl+F) for SUM(I and inspect results — look for SUM(Ix:Ix) occurrences where the second cell should be Jx.
- Or standardize column E with one formula for the whole table (in E2): =I2+J2 then fill down — this prevents one-off typos.

Notes
- SUM(Kx) / SUM(Lx:Mx) are harmless (SUM on a single cell works), but keeping formulas consistent (I+J, L+M, K only) makes the sheet easier to audit.

Summary of workbook (concise)

Sheets
- "2003-04 per credit hour", "Spring 2003 per credit hour", "Fall 2002 per credit hour".
- Each sheet is a tuition & fees table for the University of Florida (HEADER in A1:A3).

Overall layout
- Column headings: B = Per Credit Hour; F = Per Term* (A single professional-term column group for Medical/Dental/Vet in F–H).  
- Course level columns: B = Undergraduate (0000–4999), C = Graduate (5000–9999), D = Law, F = Medical, G = Dental, H = Veterinary. (Column E is unused/blank.)
- Row groups:
  - State Components (starts at A9): Matriculation (A10), Building (A11), Capital Improvement Trust Fund (A12), Student Financial Aid (A13), Subtotal State Components (A14).
  - Local Fees (A15–A19): Activity & Service, Athletic, Health, Transportation Access, Subtotal Local Fees (A20).
  - Totals and Non‑Resident section: Total Florida Resident Rate/Credit Hour (A21), Non‑Resident student & Non‑Resident student financial aid (A23–A24), Total Non‑Resident Rate/Credit Hour (A25).
- Footnote in A27–A28 explaining per‑term billing for Medical/Dental/Vet.

Formulas (pattern)
- Subtotal State Components: B14 = SUM(B10:B13); C14, D14, F14, G14, H14 use analogous SUM across their component rows.
- Subtotal Local Fees: B20 = SUM(B16:B19); C20, D20, F20, G20, H20 analogous.
- Resident total: B21 = SUM(B14,B20) (same pattern for C21,D21,F21,G21,H21).
- Non‑resident total: B25 = SUM(B21:B24) (and similarly for C25,D25,F25,G25,H25).
- Note: In "2003-04 per credit hour" H14 is set to COUNT(H10:H13) instead of SUM(H10:H13) — likely a mistake/inconsistency.

Key values / examples (from 2003–04 sheet)
- State component subtotal (Undergrad per credit hour): B14 = 71.34 (63.41 + 2.32 + 2.44 + 3.17).
- Subtotal local fees (Undergrad per credit hour): B20 = 21.34.
- Total Florida resident (Undergrad per credit hour): B21 = 92.68.
- Total non‑resident (Undergrad per credit hour): B25 = 460.28 (B21 + B23 350.10 + B24 17.50).

Other notes
- Values differ between sheets/terms (Spring 2003 and Fall 2002 have slightly different per‑credit and per‑term rates).  
- Per‑term columns (F–H) show large totals (annual/term totals) for professional programs; formulas mirror the per‑credit columns.  
- Minor numeric formatting inconsistencies appear (e.g., H13 = 219.555 in Fall 2002 vs 219.55 elsewhere).

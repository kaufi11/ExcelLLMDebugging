Faulty cell: Summary Info!B20

Error type: Omission (wrong range)

What’s wrong: B20 uses =SUM(B12:B18) so it omits the MPS row (B19 = 12). The "would like to attend" total is therefore understated by 12.

Fix (repair formula): =SUM(B12:B19)

Why this fixes it: including B19 gives the correct total (66), which then matches the Carnegie-classification total (E30) and the 66 interested attendees noted in A8. Tip: keep all three totals using the same start/end rows (or use an explicit table) to avoid range-mismatch errors.

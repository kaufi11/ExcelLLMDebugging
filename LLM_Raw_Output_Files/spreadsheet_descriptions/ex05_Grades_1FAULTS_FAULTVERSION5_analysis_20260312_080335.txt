Overview: The workbook is a five-sheet grade book (Exams, Quizzes, Projects, Homework, Summary). Each of the four score sheets uses the same student roster (rows 6–28): column A = ID #, B = Last Name, C = First Name; row 5 contains headers. Student score totals on each sheet are computed with SUM formulas. The Summary sheet lists IDs/names and contains the "Total Points Possible" row.

Sheet-by-sheet summary

- Exams (C2 = "Exams")
  - Headers in row 5: A=ID #, B=Last Name, C=First Name, D=Exam 1, E=Exam 2, F=Exam 3, G=Total.
  - Student data in A6:G28.
  - Each student Total in column G uses =SUM(Dn:Fn) (e.g., G6 =SUM(D6:F6)).
  - One anomaly: G19 (Mittelstadt) contains =SUM(E19:F19) — it omits Exam 1 (D19).

- Quizzes (D2 = "Quizzes")
  - Headers row 5: A=ID #, B=Last Name, C=First Name, D–I = Quiz 1–6, J = Total.
  - Data in A6:J28.
  - Each student Total in column J =SUM(Dn:In).

- Projects (C2 = "Projects")
  - Headers row 5: A=ID #, B=Last Name, C=First Name, D–F = Project 1–3, G = Total.
  - Data in A6:G28.
  - Each student Total in column G =SUM(Dn:Fn).

- Homework (C2 = "Homework")
  - Headers row 5: A=ID #, B=Last Name, C=First Name, D=Summary, E=Report, F=Survey, G=Service, H=Total.
  - Data in A6:H28.
  - Each student Total in column H =SUM(Dn:Gn).

- Summary (C2 = "Grade Book for Spring 2003")
  - Headers row 5: A=ID #, B=Last Name, C=First Name, D=Homework, E=Projects, F=Quizzes, G=Exams, H=Total, I=Percent, J=Letter.
  - Student IDs and names are populated in A6:C28; columns D–G (per-student aggregated scores) are blank in the provided data (no formulas shown).
  - Total Points Possible row at row 30: D30 = 120, E30 = 345, F30 = 60, G30 = 300; H30 contains =SUM(D30:G30) (total = 825).

Other notes
- The roster and ID order are consistent across all four score sheets (helps lookups/aggregation).
- Totals on each score sheet are calculated with simple SUM formulas; to populate the Summary sheet per student you would typically use LOOKUP/INDEX-MATCH or SUMIFS to pull each student's totals from the four sheets.

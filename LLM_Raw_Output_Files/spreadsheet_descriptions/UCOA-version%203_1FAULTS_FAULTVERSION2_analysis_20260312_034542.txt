Summary of workbook structure and contents

Overall layout / column mapping (consistent across pages)
- Columns:
  - A: Primary account code (number or range)
  - B: Secondary/mapped account code (where used)
  - C: Account description / grouping
  - D: GAAP exhibit / line item number or internal line-item index
  - E: Form 990 line/item mapping
  - F: Form 990‑EZ mapping
  - G: Cost principle / matching indicator (OMB A‑122 references or match type)
  - H: Accounting guide / activity or chart-range (e.g., 1000–1099)
  - I: Templates / notes / references
- Top-of-sheet metadata repeated on each sheet (cells like A9, B9, G9, H9, A10…A11 etc) and mirrored elsewhere via formulas (e.g., =A$9, =E$10). This provides consistent header values across pages.

Sheets overview

Pages 1–2: Balance sheet (Financial Position accounts, 1000–3999)
- Page 1 lists Assets by account ranges (1000 series) with detailed rows for cash, receivables, investments, fixed assets, accumulated depreciation, other long‑term assets.
- Page 2 continues Liabilities (2000 series) and Equity/Net assets (3000 series). Includes payables, accrued liabilities, deferred revenue, notes/loans, long‑term debt, and net asset classes (unrestricted, temporarily & permanently restricted).
- Key total lines:
  - C72 / D72 = Total assets (D72 = 15; E72 = 59; F72 = 25 shown as line references)
  - C124 / D124 = Total liabilities
  - C141 / D141 = Total net assets
  - C143 / E143 = Total liabilities & net assets (E143 = 74)
- Many contra and required flags shown (e.g., doubtful allowances flagged as "contra", "req" for required disclosures).

Pages 3–6: Income statement / Revenue & Expense (Financial Activities accounts, 4000–9999)
- Page 3–4 cover Contributions & Support (4000 series) and Earned Revenues (5000–5400 series). Detailed sub‑categories for individual, corporate, foundation grants, donated goods/services, government grants, split‑interest, United Way etc.
- Page 5–6 cover Expenses (7000–9999), organized into:
  - Personnel-related (salaries, benefits, payroll taxes, contract services)
  - Non‑personnel (supplies, facility & equipment, travel & meetings, other expenses)
  - Business/non‑GAAP items and fixed asset purchases (9800+ capitalized items)
- Key total lines:
  - C262 / D262 = Total revenue, gains, & other support (D262 = 66)
  - C364 / D364 = Total expenses (D364 = 146)
  - Notes where some rows use "sum" or "capitalized" as placeholder indicators for formulas or treatment.
- Account numbering often uses patterns with "-***" to indicate activity suffixes; many rows show Form 990 cross‑references (column E) and internal activity codes (column H).

Page 7: Revenue & Expense coding by Activity
- Maps activity code ranges (****-100..999) to departments/functional areas (Program A–D, M&G, Fundraising, Cost pools, etc.).
- Shows which activities are allowed/allocated per OMB A‑122 and UWAACS function codes (column D/E/G).
- Column heading index explaining the meaning of small-letter labels: (a) cost principle numbers in OMB A‑122, (b) UWAACS function codes, (c) Form 990 line items, (d) Policy 03/Template line items.

Sheet "A-122": OMB Circular A‑122 cost principles list
- Sequential list of A‑122 cost principle topics with formulas to auto‑increment numbering (cells use formulas like =A5+1 and =C5+1).
- Provides mapping between cost principle numbers and descriptors (Advertising, Bad debts, Depreciation, Travel, etc.) used in column G across other sheets.

Formulas and cross‑sheet behavior
- Header metadata is propagated across multiple sheets using absolute row references, e.g., =A$9, =E$10, =I12 — ensures consistent document metadata on each page.
- A‑122 sheet uses simple increment formulas (e.g., =A5+1) to number items automatically.
- Some cells contain textual placeholders (n/a, sum, capitalized) rather than numeric formulas; key totals reference line‑item numbers (D/E/F) rather than calculated sums in the sample extract.
- Many rows use patterned account ranges with "-***" to indicate account numbers combine base code + activity suffix (activity‑level coding).

Notable conventions and flags
- "contra" denotes contra accounts (allowances, accumulated depreciation).
- "req" marks required disclosures.
- Columns E and F map to external reporting forms (Form 990 / 990‑EZ line numbers), enabling reporting crosswalk.
- Column H often lists internal chart ranges or UWAACS function/activity ranges.
- Column G contains OMB A‑122 cost principle references or match/grant indicators (e.g., match/in‑kind, grant/match).

Use case / purpose
- The workbook is a comprehensive account code and reporting crosswalk: it maps ledgers (account numbers and descriptions) to GAAP line items, Form 990/990‑EZ line numbers, OMB cost principles, UWAACS function codes, and internal activity coding — supporting chart of accounts design, external reporting, federal grant budgeting/compliance, and allocation of costs by activity.

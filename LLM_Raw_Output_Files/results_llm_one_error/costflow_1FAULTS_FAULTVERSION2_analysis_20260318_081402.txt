Valuing Ending Inventory — I8: wrong formula. It is =+C10 (total units) but should be =C8 (beginning inventory units). 

Why it’s wrong: I8 should hold the beginning‑inventory units (C8). Using C10 (total units) distorts the per‑row unit counts, so the average unit cost and all subsequent COGS/ending inventory numbers are incorrect.

Fix: change I8 to =C8. Also update the corresponding row formulas so each row uses its own units/costs (I9 = C9; J8 = E8/I8; J9 = E9/I9; K8 = E8; K9 = E9). This will correctly compute weighted averages and COGS.

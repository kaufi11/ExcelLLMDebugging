Cell D68: formula =D65-E67 is wrong.

Why: E67 is blank so net purchases isn't subtracting purchase returns & discounts. D67 already contains the sum of returns and discounts (2570), so using E67 leaves Purchases unchanged.

Fix: Change D68 to =D65-D67  (or =D65-SUM(C66:C67)).

Result: Net purchases = 55,000−2,570 = 52,430; Cost of goods purchased = 52,430+5,200 = 57,630; COGS = 72,630−12,500 = 60,130; gross margin updates accordingly.

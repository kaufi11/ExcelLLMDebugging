Sheet1 — layout and purpose
- A1: Title "EE394, Spring 1999, Homework #2".
- Row 3 (C3:N3): column headers for the distance sweep and phasor results:

  - G: Real(VS), H: Imag(VS), I: VS mag (kV rms), J: VS angle (deg)
  - K: Real(IS), L: Imag(IS), M: IS mag (A rms), N: IS angle (deg)
- Column A (rows 4–35+): parameter labels (inputs and derived quantities), e.g. "Outer rad./inner rad.", "Relative permittivity", "VR (kV rms)", "Load S (MVA)", "Load pf", "Frequency (Hz)", "Increment (km)", then line C,L,Z0,Beta,wavelength, P/Q, load currents, load impedance, phase voltages, reflection coefficient, VSWR, etc.
- Column B: primary inputs and computed scalar parameters:
  - Key inputs:
    - B4 = 2.7 (outer/inner radius ratio used in L and C)
    - B5 = 2.5 (relative permittivity εr)
    - B6 = 100 (VR, kV rms)
    - B7 = 10 (Load S, MVA)
    - B8 = 0.9 (power factor, lagging)
    - B9 = 60 (frequency, Hz)
    - B10 = 50 (distance increment, km)
  - Derived line parameters (formulas shown in your listing):
    - B12: C (F/m) = 2π·ε0·εr / ln(B4)
    - B13: L (H/m) = 2·1e-7·ln(B4)
    - B14: Z0 (Ω) = SQRT(L/C)
    - B15: Beta (rad/m) = 2π·f·sqrt(L·C)
    - B16..B18: wavelength and characteristic speeds (1/sqrt(LC), 1/sqrt(μ·ε))
    - B19: P (MW) = B7*ABS(B8)
    - B20: Q (MVAr) = ±sqrt(B7^2 - B19^2) with sign by pf
    - B21..B24: load current magnitude and components (IR mag, IR angle, Real(IR), Imag(IR))
    - B25..B27: load impedance real/imag/magnitude
    - B28..B31: phase A/B voltages real/imag (kV)
    - B32..B34: reflection coefficient (Real, Imag, Mag)
    - B35: VSWR = (1 + |rho|)/(1 - |rho|)
- Columns C and D: distance sweep formulas
  - Cn = -(ROW()-4)*$B$10 → produces a negative distance sequence in km (increments of B10), Dn = -Cn (positive distance).
  - E,F: cos(β·d) and sin(β·d). Note argument uses B15*D*1000 (converts km → m).
- Columns G..N (repeated per row): compute sending-end phasors and sending current from the load/line parameters at each distance:
  - G (Real VS) = (B6*1000*E - B14*B24*F)/1000
  - H (Imag VS) = B14*B23*F/1000
  - I = magnitude SQRT(G^2 + H^2)
  - J = ATAN2(G,H) in degrees
  - K (Real IS) = B23*E
  - L (Imag IS) = B6*1000/B14*F + B24*E
  - M,N = magnitude and ATAN2(K,L) in degrees
  - These G..N formulas are repeated down many rows (row4 → row54) to get VS and IS versus distance.
- Notes on units and conversions:
  - Voltages in column B (B6) are kV rms; formulas multiply by 1000 where needed to convert to V for intermediate computations, then divide back to kV when storing VS.
  - Distances: B10 in km; E/F use β (rad/m) × distance (m), so D (km) × 1000 is used.
- Repetition: rows 4–54 replicate the same per-distance formulas to sweep many distances at step B10. Row formulas in columns C..N are auto-generated by copying the pattern.

Overall function

Spreadsheet summary (concise)

Overall structure
- Workbook contains three sheets: "2003-04 per credit hour", "Spring 2003 per credit hour", and "Fall 2002 per credit hour".
- Each sheet is the same layout: title rows (A1:A3), two main column groups:
  - Per Credit Hour group: columns B (Undergraduate 0000–4999), C (Graduate 5000–9999), D (Law courses).
  - Per Term group: columns F (Medical), G (Dental), H (Veterinary).
  - Column E is unused/blank and A is used for row labels and section headings.
- Footnote in A27–A28 explains medical/dental/vet charges are annual and payable in two installments.

Row/section layout (identical pattern on each sheet)
- State Components: Matriculation, Building, Capital Improvement Trust Fund, Student Financial Aid.
  - Subtotal State Components in row 14 uses SUM of the four state component cells (e.g., B14 = SUM(B10:B13)).
- Local Fees: Activity & Service, Athletic, Health, Transportation Access.
  - Subtotal Local Fees in row 20 uses SUM of the four local fee cells (e.g., B20 = SUM(B16:B19)).
- Total Florida Resident Rate/Credit Hour in row 21 = subtotal state + subtotal local (e.g., B21 = SUM(B14,B20)).
- Non-Resident section: Non-Resident Student and Non-Resident Student Financial Aid.
  - Total Non-Resident Rate/Credit Hour in row 25 = resident total + non-resident items (formulas use SUM over rows 21:24, e.g., B25 = SUM(B21:B24)).
- Per-term columns (F:H) mirror the same structure with SUM formulas over their respective rows (e.g., F14 = SUM(F10:F13), F20 = SUM(F16:F19), F21 = SUM(F14,F20), F25 = SUM(F21:F24)).

Key formulas called out
- Subtotals use SUM of ranges: B14, C14, D14, F14, G14, H14 = SUM(... state rows ...)
- Local subtotal: B20, C20, D20, F20, G20, H20 = SUM(... local rows ...)
- Resident totals: B21, C21, D21, F21, G21, H21 = SUM(subtotal state, subtotal local)
- Non-resident totals: B25, C25, D25, F25, G25, H25 = SUM(B21:B24) etc.

Representative key numeric values
- 2003–04 (per credit): Matriculation — Undergrad B10 = 63.41; Graduate C10 = 170.63; Law D10 = 193.92. Per-term Medical F10 = 6,962.69; Dental G10 = 6,054.54; Vet H10 = 5,085.75. Non-resident undergrad B23 = 350.10.
- Spring 2003 (per credit): Matriculation — Undergrad B10 = 58.45; Graduate C10 = 147.33; Per-term Medical F10 = 6,011.90. Non-resident undergrad B23 = 300.49.
- Fall 2002 (per credit): Matriculation same as Spring 2003 for credit columns (B10 = 58.45) and per-term values similar to Spring 2003 (minor rounding differences in some per-term cells).

Notable inconsistencies / observations
- Spring 2003: H14 is listed as SUM(H10:H12) (omits H13) — likely a typo; other sheets use SUM(H10:H13).
- Minor rounding differences between Spring 2003 and Fall 2002 per-term entries (e.g., G10/G13 vary by 0.01–0.02).
- Layout is consistent, formulas are simple SUMs of contiguous ranges or pairs, making the sheet easy to audit or update.

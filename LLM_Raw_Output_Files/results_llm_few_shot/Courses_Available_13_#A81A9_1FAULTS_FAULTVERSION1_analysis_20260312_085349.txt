Why A63 is a problem
- All A-column cells are supposed to be sequential IDs built from the previous row (e.g. =A4+1). A63 =5 resets the chain, so every subsequent number is wrong/shifted.

Quick fix (recommended)
1. Replace A63 with this formula:
   =A62+1
2. Fill (drag) that formula down column A for all rows below to restore the continuous sequence.

What this does: each row takes the prior row's number and adds 1, restoring the intended sequential IDs.

If you want numbers only when column B has a course (and leave headers/empty rows blank), use:
- In the first numbered row (A4): =IF(TRIM(B4)="","",1)
- In A5 and copy down: =IF(TRIM(B5)="","",MAX($A$4:A4)+1)

This will produce incremental IDs only for rows where column B contains text (so headers or truly blank rows stay blank).

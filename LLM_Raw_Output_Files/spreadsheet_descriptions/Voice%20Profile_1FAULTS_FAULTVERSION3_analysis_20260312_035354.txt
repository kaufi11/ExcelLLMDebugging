Sheets

- Example (working example showing entered data and formulas)
  - Layout: Column A = line-item descriptions; groups for COST PER HANDSET, LOCAL CALLS, LONG DISTANCE (to mobiles / not to mobiles), INTERNATIONAL, MOBILE PHONES, INBOUND NUMBERS (1800 / 1300), mixes and average durations.
  - Three data/source columns shown (B/C, D/E, F/G) with a TOTAL column in H.
  - Key example values:
    - Number of handsets: B11 = 752; planned B12 = 820
    - Local calls cost: B14 = 15,000; number of local calls B15 = 107,500
    - Long distance (not to mobiles) cost: B20 = 7,300; calls B21 = 10,152; avg duration B23 = 221s
    - LD to mobiles cost: B32 = 1,250; calls B33 = 1,114; avg duration D34 = 1s (contract price)
    - Mobile phones: B50 = 9,000; mobiles B51 = 163; calls B52 = 13,750; avg duration B53 = 105s
    - Inbound 1800: B59 = 2,500; calls B60 = 2,568; avg length B62 = 156s
    - Inbound 1300: B65 = 1,600; calls B66 = 5,512; avg length B68 = 330s; B69 = 425 calls > set duration
  - Representative formulas (examples):
    - Category totals: B7 = B8 + B9 + B10 (monthly expenditure subtotal)
    - Overall total across data columns: H7 = B7 + D7 + F7
    - Average cost per call with divide-by-zero guard: H16 = IF(H15<>0, H14/H15, "")
    - Weighted average duration: H17 = IF((B15+D15)<>0, ((B15*B17)+(D15*D17))/(B15+D15), "")
    - Per-category avg cost: B22 = B20 / B21 (and analogues)
  - Many cells contain text like “derived”, “contract price”, or “estimated” as annotations.

- FINANCIAL DATA & ANALYSIS (template for user input / aggregation)
  - Same structure and sectioning as Example, but arranged with three input columns C, E, G (DATA and SOURCE pairs) and a TOTAL column in I.
  - Totaling pattern: I# = C# + E# + G# (e.g., I7, I8, I9, I19, I31, I44, I49, I58, I64).
  - Averaging and weighted averages use IF guards; e.g., I15 = IF(I14<>0, I13/I14, ""), I16 = IF((C14+E14)<>0, ((C14*C16)+(E14*E16))/(C14+E14), "")
  - Cells mirror the Example for mixes, durations, number of calls, costs and provide slots for contract or bill source data.

Overall layout and logic
- Workbook provides: guidance (Introduction), a worked example (Example), and an input/aggregation template (FINANCIAL DATA & ANALYSIS).
- Design pattern: per-call-category rows (local, LD, mobile, international, inbound) with columns for multiple data sources and a total column that sums the sources.
- Key calculations:
  - Sum of source columns to total column (I = C + E + G or H = B + D + F).
  - Average cost per call = total cost / total calls with IF to prevent divide-by-zero.
  - Weighted average call duration = weighted mean of durations by volume across sources.
- Notes:
  - Cells contain inline comments to guide users — ensure Excel comment indicators are visible.
  - Many text placeholders (e.g., “derived”, “contract price”, “estimated”) indicate where user input or interpretation is required.

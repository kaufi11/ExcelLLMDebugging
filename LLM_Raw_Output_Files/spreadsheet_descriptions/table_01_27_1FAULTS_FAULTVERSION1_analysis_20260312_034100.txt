Spreadsheet: Sheet "1-24"

Scope and layout
- Single main table in A1:N14 titled "Table 1-27: Condition of U.S. Highway Bridges" (A1).
- Years run across columns B2:N2 (1990 through 2002).
- Primary rows describe counts by system and condition:
  - TOTAL bridges (A3)
  - Urban bridges (A4) and Rural bridges (A5)
  - Urban deficient (A6) with Structurally (A7) and Functionally (A8)
  - Rural deficient (A9) with Structurally (A10) and Functionally (A11)
  - All deficient (A12) with Structurally (A13) and Functionally (A14)
- Notes and sources in A16:A25 explain definitions and data provenance.

Formulas and calculation patterns
- TOTAL (row 3) is computed as Urban + Rural for each year: e.g., B3 = B4 + B5; some years use SUM ranges (L3 = SUM(L4:L5), M3 = SUM(M4:M5)).
- Deficient totals are computed by summing their component rows: e.g., A6 (Urban deficient) B6 = B7 + B8, and A9 (Rural deficient) B9 = B10 + B11.
- Many component cells (rows 4, 5, 7, 8, 10, 11) contain explicit arithmetic expressions adding multiple numeric subcomponents directly in the formula (e.g., B4 = 24435+11890+21403+...).
- Some summary cells use range SUMs for L and M columns (e.g., L6 = SUM(L7:L8); L12 = SUM(L13:L14)).
- Inconsistencies: most formulas reference other cells (Urban+Rural), but many detailed counts are constructed by adding constants in the formula rather than referencing sub-cells.

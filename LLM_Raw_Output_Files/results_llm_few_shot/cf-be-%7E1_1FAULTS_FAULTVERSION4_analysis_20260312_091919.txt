1) Total Fixed Costs uses the wrong source

- Fix: Use the subtotal of fixed expenses (O47).
  - Correct formula: =O47
- Why: O47 is the explicit subtotal of fixed expenses. Using O54 mixes non-operating/other cash items and gives the wrong "fixed cost" number.

2) Break-even calculation uses the wrong numerator

- Why: Break-even sales = Fixed costs / Contribution margin. Use the correct fixed-cost cell (B63 or O47).

  - O22: =SUM(C22:N22)

4) Percent-of-sales totals reference themselves (confusing / fragile)
- Problem: P57 (Total % of Sales) uses O57/O57 or O57/O$57 in places (dividing the total by itself). Many P formulas are inconsistent across sheets.

- Why: Row-level percent should be item total ÷ grand total (O$57 anchored). The totals row should show 100% explicitly. Using self-divisions is confusing and can produce divide-by-zero if O57 is zero.

5) End-of-month cash position uses SUM unnecessarily and can hide mistakes

  - And copy across: C55 = C15 - C54, etc.
- Why: =SUM(B15-B54) evaluates, but it's awkward, reduces readability and can mask copying mistakes. Use simple arithmetic for clarity and to avoid accidental range inclusion.

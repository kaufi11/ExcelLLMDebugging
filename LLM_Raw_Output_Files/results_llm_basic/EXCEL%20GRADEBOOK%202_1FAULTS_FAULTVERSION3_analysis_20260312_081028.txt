1) Cell X5 (Quiz Average for Reed, Bret) has a typo.
- Wrong formula: =(COUNT(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)
- Why it’s wrong: COUNT returns the number of quiz cells (5) — you need the SUM of quiz scores. Using COUNT instead of SUM produces a nonsense negative average (e.g. (5‑75)/4 = -17.5).
- Correct formula: =(SUM(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)

How to fix:
- Replace the formula in X5 with the correct one above, or copy X2 (or any other correct row) and paste/fill down into X5.

What this changes (example):
- Correct X5 = 95.25 (not -17.5).
- Correct AE5 final grade becomes about 87.72 (instead of ~59.54 with the bad X5).

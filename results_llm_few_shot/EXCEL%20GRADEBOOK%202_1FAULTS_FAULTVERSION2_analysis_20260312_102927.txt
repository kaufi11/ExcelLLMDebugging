Faulty cell
- Q5

Error type
- Formula / logical error (wrong operators and operands).

What’s wrong (short)
- Q5 uses =(SUM(J5:O5)+MIN(J5:O5))/(COUNT(J5:O5)+1). That adds the minimum and increases the count, so it does NOT drop the lowest classwork score — it returns a wrong (inflated) average.

Fix (repair formula)
- Replace Q5 with the same correct “drop lowest” formula used elsewhere:
  =(SUM(J5:O5)-MIN(J5:O5))/(COUNT(J5:O5)-1)

Why this fixes it (short)
- To drop the lowest score you subtract MIN(...) from the total sum, then divide by one less than the number of scores (COUNT(...) - 1). The current formula does the opposite.

Optional tips
- After fixing Q5, copy Q2’s correct formula down for consistency.
- If a student might have only one score, wrap with IF to avoid divide-by-zero:
  =IF(COUNT(J5:O5)>1,(SUM(J5:O5)-MIN(J5:O5))/(COUNT(J5:O5)-1),AVERAGE(J5:O5))

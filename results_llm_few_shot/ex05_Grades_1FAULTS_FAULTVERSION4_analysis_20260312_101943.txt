Faulty cell
- Exams!G19

Error type
- Omission / range mistake (formula excludes Exam 1 for that student)

What’s wrong (simple)
- G19 is =SUM(E19:F19). It leaves out D19 (Exam 1 for Mittelstadt), so the total is too low by 99.

Correct formula
- =SUM(D19:F19)

Correct value
- 293 (99 + 101 + 93). Current wrong value = 194.

How to fix (quick)
1. Put =SUM(D19:F19) into G19.
2. To prevent similar mistakes, select G6:G28 and use Fill Down from a correct cell (e.g., G6) so every row uses the same range.

Quick prevention tip
- Convert the student rows to an Excel Table or use consistent Fill Down so totals use the same formula for every row. You can also use formula auditing (Show Formulas) to spot inconsistent formulas.

Faulty cell: F27

Error type: Range omission (mechanical/omission)

What’s wrong: F27 uses =AVERAGE(F7:F25) so it leaves out the last student in row 26. That makes the class average for column F computed from 19 students instead of 20.

Fix (quick): Change the formula in F27 to
=AVERAGE(F7:F26)

Why this fixes it: Including F26 ensures the average uses every student score in the intended range.

Better practices to avoid this in future:
- Convert the student area to an Excel Table (Insert > Table). Then use structured references like =AVERAGE(Table1[F]) which auto-expand when you add/remove rows.
- Or use a consistent formula pattern and fill/copy across, or check counts with =COUNT(F7:F26) to verify you included all rows.

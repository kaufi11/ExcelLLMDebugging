H19 — Error: formula uses COUNT(H20:H34) but should sum hours.  
Fix: change to =SUM(H20:H34).  
Why: COUNT returns the number of numeric cells (e.g., 15) not the total hours (≈24), so row totals (H19) and the aggregated weekly totals (H35 and N19/N35) are understated. Replacing COUNT with SUM restores correct totals.

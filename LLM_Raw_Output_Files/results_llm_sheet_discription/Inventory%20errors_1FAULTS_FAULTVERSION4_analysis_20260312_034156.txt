Short answer — the error
- The balance sheet is out of balance because prior‑period inventory errors changed Assets (ending inventory) but were NOT closed to Retained Earnings (Equity). In other words: inventory was misstated, so COGS and Net Income were wrong, but Equity was not adjusted for the prior‑year error. That leaves Assets ≠ Liabilities + Equity.

Why that causes the mismatch (simple rule)
- Ending inventory error → Asset is too high/low.
- If the error is from a prior period, the correct fix is to adjust Retained Earnings (prior‑period equity) by the opposite amount so Equity reflects the restatement.
- If you don’t do that, Assets change but Equity does not, so the balance sheet won’t balance.

Where this appears in the workbook
- Sheet2 shows the raw inventory errors (e.g. B11 uses 179,800 + 17,306; D11 uses 147,200 − 25,000). Assets follow those ending inventory cells, but Retained Earnings are just set to the current year’s net income, so T Assets ≠ T Liabilities & Equity.
- Sheet3 shows the correct approach: retained earnings are adjusted for the prior‑year inventory errors (D35 = E17 + 17,000; F35 = G17 − 25,000) and the balance sheet balances.

How to fix it (two simple options)
1) Restate prior period (recommended, what Sheet3 does)
   - Adjust Retained Earnings by the opposite of the prior‑period inventory error.
   - Example formulas (from Sheet3): D35 = E17 + 17000 (add if prior inventory was understated); F35 = G17 − 25000 (subtract if prior inventory was overstated).
   - Result: Equity reflects the prior‑period correction and Assets = Liabilities + Equity.

2) Correct the inventory values themselves
   - Put the correct beginning/ending inventory amounts into the income statement lines (so COGS and Net Income are recalculated correctly).
   - Because Inventory on the balance sheet links to the ending inventory cells, fixing the inventory numbers will make Assets move to the correct amounts and Net Income/Retained Earnings will be correct — balance sheet will then match.

How to implement in Excel (quick steps)
- Identify the prior‑period inventory error amount (Error = reported ending inventory − correct ending inventory).
- If you close to retained earnings, add an adjustment to the retained earnings cell for the year following the error:
   RetainedEarnings_corrected = RetainedEarnings_calculated + (−InventoryError_prior_year)
  (i.e., subtract if inventory was overstated; add if understated).
- Or replace the incorrect ending inventory cells with the correct numbers and let formulas cascade.

One-line summary
- Fix retained earnings for prior‑year inventory errors (or correct the inventory amounts). Sheet3 demonstrates the correct adjustment and restores balance.

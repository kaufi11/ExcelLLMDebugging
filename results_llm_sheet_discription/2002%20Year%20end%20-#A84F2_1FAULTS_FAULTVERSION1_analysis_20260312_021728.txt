3) Text “-” used in numeric cells may break formulas
- Problem: Some cells that should be zero show a dash (“-”) (e.g. C39 Bank overdraft has “-”).
- Effect: If a cell contains text it may be ignored by SUM or cause errors when referenced with unary plus (e.g. =+C39) or when feeding other formulas.
- Fix: Replace textual “-” with an explicit numeric 0 (or blank if intended). Use 0 where calculations require a numeric.
- Explanation: Excel treats "-" as text; numeric formulas work reliably only with numbers (0) or empty cells.

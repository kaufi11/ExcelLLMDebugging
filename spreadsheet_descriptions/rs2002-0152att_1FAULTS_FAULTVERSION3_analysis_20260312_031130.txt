Summary of workbook structure and contents (concise)

Common layout (all sheets)
- Rows 2–5: program title and approval/resolution text. Row 6: column headers (B:I). Data rows start at row 7.
- Column mapping: B=Region # (or program code), C=RFP#, D=Applicant, E=Project Title, F=Results (Fund / Not Fund / Non-Competitive), G=Project Funds Requested, H=Approved / Awarded Funds, I=Cumulative total (running total) with a funding limit shown in the header.
- Cumulative column (I) is implemented as a running sum: first funded row I7 = H7 (or sometimes =G7), subsequent rows typically I(n) = I(n-1) + H(n) (some sheets/rows use G instead of H in the formula).
- Not-funded rows have H = 0. Some projects receive partial awards (H < G).
- Several sheets include asterisks and notes indicating funding limits or remaining available amounts for a specific project number.

Sheet-by-sheet highlights
1) CALFED Drinking Water
- Program: CALFED Drinking Water Grant Program (From Nonpoint Source Pollution Grant Program).
- Funding limit in header: $10,000,000.
- Standard columns B:I. Running cumulative in I (I7 = H7; then I8 = I7+H8, etc.).
- Example anomalies: row 9 awarded H9 (1,390,800) less than requested G9 (1,480,800); row 13 (G13 333,040) partially awarded H13 145,680.
- Mostly “Fund” results; cumulative totals increment by H.

2) NPS Pollution
- Program: Nonpoint Source Pollution Grant Program (Regions 4,6,7,8,9).
- Funding limit: $22,300,000.
- Cumulative I follows running-sum pattern (I7 = H7; I8 = I7+H8...).
- Some projects “Not Fund” with H=0. Row 32 (project #612) has a partial award H32 238,115 vs requested G32 722,500 and an asterisk notes remaining available funding ($238,115).
- Several large awards (multi-million approvals) and many Not Fund rows listed below the cumulative stopping point.

3) CALFED Watershed

6) Coastal NPS-Reg 1,2,3

Formula and data-quality observations (key points)
- Running-total formulas are consistently used in column I, but the first cumulative cell occasionally references G instead of H; later rows sometimes add G instead of H—this is an inconsistency to watch.
- Partial awards are common (H < G) and Not Fund rows have H = 0.
- Several sheets include asterisks and notes that call out remaining available funds for a particular project number, indicating funding-limit thresholds were reached.
- No single-sheet grand totals are shown; cumulative column I provides a running awarded-total up to each row relative to the sheet-specific funding limit.

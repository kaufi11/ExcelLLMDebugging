Summary of the error
- Cell F14 (Grade 7, Spring 2001) has the wrong formula. It shows =E14/170 instead of dividing by the tested count in D14. That will give a totally incorrect percent (and >1000%) for Grade 7.

Why it's wrong
- All other percent cells use Percent = Passing / Tested (e.g. =E10/D10). F14 uses a fixed number 170 instead of the tested value in D14 (274090), so the percent is meaningless.

Fix (quick)
1. In F14 replace =E14/170 with =E14/D14.
2. Format F14 as Percentage (same number of decimals as the other percent cells).

Better (robust)
- In F10 enter =E10/D10 and then copy/fill that formula down through F16 (or the whole F column). This avoids single-cell manual typos.
- Recalculate (F9) or let Excel auto-calc.

How to check for similar errors
- Show formulas (Ctrl+`) to visually scan for formulas that contain numeric literals like “/170”.
- Or use Find (Ctrl+F) and search for "/1", "/170", or search for "=" and inspect formulas that don't reference the D column.
- Use Formulas → Error Checking → Trace Precedents to ensure each percent references its Tested cell.

Effect on totals
- The All Grades percent (F17) uses E17/D17, so it is not directly affected by F14 being wrong — but individual Grade 7 percent for 2001 is wrong and can mislead readers.

Faulty cell: I19

Error type: Wrong function (mechanical/logical error)

What’s wrong: I19 uses COUNT(I7:I17). COUNT returns the number of numeric cells, not the total of the vote counts. That gives a small integer (how many responses) instead of the sum of votes in column I.

Fix (repair formula): =SUM(I7:I17)

Why this fixes it: SUM adds the numeric values in I7:I17 (the vote counts). COUNT only counts how many numeric cells exist, so it was the wrong function for a total.  
Quick extra check: after changing, confirm I7:I17 are stored as numbers (not text). If any look like numbers but SUM still behaves oddly, convert them to numbers (e.g. multiply by 1 or use VALUE).

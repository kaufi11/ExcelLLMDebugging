Overview
- Workbook has two sheets: "program" (master event timetable) and "schedule" (detailed paper/session listing).
- Purpose: define conference events (times, rooms, topics) and map individual papers to sessions.

Sheet: program
- Header/metadata: A1 conference title; A3/A5 co-chairs and organizers; A7 conference dates and venue; F7/F8 organizer/URLs.
- Main table header in row 11 with columns:
  - A: EVENT (sequential event number)
  - B: DAY
  - C: START (time values)
  - D: END (time values)
  - E: TYPE (e.g., Session-1, Break, Opening, Registration)
  - F: TOPIC (session topic / chairs)
  - G: Length (minutes)
  - H: ROOM (room or room range)
- Most EVENT cells are generated by simple increment formulas (e.g., A13 = A12+1, A14 = A13+1, etc.) to produce a running event number.
- Notable anomaly: A25 uses =A24+715 (likely a typo — jumps the sequence), while some events are hard-coded (e.g., A12 = 1, A41 = 26).
- Times in C/D are stored as Excel time values (e.g., 08:30:00), Length in G is numeric (minutes) for many sessions; some breaks have no length value.
- Contains multi-day layout (Monday, Tuesday, Wednesday) with sessions, breaks, poster slots, keynote, conference dinner, and MIT guided tour.
- Repeated topics: "Public Access", "PP-IT and e-Planning", "PP-IT in e-Government", "PP-IT in Teaching/Education", "PP-IT and Globalization".
- Some entries flagged in E include notes (e.g., "Break-A - Eliminated").

Sheet: schedule
- Header row defines key columns (some labels shown):
  - B: PAPERID
  - C: SESSIONID
  - E: AUTHOR(S)
  - F: INSTITUTION
  - G: AFFILIATION
  - H: COUNTRY
  - I: TOPIC
  - J: Paper title (appears in the sheet though not listed as header in your extract)
- Layout mixes two types of rows:
  - Section rows in column A for day/session headings, room and time information (e.g., "Monday 10, 08.30 - 09.00AM", "Session I - Public access to information").
  - Detail rows for each paper with PAPERID, SESSIONID (often annotated with suffixes like "-poster", "(p)", "(sr)", "(t)" etc.), authors, institutions, country, topic and paper title.
- The sheet contains full session membership: Session I–VIII, Posters, Keynote (Joan Blades, Jane Fountain), conference dinner and guided tour entries.
- Many rows include CITIDEP as affiliation and repeat room 66-110 and 7-431 for different sessions.

Key values / highlights
- Conference: INTERNATIONAL CONFERENCE ON PUBLIC PARTICIPATION AND INFORMATION TECHNOLOGIES, MIT, 10–12 November 2003.
- Organizers: CITIDEP and MIT-DUSP.
- Keynote: Joan Blades and Jane Fountain (Tuesday 16:45–18:00, Room 66-110).
- Conference dinner: Tuesday 20:00 (Restaurant in Cambridge).
- MIT guided tour: Wednesday 14:45 (Lobby 7).
- Program uses sequential event numbering (with one anomalous +715 formula); schedule maps individual PAPERIDs to those sessions including posters and special flags.

Overall structure / purpose
- "program" = high-level timeline (event sequencing, times, rooms, durations).
- "schedule" = detailed program contents (paper-level assignments, authors, institutions, titles).
- The workbook is largely static data with minimal formulas (mostly simple increments); one formula appears incorrect and would break the event numbering sequence.

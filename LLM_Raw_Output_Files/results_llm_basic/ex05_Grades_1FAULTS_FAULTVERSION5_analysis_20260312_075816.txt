Findings
- One clear error: Exams sheet, cell G19 contains =SUM(E19:F19). That omits Exam 1 (D19) so the total is too low (73+70 = 143) instead of the correct 71+73+70 = 214.
- This wrong total will cascade into any summary/percent calculations that pull the Exams totals.

Quick fix (one step)
- In Exams!G19 replace the formula with =SUM(D19:F19) and press Enter. The correct value 214 will appear.

Alternate quick fix (fill down)
- Select a correct Totals cell (e.g. Exams!G18 which has =SUM(D18:F18)), copy it, then paste (or drag the fill handle) into G19 to restore consistent formulas for that row.

How to prevent / find similar errors
- Use Excel’s Inconsistent Formula check: Formulas tab → Error Checking → Inconsistent Formula. It flags cells whose formulas differ from nearby formulas.
- Or show formulas with Ctrl+` and visually scan the Totals column for inconsistent ranges.
- To enforce consistency, select G6:G28 and enter one formula (e.g. =SUM(D6:F6)) and fill down.

Why this matters
- A single wrong total changes student totals, class averages, percentages, and letter grades in the Summary sheet. Fixing G19 ensures accurate downstream calculations.

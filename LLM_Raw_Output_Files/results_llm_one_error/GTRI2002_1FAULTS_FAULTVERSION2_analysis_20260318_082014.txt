Cell: 2002 Report!M46 — Error: formula uses SUM(B14:D14) (wrong rows).

Solution: Change M46 to =SUM(B46:D46) and fill/down for the other rows (M47 -> =SUM(B47:D47), etc.).

Explanation: M-column (SVEP) is pulling data from row 14+ instead of the same row, so the SVEP, N (difference) and O (ratio) results are all incorrect. Pointing the formula to the current row fixes the misalignment and yields correct comparisons.

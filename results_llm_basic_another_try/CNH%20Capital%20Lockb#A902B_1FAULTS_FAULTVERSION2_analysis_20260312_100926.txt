2. **Opportunity cost of float formulas (B33:C35):**  
   They use `$B17`, `$B18`, `$B19` for checks, but these are in column B (Pennsylvania checks). For California columns (C33:C35), it should reference California yearly checks in column C (`$C17`, `$C18`, `$C19`).

**Solutions:**

1. **Fix C41:**  
   Change formula from `=C19/C$6` to `=C19*C$6`.  
   Explanation: Check processing charge = number of checks * cost per check, so multiplication is correct.

2. **Fix opportunity cost formulas for California (C33:C35):**  
   Change references from `$B17`, `$B18`, `$B19` to `$C17`, `$C18`, `$C19` respectively.  
   Explanation: Use the correct check origination data for California to calculate opportunity cost properly.

**Summary:**  
- Correct the division error in C41 to multiplication.  
- Use California check data in California opportunity cost formulas instead of Pennsylvania data.

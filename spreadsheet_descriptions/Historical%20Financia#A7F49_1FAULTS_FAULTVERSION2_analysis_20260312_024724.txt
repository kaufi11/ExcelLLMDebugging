Overview
- Workbook contains two sheets: "Seven-Year Financial Summary" (annual view) and "Nine-Quarter Financial Summary" (quarterly view).
- Both sheets present a consistent layout: labelled line items in column A, a column for percent-change/notes in column C (Seven-Year) or headers for latest quarter in column C (Nine-Quarter), and year/quarter columns to the right with values and formulas.

Sheet: Seven-Year Financial Summary
- Layout: A: row labels (Income Statement, Memo, Gross Profit, Operating Expenses, Income From Ops, Net Income, Balance Sheet, Supplemental). Columns D–J = FY2003 → FY1997. Column C mostly shows "% Change" (2003 vs. 2002) or point-change text for margins/ratios.
- Key line items:
  - Revenue components (Hardware, Software, Service) in rows 5–7. D5:D7 = 137,013; 71,251; 191,927 respectively.
  - Total Revenue (row 8): D8..J8 use SUM(D5:D7) etc. FY2003 total (D8) = 400,191.
  - Gross profit rows (13–19): component gross profits and totals. Total Gross Profit (D19) = D13+D15+D17 (198,560).
  - Operating expenses (rows 24–28) include SG&A, R&D, Office Closure (some years), Depreciation. Total Operating Expenses are SUM of the components (row 28).
  - Income From Operations (row 31) and Net Income (row 35): FY2003 D31 = 38,222; D35 = 21,782.
  - EPS diluted (row 39): FY2003 = 1.22. Weighted average shares (row 43) D43 = 17,838.
  - Balance sheet (rows 46–64): Working capital, Net cash from ops, Cash & equivalents (D50 = 45,682), Total Assets (D52 = 370,015), Total Liabilities (D60 = 148,787), Shareholders' Equity (D62 = 221,228), Book value/share (D64 = 12.28).
  - Ratios (rows 66–68): Return on Equity (D66 = 0.109), Return on Assets (D68 = 0.063).
  - Supplemental items (rows 71–75): software amortization, capitalized software, capital spending.
- Key formulas:
  - Column C percent-change formulas: e.g., C5 = (D5-E5)/E5 (applies to many rows comparing FY2003 vs FY2002).
  - Totals use SUM ranges (e.g., D8 = SUM(D5:D7), D28 = SUM(D24:D27)).
  - Percentage rates: gross profit % rows use formulas like D14 = D13 / D5; overall gross profit % D21 = D19 / D8; operating-expense % D29 = D28 / D8; income% rows = line / revenue.
- Notes / anomalies:
  - Many formulas include a redundant leading plus sign (=+ ...).
  - J14 uses "/37" (J14: =+J13/37) — likely a typo (should be J13/J5).
  - Some cells contain "-" placeholders (e.g., E26, F26, G26 for Office Closure Costs).

Sheet: Nine-Quarter Financial Summary
- Layout: A: labels similar to annual sheet. Columns C–M cover FY2003 Q4 → FY2001 Q4 (C:D:E:F for FY2003 Q4→Q1; H:K for FY2002 Q4→Q1; M for FY2001 Q4). G and L are unused between yearly groups.
- Key line items:
  - Quarterly revenue components (rows 5–7). Example FY2003-Q4 (C5:C7) = 41,671; 22,597; 56,274. Total Revenue C8 = SUM(C5:C7) = 120,542.
  - Maintenance revenue memo (row 11) included in Service revenue.
  - Gross profit components (rows 13–19). Total Gross Profit C19 = C13+C15+C17.
  - Quarterly operating expenses (rows 23–26). Total operating expenses C26 = SUM(C23:C25).
  - Income From Operations (row 29) often computed as Total Gross Profit minus Total OpEx (C29 = C19 − C26).
  - Net income (row 33): FY2003-Q4 C33 = 8,786. EPS diluted (row 37) C37 = 0.48 (Q4 FY2003).
  - Balance sheet quarterly snapshots (rows 45–61): e.g., Cash & equivalents C49 = 45,682; Total Assets C51 = 370,015; Total Liabilities C59 = 148,787; Shareholders' Equity C61 = 221,228.
  - Supplemental quarterly items (rows 64–68): quarterly software amortization, capitalized software, capital spending.
- Key formulas:
  - Quarter totals and gross profit totals use SUM or addition of components (e.g., C8 = SUM(C5:C7); C19 = C13 + C15 + C17).
  - Percent rows (gross profit %, operating expense %) reference line / total revenue: e.g., C20 = C19 / C8; C27 = C26 / C8.
- Notes / anomalies:
  - Same redundant =+ usage in many formulas.
  - Column groupings skip columns G and L (structural spacing).
  - Some percentages are written as numeric constants in a few cells (e.g., H31 = 0.075) rather than formulas.

Overall observations
- The workbook is a standard financial summary: revenue → gross profit → operating expenses → operating income → net income, plus balance sheet and supplemental capital/amortization items, shown across seven years and nine quarters.
- Formulas rely on simple arithmetic and SUM ranges; percent-change comparisons in the seven-year sheet compare FY2003 vs FY2002.
- Two minor data/formula issues to review: J14’s "/37" divisor and inconsistent use of "-" placeholders for missing values.

**Solutions:**

1. **Fix Weighting Sum Checks:**  
   - Change the checks from 100% to 1 (since weights sum to 1.0).  
   - Example: In F12, replace `=IF(SUM(F6:F10)<>100%,"ERROR","100%")` with `=IF(ROUND(SUM(F6:F10),4)<>1,"ERROR","100%")`  
   - Use ROUND to avoid floating point precision issues.

2. **Make IF Formulas Case-Insensitive:**  
   - Use `LOWER()` function on the answer cell to convert text to lowercase before comparison.  
   - Example: Change `=IF(C6="yes",(1*F6),IF(C6="no",(0*F6),""))` to `=IF(LOWER(C6)="yes",1*F6,IF(LOWER(C6)="no",0*F6,""))`  
   - Apply this fix to all similar IF formulas in column G.

3. **Fix Section IV Weighted Score IF Formulas:**  
   - Use LOWER() on the answer cell for comparisons.  
   - Example: Change `=IF(C41="yes",(1*F41),IF(C41="no",(0*F41),IF(C41="small extent",(0.33*F41),IF(C41="large extent",(0.67*F41),""))))`

---

**Explanation:**

- The weighting sums are decimals (like 1.0) not percentages (100%), so the formula must check against 1, not 100%.  
- Excel text comparisons are case-sensitive, so "Yes" ≠ "yes". Using LOWER() standardizes the text to lowercase, ensuring the IF condition matches regardless of case.  
- This ensures weighted scores calculate correctly and total weighting checks do not falsely show errors.

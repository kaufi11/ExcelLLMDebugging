Cell: Sheet1!B33 — Retained earnings (formula =C15)

Error: It links retained earnings to only the current year’s Net Income. That ignores prior retained earnings (and prior-year inventory error adjustments), so Assets ≠ Liabilities + Equity.

Solution: Make retained earnings equal to beginning retained earnings plus current net income (and include any prior-year inventory error adjustment). For example, adjust the formulas so they add the prior-year correction: for 2002 use =E15+17000 and for 2003 use =G15-25000 (or generally =Prior_Retained_Earnings + ThisYear_Net_Income). This restores the missing equity and balances the statement.

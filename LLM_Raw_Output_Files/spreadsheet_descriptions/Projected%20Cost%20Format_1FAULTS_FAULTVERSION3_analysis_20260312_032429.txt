Summary of workbook — 3 sheets (concise)

Common top matter (all sheets)
- Header instructions in A1–A7: enter proposed grant year (1–3) and center name; do not edit gray/calculated cells; include amortized equipment (3 yrs) and indirect costs.
- Layout convention: Column A = category labels; Column B = $ amounts; Column C/E/H or adjacent columns = % of relevant grand total; additional columns (D, G, etc.) used for “Finished” and “Other Activities” in Sheet1 and for multiple cost blocks in Sheet3.

Sheet 1: "Seq Production Costs" (Sequence production projected costs)
- Purpose: capture projected one‑year totals for shotgun reads, Q20 bases, finished basepairs and other supporting activities, and compute cost per read/finished basepair.
- Key input cells: B10 = total attempted shotgun reads (one‑year); C10 = total Q20 bases; D10 = finished bases; G10 = list/name of Other Activities.
- Cost category rows (A13–A18): Amortized Equipment, Labor, Materials & Reagents, Sub‑contracts, Travel, Other — enter $ amounts in B (for Shotgun), D (for Finished), and G (for Other).
- Calculated rows:
  - Row 19 (Total Direct Costs): B19 = SUM(B13:B18); D19 = SUM(D13:D18); G19 = SUM(G13:G18).
  - Row 20 (Indirect Costs): B20, D20, G20 are inputs for indirects; % columns reference their grand totals.
  - Row 21 (Grand Cost Totals): B21 = SUM(B19:B20); D21 = SUM(D19:D20); G21 = formula =COUNT(G19:G20) (note: G21 is set to COUNT of G19:G20 here in the sheet).
  - Row 11 (Average cost): B11 = B21/B10 (avg cost per shotgun read); C11 = B21/C10 (avg cost per Q20 base); D11 = D21/D10 (avg cost per finished basepair).
  - Row 23 (One‑Year Total All Activities): B23 = B21 + D21 + G21 (sum of grand totals).
- Percent columns: e.g., C13 = B13/$B$21 gives each cost category as fraction of shotgun grand total; analogous for Finished (E column) and Other (H column).
- Notes: Gray/calculated cells should not be edited; amortization and indirect cost treatment explained in header.

Sheet 2: "Tech Dev Costs" (Technology development projected costs)
- Purpose: capture technology development expenditures for the grant year and their share of total requested funds.
- Key cells:
  - B6 = TOTAL COSTS (total funds requested for the year) — used as denominator.
  - Cost categories in A9–A14: Equipment (amort 3 yrs), Labor, Supplies, Sub‑contracts, Travel, Other (including loan repayment).
  - B17 = SUM(B9:B16) → Total Direct Tech Dev Costs.
  - B16 (Indirect Costs) added into B17 via that SUM range if placed in B9:B16; B17 then is total tech dev costs.
  - B18 = B17 / B6 → % of total funds that tech dev represents.
  - A19: “Contribution to total read cost ($)” is a field for allocation to read costs.
- Notes: cells C9, C10, C12 provide free text guidance/breakdowns.

Sheet 3: "Bioinformatics Costs" (Bioinformatics projected costs)
- Purpose: capture infrastructure, assembly and annotation costs, each with equipment, labor, supplies, subcontracts, travel, other, plus totals and percent-of-total.
- Structure:
  - B6 = TOTAL COSTS (total funds requested for the year) — denominator for % calculations.
  - INFRASTRUCTURE block (A7–A17): B17 = SUM(B9:B16) → Infrastructure total; B18 = B17 / $B$6 → % of total.
  - ASSEMBLY block (A20–A30): B30 = SUM(B22:B29) → Assembly total; B31 = B30 / $B$6 → % of total.
  - ANNOTATION block (A33–A43): B43 = SUM(B35:B42) → Annotation total; B44 = B43 / $B$6 → % of total.
  - A19/A32/A45: fields labeled “Contribution to total read costs ($)” for each block to allocate portions to overall read costs.
- Notes: C‑cells again used for explanatory text/breakdowns; amortization and indirect cost rows included in each block.

Overall calculation flow and key formulas to watch
- Category % = category $ / corresponding grand total (e.g., C13 = B13 / $B$21).
- Direct costs = SUM of category rows (e.g., B19 = SUM(B13:B18); B17 = SUM(B9:B16); B30 = SUM(B22:B29)).
- Grand totals include indirects (e.g., B21 = SUM(B19:B20)).
- Average cost metrics (Sheet1 row 11) divide grand total by activity volumes (reads, Q20 bases, finished bases).
- Final total requested for sequence activities: Sheet1 B23 = B21 + D21 + G21.

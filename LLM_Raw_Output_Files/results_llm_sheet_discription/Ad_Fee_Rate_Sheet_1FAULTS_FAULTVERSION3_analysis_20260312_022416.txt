What caused it
- #REF! occurs when referenced cells were deleted or replaced.
- Other wrong results are due to formulas pointing to the wrong cells (e.g., F62 using F60+F45 instead of summing all percent rows) or referencing cells that were never populated (D21).

How to fix it (short, actionable steps)
1. Find and repair all #REF! formulas
   - Use Excel: Home → Find & Select → Go To Special → Errors to list error cells.
   - For each #REF! formula, use Trace Precedents (Formulas → Trace Precedents) to understand what original source it should point to.
   - Replace #REF! with the correct cell reference. Typical fixes in this worksheet:
     - If a formula multiplies a discipline/firm total by a percent, replace #REF! with the cell that holds that discipline total (example: change =#REF!*0.0547 to = $E$39 * 0.0547 if E39 = 69,500).
     - If the intended source is the discipline fee cell (E36:E57), use those cells (absolute references $E$## if needed).

   Example fix (common pattern in your sheet):

5. Replace hard-coded constants with references

6. Recalculate and verify

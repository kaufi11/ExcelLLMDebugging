D34: Formula uses the wrong sign — currently =D30+D32. Fix to =D30 - ABS(D32) (and do the same for E34).

Why: D32 is “Accumulated depreciation” and must be deducted from gross property (D30) to get net property. Using +D32 will give incorrect results if D32 is stored as a positive amount (or double-subtract if it’s already negative). ABS makes the formula robust.

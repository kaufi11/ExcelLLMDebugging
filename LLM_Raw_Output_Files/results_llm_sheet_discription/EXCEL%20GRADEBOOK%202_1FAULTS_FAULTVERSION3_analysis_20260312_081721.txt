Summary: there's one clear formula error causing wrong quiz average (and a bad final grade) for Reed (row 5).

Error
- Cell X5 contains: =(COUNT(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)
  - That uses COUNT instead of SUM, so it computes (5 − 75)/(5−1) = −17.5 instead of the average after dropping the lowest quiz.

Correct formula
- Replace X5 with:
  =(SUM(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)

Why this fixes it
- The intended calculation is (total of quizzes − lowest quiz) / (number of quizzes − 1). Using SUM gives the total; using COUNT does not.

How to fix (quick steps)
1. Select X5.  
2. Enter: =(SUM(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)  
3. Press Enter.  
4. (Optional) Drag-fill X5 down/up to apply the same correct formula to other rows if needed.

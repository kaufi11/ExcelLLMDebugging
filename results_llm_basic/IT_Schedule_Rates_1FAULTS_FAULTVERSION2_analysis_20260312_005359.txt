How to prevent this and easier maintenance
- Use a single, copyable formula that references the escalation percent (column C). Example put this in D6 and copy across/right and down:
  - In D6: =B6*(1+$C6)^(COLUMN()-3)
  Explanation: COLUMN()-3 gives exponent 1 for column D, 2 for E, etc., so each column is B * (1+escalation)^n.
- Or, simpler and very clear: in D6 use =B6*(1+$C6) and in E6 use =D6*(1+$C6) then copy E6 across; or set D6 =B6*(1+$C6) and then fill right with relative references if $C6 is used correctly where needed.

Summary of errors
- I15 on sheet "B Direct Payments" is wrong: it contains =COUNT(I10:I14). COUNT returns the number of numeric cells (usually 5), not the sum of payment values. So the "Total value of payments" cell shows a count, not the total amount.
- No other obvious formula errors in the provided sheet summary, but you should scan for other accidental COUNT usages.

Fix (quick)
- Replace the formula in I15 with either:
  - =SUM(I10:I14)  — sums the per-row totals, or
  - =SUM(F10:H14)  — sums all payment values directly.
- Steps: select I15 → type one of the formulas above → Enter.

Why this fixes it (short)
- SUM adds the numeric values; COUNT only counts how many numeric cells exist. Using SUM yields the actual total payment amount instead of a row count.

Extra checks (recommended)
- Verify I10:I14 are numeric (not text). If any are text like "n/a" the SUM will ignore them; if you need them treated as zero, clear/convert those cells.
- Use Find (Ctrl+F) to search for "COUNT(" across the workbook to make sure there are no other accidental COUNT→SUM mistakes.

Sheet1 (SECTION 01 MW 11–11:50)
- Header rows define categories and weights:
  - Homework assignments in D1:S1 (10 points each). T column holds SUM(D#:S#) for assignment total.
  - U column: % for homework (example U5 formula = T5/150*100 — homework max = 150).
  - Lab / First & Second Research Project in Y:AF and AG:AO. AF1 = SUM(Y2:AF2); AO1 = SUM(AG2:AO2); AQ (per-row) = SUM(Y#:AP#) (AP appears used in many rows).
  - AR column: project/lab % (e.g., AR5 = AQ5/350*100 — projects/lab max = 350).
  - Exams grouped across BA:BS (subtotals BA, BF, BM, BS), BT = BA+BF+BM+BS; BU = BT/400*100 (exams max = 400).
  - CE column: combined points = SUM(BZ:CD) where BZ = AQ (projects), CA = BT (exams), CB = T (homework). CF = CE/830*100 (overall percent with 830 total possible).
  - CG column contains assigned letter grades (manually entered in many rows).
- Student rows (A5:C19 etc): contain name, instructor, ID, then per-assignment scores in D:...:S; computed sums and percentages in T, U; repeated blocks for projects (Y:AO), labs/exams (AV:… BF/BM/BS groups) and overall totals (BZ:CE).
- Key formulas (examples):
  - Tn = SUM(Dn:Sn) — assignment sum.
  - Un = Tn/150*100 — assignment %.
  - AQn = SUM(Yn:APn) — projects/lab subtotal.
  - ARn = AQn/350*100 — projects/lab %.
  - BAn, BFn, BMn, BSn = SUM(...) — exam subgroups; BTn = BA+BF+BM+BS; BUn = BTn/400*100.
  - BZn = AQn; CAn = BTn; CBn = Tn; CEn = SUM(BZn:CDn); CFn = CEn/830*100.
- Summary row formulas (class averages): e.g., U22 = AVERAGE(U5:U21), AJ22 = AVERAGE(AJ5:AJ21), AR22 = AVERAGE(AR5:AR21), and other AVERAGEs for exam subgroups and overall CF22 = AVERAGE(CF5:CF21).

Summary of workbook (concise)

General structure
- 3 sheets: Annex1, Annex2, Annex3. Each sheet reports Moroccan financial system aggregates by instrument/institution and by year (columns for years, rows for items).
- Common formula patterns: SUM() to aggregate groups, cell references to column totals for percentage shares (e.g., X/Y), and ratios to GDP or to a yearly grand total (fixed references like $B$32, $D$32, etc.). Several cells use simple arithmetic constants (e.g., =6.07+0.2).

Annex1 — Overview (page-style summary)
- Purpose: high-level snapshot (1992 / 1998) of the banking system, other financial institutions, contractual savings, capital market, money market and government domestic debt.
- Layout: left column labels; year/value columns arranged in pairs (Fin 1998 / Fin 1992) for Assets, Credits, Deposits and subshares.
- Key raw values:
  - Total assets (row 6): 301 (Fin 1998), 185.6 (Fin 1992).
  - Total credits (row 6 D6): 167.6 (1998) / 95.9 (1992).
  - Total deposits (row 6 F6): 183.3 (1998) / 111.4 (1992).
  - Market capitalization (row 26/30): 145.1 (1998), 75.6 (1996), 26 (1993).
  - Domestic government debt (row 52): 131 (1998) / 67.2 (1992).
- Key formulas and ratios:
  - Bank / specialized banks shares computed as =group_value/total (e.g., B7 =234.2/B6).
  - Total assets / GDP and other /GDP ratios use explicit GDP denominators (e.g., =B6/350; some use 341 or 242.9 depending on year).
  - Capital-market ratios: trading volumes summed (D38 = B38+C38), trading-volume/market-cap formulas reference combined shares and bond flows (e.g., B39 =B38/($D$38+$D$41)).
  - Contractual savings: pension = 84.73 - Insurance (B22 formula).
- Notes/footnotes present (e.g., each trade counted twice).

Annex2 — Financial assets of non‑financial agents (time-series table)
- Purpose: detailed breakdown of households/non‑financial agents’ financial assets 1992–1998 (amounts and % of total).
- Layout: year columns across (B, D, F, H, J, L, N), rows grouped: Liquid assets, Short-term investments, Medium-term investments, Institutional savings, Total. Each amount column accompanied by a % column that divides by that year’s grand total (e.g., C10 = B10/$B$32).
- Key components and values (examples):
  - Currency/coins (row 11): 35,745 (1992) up to 50,644 (1998).
  - Demand deposits (row 12): 65,921 → 103,145 (1998).
  - Passbook savings (row 15): 14,912 → 29,523 (1998).
  - Pension funds (row 29) and Insurance reserves (row 30) each ~20,600–38,000 across years.
  - Total (row 32): assembled as B28+B23+B14+B10 (plus mutual funds where present).
- Key formulas:
  - Component totals use SUM() and simple arithmetic (e.g., B14 = SUM(B17:B19)+B15).
  - Percent columns compute shares of the year total using $-anchored references (e.g., C14 = B14/$B$32).
  - Memo/GDP row shows total/GDP in percent for specified years.
- Notes: (1) 1997 estimates; (2) totals exclude corporate securities.

Annex3 — Bank-by-bank assets, deposits, credits (detailed bank-level time series)
- Purpose: three-part annex (assets, deposits, credits) showing 1992–1997 by bank and bank group.
- Layout:
  - Part 1 (ASSETS): columns for years; rows list individual banks grouped into Moroccan commercial banks (I), Foreign commercial banks (II), and Specialized banks (III). Group rows sum members (SUM()) and percent columns divide by the group/year total (e.g., C12 = B12/$B$38).
  - Part 2 (DEPOSITS): same structure with a different total row reference (Total I+II+III = B77).
  - Part 3 (CREDITS): same structure with total at B116.
- Key banks and values (examples):
  - Large commercial banks: CPM (e.g., assets B19 = 42,069 in 1992), BMCE, Wafabank, BCM.
  - Specialized banks: CIH, CNCA, BNDE.
  - Totals: group and grand totals are computed by summing group rows (B38, B77, B116) and used as denominators for % columns.
- Key formulas:
  - Group sums: e.g., B12 = SUM(B13:B24); B26 = SUM(B27:B31); B33 = SUM(B34:B36).
  - Grand totals aggregate groups (e.g., B38 = B33+B26+B12).
  - Percent columns: each amount divided by that year’s grand total cell (e.g., C12 = B12/$B$38).
- Ancillary: page markers and source footers (Bank Al-Maghrib).

Observations (useful when editing)
- Percentage columns consistently use year-specific grand totals (anchored references like $B$32, $B$38, $B$77, $B$116).
- Many formulas are simple SUMs or divisions; a few cells use direct arithmetic (e.g., =6.07+0.2) instead of a pre-calculated source cell.
- Ensure denominators (GDP or totals) are correct and anchored if copying formulas across columns to avoid unintended relative references.

Summary of errors found
1) Wrong operator in O14
   - Cell O14 currently: =N14*$N$3 (multiplication).
   - All other student O# cells use division (N# / $N$3) to compute homework percent. O14 multiplying gives a huge, wrong homework percent and therefore wrong BC and BD for row 14.

2) Letter‑grade formulas overwritten (hard‑coded text)
   - Cells BD9 and BD14 (at least) contain literal text "B-" instead of the grading IF formula used elsewhere (BD4, BD5, etc.). Any manual overwrite like this makes that student's letter grade not update if BC changes.

Possible additional problem to watch for
- If any other BD cells or percent cells have been accidentally overwritten, those will give inconsistent results. (I only saw BD9 and BD14 as examples in your listing.)

   - Change O14 to the correct formula: =N14/$N$3
   - Effect: restores O14 to be the homework percent (0–1), bringing BC14 and BD14 back in line.

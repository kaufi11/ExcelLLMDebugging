Overview
- Workbook contains 3 sheets: Sheet1 (primary Cost Proposal Template), Sheet2 and Sheet3 (no content provided).
- Sheet1 is a task-based labor/cost template for the "Chittenden County MPO 2004 Transportation Model Update" (A1) dated 2004-01-14 (A3).

Layout (Sheet1)
- Header: title (A1), template label (A2), date (A3).

  - D‚ÄìG: Labor hours input by Job Category 1‚Äì4 respectively.

- Section/subtotal aggregation: Most section header rows (e.g., D14, E14, ‚Ä¶) use SUM over their child rows (e.g., D14 = SUM(D15:D17)).

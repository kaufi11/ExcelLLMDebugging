Sheet name: "2001 - 1994"

Purpose:
- Presents Texas Assessment of Academic Skills (TAAS) counts and passing percentages for "All Students" by grade for spring 2001 down through spring 1994.

Overall layout:
- Single sheet with repeated year blocks. Year headings in column A (e.g., A6: Spring 2001, A21: Spring 2000, etc.).
- Each year block has a small subheader (columns D–F) showing "Number" (tested), "Number" (passing), and "Percent" (passing).
- Grade labels in column C (Grades 3, 4, 5, 6, 7, 8*, 10, and "All Grades").
- Numeric data in column D (Tested) and column E (Passing).
- Percent values in column F are calculated formulas (Passing / Tested).

Key formulas and ranges:
- Per-grade percent: F# = E# / D# (e.g., F10 = E10/D10).
- Year totals: Tested total is SUM over that year’s D cells (examples):

- Consistent column use: C = grade label, D = tested counts, E = passing counts, F = computed percent.

- Grade 8 marked with *: "All tests taken at Grade 8 include the reading, mathematics, and writing tests only." (A118).

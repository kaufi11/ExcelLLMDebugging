Cell: Fall 2002!H14 — Error: formula uses COUNT(H10:H13) instead of SUM(H10:H13).

Why it’s wrong: COUNT returns the number of numeric cells (4), not the sum of their values, so the subtotal is 4 instead of ~4706.02; this mis‑value propagates to H21/H25.

Fix: edit H14 and replace the formula with =SUM(H10:H13) (then press Enter). SUM adds the fees; COUNT only counts entries.

Found one clear error:

1) Wrong quiz-average formula in X5
- Current: =(SUM(R5:V5)+MIN(R5:V5))/(COUNT(R5:V5)+1)
- Problem: that adds the lowest score and increases the count — it does not "drop the lowest" like the other rows. It produces a wrong (too low) average for the row.
- Example (row 5 quizzes = 75,93,97,95,96): correct drop-lowest average = (456 − 75) / (5 − 1) = 381 / 4 = 95.25. The bad formula gives (456 + 75) / (5 + 1) = 531 / 6 = 88.5.

Fix (make X5 like the other quiz rows)
- Replace X5 with:
  =(SUM(R5:V5)-MIN(R5:V5))/(COUNT(R5:V5)-1)

Optional good-practice suggestion
- If you ever move/copy the Final Grade formula across columns, make the weight references absolute so they never shift (e.g. use $C$1, $K$1, $S$1, $Z$1). As currently used (C$1 etc.) copying down is fine.

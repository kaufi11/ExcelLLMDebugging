Findings — error
- Cell I19 is wrong. It currently has =COUNT(I7:I17). COUNT returns how many numeric cells (9), not the sum of the values.
- All other category totals (G19, H19, J19, K19) use SUM, which is correct.

Why it's wrong
- The I column contains numeric vote counts. You want the total number of votes (sum), not the count of cells that contain numbers.

Quick fix
1. Select I19.
2. Replace the formula with: =SUM(I7:I17)
3. Press Enter.

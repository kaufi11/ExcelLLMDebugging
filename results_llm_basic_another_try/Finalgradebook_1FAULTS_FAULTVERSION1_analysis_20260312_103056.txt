3. Correct the typo in Sheet3 F14:

   Explanation: The test score is 101, not 510.

4. Fix the VLOOKUP range and lookup values in Sheet1 column J:

   Explanation: VLOOKUP needs the lookup values to be on the same scale as the lookup key; otherwise, it returns wrong results.

- Correct typos in formulas.

O95 — Error: cell contains the hard-coded value 7 instead of calculating the underspend.  
Fix: replace 7 with the formula =D95-M95 (or =SUM(D95-M95) to match sheet style).  
Why: all other "underspend" cells use D - M; the hard-coded 7 won't update if allocations or forecasts change and will make subtotal O99 (and overall totals) incorrect. Replacing it ensures automatic, correct recalculation.

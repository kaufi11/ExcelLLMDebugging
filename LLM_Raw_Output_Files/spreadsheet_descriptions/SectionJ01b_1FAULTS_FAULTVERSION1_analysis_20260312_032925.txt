Summary of workbook structure and key contents

Overall
- Purpose: LOE (level-of-effort) and pricing workbook for a small-business set‑aside procurement showing Base Period + Options 1–7 by labor category across three task‑order types: Fixed Price (1% LOE), Cost Type (29% LOE) and T&M/Labor‑Hour (70% LOE).
- Common layout across sheets: Column A = Labor Category; Columns B = Base Period (hours or hours × rate); subsequent groups of 2–3 columns represent Options 1–7 (hours, rates/unit price/loaded rate, extended price).
- Main calculation pattern: Option values cascade from the prior period with a +2% uplift per option (i.e., next = prior + prior*0.02). In several places the Option 4 step uses +2.3% (0.023) instead of 2% — see anomalies below.
- Totals: Each sheet has a “Total” row summing the option columns, and a “Grand Total” aggregating across periods.

Sheet-by-sheet

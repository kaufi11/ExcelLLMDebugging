Quick checklist to avoid similar issues
- Use consistent absolute/relative references (and $ where appropriate) and keep the same pattern across columns so copying formulas doesn’t pick up the wrong cell.
- Highlight important scenario cells (baseline price, post-add price, upgrade price) and name them with defined names so formulas can’t accidentally point to the wrong one.
- After structural changes, re-run sanity checks: compare E21..H21 to D21 etc (look for a single column that differs) and check totals/IRRs for step-changes that look like typos (e.g. /380).

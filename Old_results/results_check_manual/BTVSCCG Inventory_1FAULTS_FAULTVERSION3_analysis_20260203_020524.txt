Summary — primary error found
- On the Class of '99 sheet row 18 (cell E18) the formula is =COUNT(I18:J18). That returns the number of numeric entries (1), not their sum (6). This makes the "Have" and "Have Total" values wrong for that card.

Why it's wrong (short)
- COUNT(range) counts how many numeric cells are in the range.
- You want the total number of copies, so you must add the numeric values: SUM(range).

Fix (exact)
- Change E18 from =COUNT(I18:J18) to =SUM(I18:J18).

How to check for the same mistake elsewhere
- Search the workbook for "COUNT(" (Ctrl+F) and inspect each hit. Any COUNT used where you expect a sum of quantities should be replaced with SUM(range).
- Optionally replace COUNT(...) → SUM(...) for these cases.

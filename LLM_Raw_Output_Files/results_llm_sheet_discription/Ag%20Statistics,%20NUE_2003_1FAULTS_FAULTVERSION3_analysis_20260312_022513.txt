1) Wrong cell reference (World 1996)

2) Incorrect logic for "N fertilizer savings per 1% increase in NUE" (many sheets: World 1996 E32, World 2002 D44, Developed/Developing sheets E31/E32 etc.)
- Error: Formulas like =E4*(0.343) - E28 compute a difference between a hard-coded target fraction and current removed N. That does NOT compute the reduction in fertilizer applied when NUE increases by 1 percentage point.
- Correct (exact) formula for saved applied N when NUE increases by 1 percentage point (absolute 0.01) while yield (crop N removed) stays the same:
  Let applied = A (cell with cereal fertilizer consumption, e.g. E4 or D6)
  Let removed = R (cell with fertilizer-derived N removed, e.g. E28 or D31)
  current_NUE = R / A
  new_applied_needed = R / (current_NUE + 0.01)
  savings = A - new_applied_needed
- Excel form (replace the wrong cell formula with this):
  = A - ( R / ((R / A) + 0.01) )
  Example (World 1996): replace E32 with
  =E4 - (E28 / ((E28 / E4) + 0.01))
- Why: This computes how much less fertilizer must be applied to achieve the same crop N removal if NUE increases by 0.01 (1 percentage point). The workbook’s current formula instead compares to a fixed fraction (0.343 etc.) and is conceptually wrong for “per 1% increase”.

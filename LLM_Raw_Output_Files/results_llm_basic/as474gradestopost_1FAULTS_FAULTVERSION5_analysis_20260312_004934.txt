2) Missing / ignored scores vs zeros (possible data issue)
- Problem: Some cells contain 0 (e.g. H17). AVERAGE will treat 0 as a real score and lower the average. If 0 means “missing” rather than a scored zero, your averages will be incorrect.
- Fix options:
  - If 0 = missing, change those 0s to blank (delete the 0), or use an average that ignores zeros, e.g. per-student average in I7:
    =AVERAGEIF(B7:H7,">0")
  - If 0 = actual score, keep as is.
- Why: AVERAGE includes zeros. AVERAGEIF(...,">0") excludes blank and zero entries.

- Why: Using 1 misrepresents the scoring scale and will give incorrect normalization/percent calculations.

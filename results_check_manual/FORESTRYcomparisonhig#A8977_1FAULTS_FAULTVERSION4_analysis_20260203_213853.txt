Short answers first

- Main error: on sheet "DATA TEXT" cell C13 has the wrong formula. It reads =VLOOKUP(D14,sort0,2,FALSE) but should be =VLOOKUP(D13,sort0,2,FALSE). This causes C13 to look up "Conifer - Mixed conifers" instead of "Conifer - Other conifers" (so C13 and C14 both return the same lookup and everything below may be misaligned).
- Secondary (minor) error: on sheet "DATA" cell F28 is entered as "23,397" (text with comma). That may break numeric calculations if Excel treats it as text.
- Where is C13? There are C13 cells on each sheet. The problematic one is on the sheet named "DATA TEXT" (cell C13 on that sheet). On the "Title page" sheet C13 contains the value 8 (the species order number).

What to do (quick fixes)

1) Fix the mis‑reference
- On sheet DATA TEXT change cell C13 to:
  =VLOOKUP(D13,sort0,2,FALSE)
- Why: the lookup must use the species name in the same row (D13). Using D14 shifts the mapping and creates duplicated/wrong order numbers.

2) Fix the numeric text
- On sheet DATA select F28 and remove the comma so the cell is numeric (enter 23397) OR convert it with =VALUE("23,397") then paste as value.
- Why: numbers stored as text can be ignored by SUM and other numeric functions or give unpredictable results.

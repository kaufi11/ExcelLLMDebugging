5) Variance sheets: possible DIV/0 and reference consistency issues
- Problem: variance formulas use (D-C)/C (or (D8-C8)/C8) without handling C=0. If the base cell is zero, you'll get #DIV/0!. Also some variance sheets copy formulas that reference the wrong months (make sure each actual & base column point to the correct Year 1 month).

- Why: prevents #DIV/0 and makes variance cells robust.

Error: Formula uses the constant 784 instead of the key cell (K20), so it always evaluates false and returns 0.

Fix (replace the formula in Ap3!L20):
=IF(K20=2,Rate!E15*0.33,IF(K20=3,Rate!E15*0.66,IF(K20=4,Rate!E15*1,0)))

Workbook purpose
- Presents percentage distributions of class grades at BYU–Hawaii by semester (Fall 2000, Winter 2001, Spring 2001, Summer 2001), by class level (100–299 vs 300+), and by department/school; plus an Average GPA sheet converted from those letter‑grade distributions.

Overall structure (sheets)
1. "University Total by Sem." — University‑level summary by semester & class level (grades A → WF, X). Columns grouped by semester (Fall in B–D, Winter in E–G, Spring in H–J, Summer in K–M) with a Grand total block (N–O). Totals row uses SUM formulas and a note about rounding (A34).
2. "University Total" — Consolidated percent table by Division/School (combined semesters) and several repeated blocks that break out by Departments/Schools. Each block lists grades (A → WF, X) down the rows and percent values across school/department columns.
3. "Class Grades by div." — Multiple sheets or continuous blocks for each school/department (School of Business, Computing, Education, Biology, Biochemistry, English, EXS, Fine Arts, Hawaiian Studies, History, ICS, Languages & Linguistics, Political Science, Psychology, Religion, Social Work, IDS, classes with no department). Each block follows the same layout: semesters across columns, subcolumns for 100–299 / 300+ / Total, grade rows, and a Total row at the bottom.
4. "Sheet1" — Aggregation sheet. For each grade row it sums the corresponding department/division totals from 'University Total' (e.g. A1 = SUM('University Total'!B8:L8), etc.). Column D computes ratios dividing the combined totals (Cx) by C24 (overall cumulative total).
5. "Average GPA" — Same block structure as the percent tables but shows numeric average GPA values (converted from the letter distributions). Contains per‑division and per‑department average GPAs by semester/class level and Grand totals. Final row/column give overall average (e.g., Total row B27 = 3.15, C27 = 3.24, N27 = 3.19).

Common layout and conventions
- Columns: semesters are arranged in blocks starting at B, E, H, K for Fall/Winter/Spring/Summer. Within each semester block there are three subcolumns: 100–299, 300+, Total. A Grand total column aggregates across semesters.
- Rows: grade categories run vertically: A, A-, B+, B, B-, C+, C, C-, D+, D, D-, F, I, NP, P, T, UW, V, W, WF, X, then a Total row which uses SUM formulas.
- Totals: almost every departmental and school block has a Total row that is a SUM of the grade rows (e.g., =SUM(B9:B29) or similar). Many Grand totals normalize to 1.0 (or very close) because the values are percentages of the class grades.
- Formulas: mainly SUM formulas aggregating the grade rows; Sheet1 references 'University Total' ranges to compute cross‑department sums and computes relative proportions by dividing by a cumulative total (e.g., D1 = C1 / C24 pattern). Average GPA sheet has no visible conversion formulas in the excerpt, but values are described as converted from letter‑grade tables.

Key formula patterns and important cells
- Department/school totals: e.g., in many department blocks the Total row uses =SUM(range) (examples: 'University Total by Sem.' B31 = SUM(B9:B29), 'University Total' B65 = 1 (overall total), departmental totals such as B137 = SUM(B116:B136)).
- Grand normalization: some cells explicitly equal 1 or very close (e.g., M31 = 1 in the first sheet block), indicating the percent distribution sums to 100%.
- Aggregation / ratio sheet: "Sheet1" uses references to 'University Total' to build combined totals (A1 = SUM('University Total'!B8:L8), etc.) and computes D‑column ratios dividing those sums by C24 (overall total).
- Rounding note: A34 in "University Total by Sem." — numbers are rounded to one decimal place.

Key values and summary metrics
- Overall average GPA (from "Average GPA" sheet): Total row B27 = 3.15 (Fall 2000), C27 = 3.24 (300+), D27 = 3.18 (Total Fall+300+ combined), Grand total N27 = 3.19 (overall).
- Many departments have high A proportions in EXS, Fine Arts, and classes with no department (examples: EXS grand A ≈ 0.416; "classes that belong to no specific department" grand A ≈ 0.531).
- Totals across grade rows generally sum to 1 (or are expected to), confirming these are proportional distributions.

Notes, gotchas, and observations
- The workbook is highly repetitive: each department/school uses the same grade × semester × level layout and SUM totals.
- Some cells are marked N/A or blank where no data exist (noted in Average GPA and some departmental percent cells).
- Some SUM ranges differ slightly across blocks (some include one extra row or stop at slightly different row numbers), so double‑check ranges when updating structure.
- Sheet1 uses a division by C24 — if C24 is blank or zero this will produce errors; it assumes an overall total present.

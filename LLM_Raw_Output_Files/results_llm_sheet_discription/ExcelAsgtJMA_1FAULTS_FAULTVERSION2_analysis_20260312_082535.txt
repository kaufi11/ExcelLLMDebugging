Summary of errors
1. Student total ranges use the wrong column end: formulas like =SUM(D9:CC9) should end at I, not CC. Using CC extends far beyond your data and can capture unintended values later.
2. Row 19 uses COUNT instead of SUM: C19 = COUNT(D19:CC19) returns "6" (number of numeric cells) instead of the points total.
3. Grade % formulas wrap the division in SUM unnecessarily: B9 = SUM(C9/$C$6) — SUM is redundant (not harmful) but confusing. Also column B must be formatted as Percentage.

Correct formulas (what to put)
- C6 (total possible): =SUM(D6:I6)  — this is already correct.
- Student points totals (copy down C9:C24): =SUM(D9:I9)  (replace row number for each row)
- Fix row 19 particularly: C19 =SUM(D19:I19)
- Grade % (copy down B9:B24): =C9/$C$6  (then format column B as Percentage)

Why these fixes
- SUM(Dn:I n) sums only the assignment columns (D–I). SUM(Dn:CCn) spans many extra columns and may include unintended data; COUNT returns the count of numeric cells, not their sum.
- Removing the unnecessary SUM around the division makes the formula clearer. Formatting the result as Percentage displays the correct grade percent.

Quick steps to fix
1. In C9 enter =SUM(D9:I9). Drag-fill down to C24.
2. In B9 enter =C9/$C$6. Drag-fill down to B24.
3. Format B9:B24 as Percentage (Home → Number → %).
4. Verify C6 =SUM(D6:I6) still equals 100.

Optional improvement
- Convert the range into an Excel Table or use structured references so adding future assignment columns won’t break the totals.

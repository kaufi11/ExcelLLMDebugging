Cell: C24 (Total Assets) — ERROR: formula =SUM(B20:B23) is summing column B while the year totals are in column C, so Assets and Liabilities+Equity are summing different columns and the balance doesn’t match.

Solution: Make the totals use the same year column. Change C24 to =SUM(C20:C23) (or move the asset values into column C) so Total Assets and Total Liabilities & Equity reference the same column/year. Explanation: the balance sheet must sum the same column for assets and for liabilities+equity; mixing B and C produces the imbalance.

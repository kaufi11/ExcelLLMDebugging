Summary of error
- On the "Spring 2003 per credit hour" sheet cell H14 (Subtotal State Components for Veterinary, per term) has the wrong formula: it is =SUM(H10:H12) and therefore omits H13 (Student Financial Aid). All other similar subtotals use H10:H13.

Why this is a problem
- Because H13 is omitted, H14 is too low by H13 (219.55). That error propagates to:
  - H21 (Total Florida Resident per term) which uses H14, and
  - H25 (Total Non‑Resident per term) which uses H21.
- So H14, H21 and H25 on the Spring sheet are all understated by 219.55.

Correct values (Spring 2003, Veterinary per term)
- H14 correct = SUM(H10:H13) = 4,706.01 (current = 4,486.46)
- H21 correct = H14 + H20 = 5,104.01 (current = 4,884.46)
- H25 correct = SUM(H21:H24) = 13,890.29 (current = 13,670.74)

Fix (quick)
1. Edit H14 and change the formula to:
   =SUM(H10:H13)
   or copy the formula from G14 (which is correct) into H14.
2. Recalculate or press F9 if needed.

How to verify / prevent
- After the change, confirm H14, H21 and H25 match the “correct values” above.
- Quick audit: search each sheet for SUM ranges in rows 14 and 20 and confirm they include rows 10:13 and 16:19 respectively. This catches any other accidental omissions.
- Consider using consistent formulas filled across the range (fill right) to avoid manual typos.

Optional note
- Minor 0.01–0.02 differences between sheets are just rounding or input-value differences; the main functional error is the missing H13 in the Spring H14 formula.

Summary — MODEL07 sheet (MODEL 7: PUSH‑DOWN WORKSHEET & CONSOLIDATION WORKSHEET)

Purpose
- Combined push‑down accounting worksheet and consolidation worksheet used for Chapter 7 problems. Supports loading problem data, posting push‑down adjustments, and posting the basic elimination entry.

Layout (major blocks and row locations)
- Header/instructions: rows 1–37 — file title, general instructions (calls out row 138 for data load), macro usage notes.
- Analysis of Investment Account — Excess cost elements: rows ~15–23 — area to enter assigned amounts and SUM formula (B22 = SUM(C22:I22)).
- Entries to implement push‑down accounting: rows 24–46 — list of debit/credit lines; macros Q will post many of these (notes in column E), manual lines reserved for others; totals at row 39 (C39 = SUM(C27:C38), D39 = SUM(D27:D38)).
- PUSH‑DOWN worksheet area: begins row 49 — columns show Old Basis (C), Adjusting Entries Dr (D)/Cr (E), and New Basis (F). Balance sheet lines run rows ~53–65:
  - Each New Basis cell uses the pattern: Fxx = (Cxx + Dxx - Exx). Total Assets row 65 sums C54:C64 etc.
  - Proof of debits/credits: D76 = D65 + D74; E76 = E65 + E74.
- Analysis of Parent’s Investment — Book value element: rows ~78–86 — formulas to sum equity accounts (e.g., B85 = SUM(C85:F85)).
- Basic elimination entry: rows 88–95 — elimination lines (Common stock, APIC, RE, Revaluation) with C90..C93 = ($C$85,$D$85,$E$85,$F$85) and D94 = ($B$85). Totals at row 95 sum dr/cr. Instruction to use macro P to post to CONSOLIDATION.
- CONSOLIDATION WORKSHEET area: begins row 100 — consolidation entries header (row 102) and a full balance sheet layout (rows 104–132) with:
  - Left columns (B–F) reference the push‑down area via formulas (e.g., C105 = ($F$54); B118 = SUM(B105:B117); F105 = (B105+C105+D105-E105)).
  - Problem data block to the right (columns K:U, rows 105–132) — stores unadjusted problem company balances for multiple problems.
  - Many numeric sample problem values are present in K105:U132 (cash, AR, inventory, notes receivable, land, bldg & equip, accum. depr, patent, goodwill, payables, long‑term debt, equity lines).
  - Row 118 and row 132 compute totals for assets and liabilities/equity per column (SUM ranges shown).
  - Consolidation proof: D134 = D118 + D132; E134 = E118 + E132.

Key formulas & links
- New basis pattern (push‑down): Fxx = Cxx + Dxx - Exx (applies to most balance sheet rows).
- Totals: row 65 (assets) and row 74 (liab & equity) use SUM across the C/D/E/F columns; corresponding proofs compare totals of assets vs. liabilities + equity.
- Basic elimination entry pulls amounts from the Book Value analysis (absolute references to $C$85:$F$85).
- Consolidation sheet references push‑down new basis cells (examples: C105 = $F$54 etc.), so push‑down adjustments drive consolidation results.
- Problem data area (K105:U132) is the source for loading data into the push‑down and consolidation areas via macros.

Macros (row 138+; keystrokes shown)
- A–G: load UNADJUSTED problem data into PUSH‑DOWN (A–C load 7‑1 to 7‑3 into push‑down only; D–G load 7‑4 to 7‑7 into both PUSH‑DOWN and CONSOLIDATION).
- P: post basic elimination entry to CONSOLIDATION.
- Q: post unamortized excess cost reclassification entry to PUSH‑DOWN (posts many push‑down debit/credit lines).
- U: clear columns C, D, E in PUSH‑DOWN.
- W: clear column B in CONSOLIDATION.
- X: clear columns D & E in CONSOLIDATION.

Key values present (problem data block K105:U132 — selected examples)
- Example column K (one problem): Cash 55,000; AR 180,000; Inventory 300,000; Land 500,000; Bldg & Equip 723,000; Accum. Depr -123,000; Goodwill 120,000; Payables 300,000; Long‑term debt 700,000; Subsidiary common stock 100,000; Retained earnings 300,000.
- Other columns L–U contain similar per‑problem balances (see K105:U132 for full list and sums at rows 118 and 132).

Notes / Usage
- Load problem data using macros A–G (or manually enter into K105:U132). After loading, use macro Q to post push‑down adjustments (rows 27–38) and macro P to post the elimination entry to the consolidation area.
- Push‑down worksheet begins at row 49; consolidation worksheet begins at row 100. Problem data is stored in K105:U132 (noted in row 136).
- Many formulas use absolute references to the analyses so changes in the analysis block update the posted entries and consolidation automatically.

C122 — Error: contains a hard‑coded value (-26908) instead of linking to the operating cash reconciliation.  
Solution: replace the contents of C122 with =C511 (or =+C511).  

Explanation: C511 is the computed "Cash (applied to) / generated from operations" from the reconciliation (note 17.1). C122 should use that dynamic result so the operating cash inflow (C117) updates correctly when underlying numbers change. Replacing the fixed number with =C511 fixes the cash‑flow totals and prevents mismatches.

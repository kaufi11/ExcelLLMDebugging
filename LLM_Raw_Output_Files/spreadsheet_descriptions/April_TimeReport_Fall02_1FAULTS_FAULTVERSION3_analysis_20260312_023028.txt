Summary — Sheet: "Reporting"

Layout
- Header: Row 1 = Week labels (B1:M1 for Weeks 4–15), Row 2 = date ranges (B2:M2). Column A contains task/category labels.
- Data area: Rows 3–34 list categories and their sub-activities (Reading/Preparing, Homework, Class, Meetings, etc.). Values in B:M are hours per week for each activity. Column N is the row total (sum across B:M).

Structure & formula patterns
- Category total rows:
  - Management (row 3): each week cell = SUM of its three subrows (B3 = SUM(B4:B6), etc.).
  - Models (row 7): SUM(B8:B10) etc.
  - Requirements (row 11): SUM(B12:B14) etc.
  - Elective (row 15): SUM(B16:B18) etc.
  - Studio I (row 19): SUM(B20:B34) etc.
- Row totals: For each activity row, column N uses N = SUM(B:M) (e.g., N4 = B4+C4+...+M4).
- Weekly totals: Row 35 sums the five category-total rows per week (B35 = B3+B7+B11+B15+B19; similarly for C35:M35). N35 = N3+N7+N11+N15+N19.

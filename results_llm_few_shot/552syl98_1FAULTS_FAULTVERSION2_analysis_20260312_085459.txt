Faulty cell
- B18

Error type
- Typo / data-entry error that propagates (mechanical).

Why it's wrong (short)
- B18 has =B15+663 which adds 663 days (nearly two years) instead of advancing one week. Many later date formulas reference B18 (and later weekly offsets), so many class dates become wrong.

Fix
- Replace B18 with: =B15+7

What that does / why it fixes it
- +7 advances one week from the prior Monday (B15). That restores the intended pattern (alternating +2 for the next class day, +7 to move a week forward) and corrects all downstream dates that depend on B18.

Extra notes (optional clean-up)
- Leading plus signs (=+B16+2) are harmless but unnecessary; you can remove them for clarity.
- If you want a more robust approach, use a consistent fill pattern (fill handle) or use WORKDAY/WEEKDAY logic if skipping weekends/holidays.
